# Medical Quiz Master - 医学quiz练习技能

> 医学知识Quiz练习技能，支持执业医师考试复习、医学生自主学习、医学知识考察

## 功能特性

- **Quiz出题系统** - 每次1道题，难度循序渐进
- **即时反馈机制** - 正误判断、详细解析、记忆技巧
- **学习进度跟踪** - 实时统计正确率、薄弱点分析
- **错题本系统** - 自动记录、针对性强化
- **高频考点速记表** - CSF鉴别、时间窗、首选药物、特征性体征

## 使用方法

### 触发技能

在 Claude Code 中输入以下任意一种：

```
出题
练习医学知识
考我
做几道题
医学quiz
神经病学问答
```

### 交互命令

| 命令 | 功能 |
|------|------|
| 继续 / 下一题 | 出下一道题 |
| 记录错题 | 更新错题本 |
| 复习错题 | 从错题本随机抽取重做 |
| 停止 / 结束 | 结束练习，显示总结 |

## 题型分类

| 类型 | 描述 |
|------|------|
| 病因诊断型 | 询问病因或致病因素 |
| 临床表现型 | 询问典型症状或体征 |
| 鉴别诊断型 | 区分相似疾病 |
| 检查诊断型 | 询问首选或确诊检查 |
| 治疗原则型 | 询问首选治疗方案 |
| 药物选择型 | 询问首选药物 |
| 时间窗型 | 询问关键时间节点 |
| 预后判断型 | 询问预后相关因素 |

## 高频考点速记表

### CSF鉴别表

| 疾病 | 外观 | 细胞 | 糖 | 蛋白 |

| 结核性脑膜炎 | 毛玻璃样 | 淋巴细胞↑ | ↓ | ↑ |
| 病毒性脑膜炎 | 清亮 | 淋巴细胞↑ | 正常 | 正常或轻度↑ |
