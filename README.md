# Medical Quiz Master - 医学quiz练习技能

> 医学知识Quiz练习技能，支持306西医综合考纲复习、执业医师考试、医学生自主学习

## 功能特性

- **Quiz出题系统** - 每次1道题，自适应难度（5档）
- **连续练习模式** - 解析后自动出下一题，无需手动继续
- **即时反馈机制** - 正误判断、详细解析、记忆技巧
- **学习进度跟踪** - 实时统计正确率、薄弱点分析
- **错题本系统** - 自动记录、遗忘曲线复习
- **高频考点速记表** - 23个科目速查表
- **文档导入** - 支持PDF/DOCX/PPTX导入

## 支持学科（23个）

### 基础医学（5科）
生理学、生物化学、病理学、解剖学、药理学

### 临床医学（16科）
内科学、外科学、神经病学、精神病学、儿科学、妇产科学、急诊医学、皮肤病学、眼科学、耳鼻咽喉科学、老年医学、康复医学、麻醉学、肿瘤学、预防医学

### 其他（5科）
诊断学、临床检验诊断学、临床医学人文精神、传染病学、寄生虫学、免疫学

## 快速开始

| 命令 | 功能 |
|------|------|
| `出题` | 开始练习 |
| `/连续` | 开启连续练习模式（解析后自动出下一题） |
| `/单题` | 切换回单题模式 |
| `/专项 生理学` | 专项练习 |
| `/难度 3` | 设置难度（1-5档） |
| `/题型 病例` | 病例分析题模式 |
| `/考纲306` | 306全真模拟 |
| `/统计` | 查看进度 |
| `/遗忘曲线` | 查看待复习错题 |

## 自适应难度系统

| 等级 | 标识 | 含义 |
|------|------|------|
| ★☆☆☆☆ | 基础记忆 | 病因、定义、分类 |
| ★★☆☆☆ | 理解应用 | 机制、鉴别、推理 |
| ★★★☆☆ | 临床综合 | 诊断、治疗、预后 |
| ★★★★☆ | 复杂病例 | 多系统整合、复杂鉴别 |
| ★★★★★ | 考前冲刺 | 真题难度、综合应用 |

**升降规则**：连续答对2题升一档，连续答错2题降一档
