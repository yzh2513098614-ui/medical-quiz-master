## 一、A型选择题（单选题）

### 格式规范

```
## 📝 第X题（A型题 - [学科]）

[题干描述]

A. [选项1]
B. [选项2]
C. [选项3]
D. [选项4]

**请选择正确答案：**
```

### 特征
- **最佳选择题**：五选一，考察最佳答案
- **否定型**：题干中出现"错误的是"、"不包括"等词
- **定位型**：常考察"最常见"、"首选"、"特征性"等

### 示例

```
## 📝 第1题（A型题 - 生理学）

心室肌动作电位的平台期主要与哪种离子跨膜流动有关？

A. Na+内流
B. K+外流
C. Ca2+内流与K+外流同时存在
D. Cl-内流
E. Mg2+内流

**正确答案**：C
```

```
## 📝 第2题（A型题 - 内科学）[否定型]

关于急性心肌梗死的治疗，以下哪项是错误的？

A. 早期溶栓治疗
B. 绝对卧床休息
C. 立即使用强心苷
D. 吸氧监护
E. 抗血小板治疗

**正确答案**：C

**解析**：急性心肌梗死禁用强心苷，因其会增加心肌耗氧量，加重心肌缺血。
```

## 二、B型选择题（配伍型/共用选项型）

### 格式规范

```
## 📝 第X-X题（B型题 - [学科]）

**答题说明**：以下问题提供了共用的选项，请根据描述选择最匹配的答案。

[A. 选项1]
[B. 选项2]
[C. 选项3]
[D. 选项4]
[E. 选项5]

---

**问题X**： [描述1]
**问题X+1**： [描述2]
**问题X+2**： [描述3]

**请依次回答**：问题X: ___ 问题X+1: ___ 问题X+2: ___
```

### 特征
- **共用选项**：5个选项供2-5道题共用
- **一一对应**：每题只能选择一次
- **考察共性**：常考察相似易混的知识点

### 示例

```
## 📝 第3-5题（B型题 - 生物化学）

[A. 乳酸脱氢酶]
[B. 丙酮酸脱氢酶复合体]
[C. 柠檬酸合酶]
[D. 异柠檬酸脱氢酶]
[E. 琥珀酸脱氢酶]

**问题3**：糖酵解中唯一以NAD+作为辅酶的酶是？
**问题4**：三羧酸循环的第一个酶是？
**问题5**：受ATP/ADP调节的限速酶是？

**正确答案**：3-A  4-C  5-D

**解析**：
- 乳酸脱氢酶催化丙酮酸→乳酸，以NADH为辅酶
- 柠檬酸合酶催化乙酰CoA+草酰乙酸→柠檬酸，是三羧酸循环第一步
- 异柠檬酸脱氢酶是三羧酸循环的主要限速酶，受ATP/ADP调节
```

## 三、X型选择题（多选题）

### 格式规范

```
## 📝 第X题（X型题 - [学科]）

[题干描述]

A. [选项1]
B. [选项2]
C. [选项3]
D. [选项4]
E. [选项5]

**请选择所有正确答案**（少选、多选、错选均不得分：
```

### 特征
- **五选多**：2-5个正确答案
- **全对才得分**：漏选、少选、多选、错选均不得分
- **常考类型**：适应证、禁忌证、并发症、特点

### 示例

```
## 📝 第6题（X型题 - 病理学）

关于癌前病变的描述，正确的有：

A. 黏膜白斑可能发展为鳞状细胞癌
B. 慢性萎缩性胃炎可能发展为胃癌
C. 结肠多发性息肉是一种癌前病变
D. 所有癌前病变都会发展为癌症
E. 癌前病变是良性病变但具有恶变潜能

**正确答案**：ABCE

**解析**：
- 癌前病变是统计学概念，并非所有都会发展为癌
- 常见癌前病变：黏膜白斑、慢性萎缩性胃炎、结肠多发性息肉、乳腺囊性增生症
```

## 四、简答题

### 格式规范

```
## 📝 第X题（简答题 - [学科]）

**题目**：[问题描述]

---

**答题要点**：
1. [要点1]
2. [要点2]
3. [要点3]
...

**参考答案**：（见解析区）

**请简要回答：**
```

### 答题要点模板

| 要素 | 说明 |
|------|------|
| **定义** | 是什么/主要特征 |
| **机制** | 怎么发生/原理 |
| **特点** | 重要特征/分类 |
| **临床意义** | 应用/注意事项 |

### 评判标准

| 等级 | 分值 | 标准 |
|------|------|------|
| 优秀 | 90-100% | 要点完整、表述准确、逻辑清晰 |
| 良好 | 75-89% | 要点基本完整、表述较准确 |
| 中等 | 60-74% | 要点有遗漏、表述基本正确 |
| 及格 | 45-59% | 要点明显遗漏、部分表述错误 |
| 不及格 | <45% | 要点严重缺失、概念错误 |

### 示例

```
## 📝 第7题（简答题 - 生理学）

**题目**：请简述心室肌动作电位的分期及特点。

---

**答题要点**：
1. **0期（快速去极化）**：Na+内流，膜电位-90mV→+30mV
2. **1期（快速复极化）**：K+外流，膜电位迅速下降
3. **2期（平台期）**：Ca2+内流与K+外流并存，动作电位呈平台状
4. **3期（快速复极化）**：Ca2+通道关闭，K+外流加速
5. **4期（静息期）**：Na+-K+泵、Na+-Ca2+交换体恢复离子分布

**请简要回答：**
```

## 五、病例分析题

### 格式规范

```
## 📝 第X题（病例分析题 - 内科学）

**问题1**：该患者最可能的诊断是什么？诊断依据有哪些？

**问题2**：需要与哪些疾病鉴别？鉴别要点是什么？

**问题3**：首选的辅助检查是什么？

**问题4**：治疗原则是什么？

```

### 五维评分维度

| 维度 | 权重 | 考察要点 |
|------|------|---------|
| **诊断** | 30% | 病因诊断、部位诊断、性质诊断 |
| **依据** | 25% | 病史特点、体征意义、辅助检查结果运用 |
| **鉴别** | 15% | 是否考虑相关鉴别诊断、鉴别要点 |
| **检查** | 15% | 首选检查选择、结果解读 |
| **治疗** | 15% | 治疗原则、紧急处理、用药方案 |

### 示例

```
## 📝 第8题（病例分析题 - 内科学）

**问题1**：该患者最可能的诊断是什么？诊断依据有哪些？

**问题2**：需要与哪些疾病鉴别？

**问题3**：首选的辅助检查是什么？

**问题4**：急诊应采取哪些治疗措施？

**请依次回答：**
```

## 六、论述题

### 格式规范

```
## 📝 第X题（论述题 - [学科]）

**题目**：[综合性问题]

---

**答题框架**（供AI评判参考）：
一、[第一部分主题]
    1. [要点1]
    2. [要点2]
    ...

二、[第二部分主题]
    1. [要点1]
    2. [要点2]
    ...

三、[第三部分主题]
    1. [要点1]
    2. [要点2]
    ...

**参考答案**：（见解析区）

**请写出你的论述：**
```

### 答题框架模板

| 框架类型 | 适用主题 | 结构 |
|----------|----------|------|
| **三段式** | 机制/过程类 | 发生→发展→结局 |
| **对比式** | 鉴别/比较类 | 异同点对比 |
| **层次式** | 系统类 | 从浅到深/从局部到整体 |
| **循环式** | 生理/代谢类 | 循环通路各环节 |

### 五级评分等级

| 等级 | 分值范围 | 特征描述 |
|------|----------|----------|
| **优秀** | 90-100% | 框架完整、逻辑清晰、要点全面、论述深入、有个人见解 |
| **良好** | 75-89% | 框架完整、要点基本全面、论述较深入 |
| **中等** | 60-74% | 框架基本完整、要点有遗漏、论述一般 |
| **及格** | 45-59% | 框架不完整、要点明显遗漏、论述浅显 |
| **不及格** | <45% | 框架缺失、要点严重遗漏、概念错误 |

### 示例

```
## 📝 第9题（论述题 - 病理学）

**题目**：试述急性炎症的病理变化过程及结局。

---

**答题框架**：
一、急性炎症的病理变化
    1. 变质（变性和坏死）
    2. 渗出（血管反应、液体渗出、细胞渗出）
    3. 增生（实质细胞增生、间质细胞增生）

二、急性炎症的结局
    1. 痊愈（完全痊愈、不完全痊愈）
    2. 迁延为慢性炎症
    3. 蔓延播散（局部蔓延、淋巴道扩散、血道扩散）

三、影响炎症的因素
    1. 致病因素
    2. 机体免疫状态
    3. 治疗措施

**请写出你的论述：**
```
