## 各科目考查内容

### 一、生理学（约14%）

| 章节 | 考查要点 | 高频考点 |
|------|----------|----------|
| 1. 细胞的基本功能 | 跨膜物质转运、信号转导、细胞电活动、神经-肌接头、肌肉收缩 | 受体类型、动作电位、兴奋传导 |
| 2. 血液 | 血液组成、血细胞功能、生理止血、凝血抗凝、血型系统 | 凝血因子、血型、纤溶系统 |
| 3. 血液循环 | 心肌电生理、泵血功能、动脉血压、调节机制、冠脉脑循环 | 心电图、心输出量、调节 |
| 4. 呼吸 | 肺通气、肺换气、气体运输、呼吸运动调节 | 肺活量、气体交换、呼吸控制 |
| 5. 消化和吸收 | 消化液分泌、胃肠运动、吸收机制 | 胰液胆汁调节、胃排空 |
| 6. 能量代谢与体温 | 能量代谢测定、产热散热、体温调节 | 基础代谢率、呼吸商 |
| 7. 尿的生成和排出 | 肾小球滤过、肾小管重吸收、尿液浓缩稀释、调节 | 肾小球滤过率、清除率 |
| 8. 感觉器官 | 感受器特性、视觉听觉前庭功能 | 视力听力原理 |
| 9. 神经系统 | 神经元、突触传递、递质、感觉运动调节、内脏调节、脑电睡眠 | 突触传递、反射中枢 |
| 10. 内分泌 | 激素机制、下丘脑-垂体、甲状腺、肾上腺、胰岛 | 激素调节轴、重要激素 |
| 11. 生殖 | 睾丸卵巢功能、月经周期、妊娠分娩 | 月经周期调控 |

### 二、生物化学（约12%）

| 章节 | 考查要点 | 高频考点 |
|------|----------|----------|
| 1. 蛋白质的结构与功能 | 氨基酸分类、蛋白质结构、性质、分离纯化 | 20种氨基酸、蛋白质变性 |
| 2. 核酸的结构与功能 | 核苷酸组成、DNA双螺旋、RNA类型 | 碱基配对、核酸变性复性 |
| 3. 酶 | 酶结构、作用机制、动力学、抑制、调节 | 米氏方程、酶抑制类型 |
| 4. 糖代谢 | 糖酵解、有氧氧化、糖原代谢、糖异生、磷酸戊糖途径 | 关键酶、能量计算 |
| 5. 脂类代谢 | β氧化、合成、磷脂胆固醇代谢 | 脂肪酸代谢、酮体 |
| 6. 生物氧化 | 呼吸链、ATP生成、氧化磷酸化 | 呼吸链复合物、ATP合酶 |
| 7. 氨基酸代谢 | 脱氨基、转氨、尿素循环、一碳单位 | 尿素循环、氨基酸脱羧 |
| 8. 核苷酸代谢 | 嘌呤嘧啶合成与分解 | 痛风、药物作用靶点 |
| 9. 基因信息传递 | DNA复制、RNA转录、蛋白质合成 | 复制/转录/翻译关键酶 |
| 10. 基因表达调控 | 原核真核调控、基因工程 | 操纵子、表观遗传 |
| 11. 信号转导 | 受体类型、信号通路 | cAMP/PKC/IP3通路 |
| 12. 肝生物化学 | 肝代谢功能、生物转化 | 黄疸鉴别 |
| 13. 维生素 | 维生素分类、功能、缺乏症 | B族维生素辅酶 |

### 三、病理学（约12%）

| 章节 | 考查要点 | 高频考点 |
|------|----------|----------|
| 1. 细胞组织损伤与修复 | 变性、坏死、凋亡、适应、再生、肉芽组织 | 细胞水肿、坏死类型 |
| 2. 局部血液循环障碍 | 充血、出血、血栓形成、栓塞、梗死 | 血栓形成条件、梗死类型 |
| 3. 炎症 | 病理变化、介质、类型、结局 | 急性慢性炎症区别 |
| 4. 肿瘤 | 异型性、生长扩散、命名分类、良性恶性区别 | 良恶性鉴别、转移 |
| 5. 心血管疾病 | 风湿病、动脉粥样硬化、高血压病、心肌病 | 粥样硬化斑块、风湿小结 |
| 6. 呼吸系统疾病 | 肺炎、慢支、肺气肿、硅肺、肺癌 | 肺炎分类、肺癌类型 |
| 7. 消化系统疾病 | 胃炎溃疡、肝炎肝硬化、消化道肿瘤 | 肝硬化假小叶 |
| 8. 泌尿系统疾病 | 肾小球肾炎、肾盂肾炎、肾肿瘤 | 肾小球疾病病理类型 |
| 9. 乳腺及女性生殖 | 乳腺癌、宫颈癌、卵巢肿瘤 | 乳腺癌分类 |
| 10. 传染病寄生虫病 | 结核病、流脑乙脑、伤寒痢疾、血吸虫病 | 结核病病理特点 |
| 11. 性传播疾病 | 梅毒、艾滋病 | 病理变化 |
| 12. 免疫病理 | 变态反应、移植排斥、自身免疫病、免疫缺陷 | 四型超敏反应 |

## 复习建议

### 基础学科（生理/生化/病理）备考策略
- 生理：理解机制为主，配合图表记忆
- 生化：代谢途径是关键，画图串联
- 病理：形态学为基础，结合临床疾病

### 临床学科（内科/外科）备考策略
- 内科：按系统学习，串联诊断与治疗
- 外科：掌握手术指征、围手术期管理
- 重视病例分析题的训练

### 题型分布与时间分配
- A型题（1-115）：165题，约占60%，需快速准确
- B型题（116-135）：20题，约占11%，注意配伍关系
- X型题（136-165）：30题，约占22%，多选少选均不得分
