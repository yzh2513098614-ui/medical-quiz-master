## 第一篇 运动系统

### 核心概念

| 要素 | 内容 |
|------|------|
| **骨** | 206块骨（颅骨29块+躯干骨51块+四肢骨126块），骨质（骨密质+骨松质） |
| **关节** | 关节面、关节囊（外层纤维膜+内层滑膜）、关节腔、辅助结构（韧带/半月板/滑膜皱襞） |
| **骨骼肌** | 肌腹（收缩功能）、肌腱（传寻张力）、腱鞘（保护肌腱） |

### 高频考点

**骨的构造：**
- 骨膜：含成骨细胞和破骨细胞，参与骨的生长和修复
- 骨髓：红骨髓（造血）、黄骨髓（脂肪），5岁后长骨骨干为黄骨髓

**重要骨性标志：**

| 部位 | 骨性标志 | 临床意义 |
|------|----------|----------|
| 头部 | 枕外隆凸、乳突、颧弓、下颌角 | 定位标志 |
| 颈部 | 舌骨、甲状软骨（C4）、环状软骨（C6）、颈动脉结节 | 气管切开位 |
| 胸部 | 胸骨角（第2肋）、肩胛冈、肩胛下角 | 计数肋骨 |
| 背部 | 棘突（第7颈椎）、肩胛冈 | 定位标志 |
| 上肢 | 肱骨大结节、肱骨内上髁、尺骨鹰嘴 | 肌肉附着 |
| 下肢 | 髂嵴、髂前上棘、坐骨结节、股骨大转子、胫骨粗隆 | 定位标志 |

**颅骨：**
- 脑颅骨（8块）：额骨、顶骨、枕骨、颞骨、蝶骨、筛骨
- 面颅骨（15块）：上颌骨、鼻骨、泪骨、颧骨、下鼻甲、腭骨、舌骨、下颌骨
- 翼点：额骨、顶骨、蝶骨、颞骨交汇处，内面有脑膜中动脉通过

**脊柱：**
- 颈椎（7块）：C1寰椎、C2枢椎、C3-7典型颈椎，椎孔大呈三角形
- 胸椎（12块）：有肋凹，棘突长向后下
- 腰椎（5块）：椎体最大，棘突水平后伸
- 椎间盘：纤维环+髓核，颈部/腰部好发椎间盘突出

**上肢骨：**
- 肱骨：外科颈（易骨折）、桡神经沟（桡神经损伤）、内上髁（尺神经沟）
- 尺骨：鹰嘴、滑车切迹、桡切迹
- 桡骨：桡骨头、桡骨粗隆

**下肢骨：**
- 股骨：股骨头、股骨颈（易骨折）、大转子、小转子
- 髌骨：人体最大的籽骨
- 胫骨：内侧髁、外侧髁、胫骨粗隆
- 腓骨：腓骨头、外踝

**重要关节：**

| 关节 | 构成 | 结构特点 |
|------|------|----------|
| 肩关节 | 肱骨头+肩胛骨关节盂 | 球窝关节，最灵活，盂唇加深，韧带薄弱 |
| 肘关节 | 肱尺/肱桡/桡尺近侧关节 | 屈戌关节，韧带强，稳定性好 |
| 腕关节 | 桡骨远端+腕骨 | 椭圆关节 |
| 髋关节 | 股骨头+髋臼 | 杵臼关节，关节囊厚紧，稳定性好 |
| 膝关节 | 股骨髁+胫骨平台+髌骨 | 复杂关节，半月板、交叉韧带、侧副韧带 |
| 踝关节 | 胫腓骨远端+距骨 | 屈戌关节，韧带强 |

### 记忆口诀

> **颅骨构成**：额筛蝶颞各一块，顶枕再加两块颞，舌骨下颌不成对
> **脊柱组成**：颈椎七来胸椎十二腰五，荐一尾一不独立
> **关节组成**：关节结构有特点，关节面软骨关节囊，滑膜分泌关节液

## 第二篇 内脏学

### 核心概念

| 系统 | 组成器官 | 主要功能 |
|------|----------|----------|
| **消化系统** | 口腔、咽、食管、胃、小肠（十二指肠/空肠/回肠）、大肠（盲肠/阑尾/结肠/直肠/肛管）、肝、胰、涎腺 | 消化食物、吸收营养 |
| **呼吸系统** | 鼻、咽、喉、气管、支气管、肺 | 气体交换 |
| **泌尿系统** | 肾、输尿管、膀胱、尿道 | 排泄代谢废物 |
| **生殖系统** | 睾丸、附睾、输精管、前列腺/卵巢、输卵管、子宫、阴道 | 繁殖后代 |

### 高频考点

**消化管：**

| 部位 | 长度/特点 | 重要考点 |
|------|-----------|----------|
| 食管 | 25cm，3个狭窄（环状软骨/左主支气管/膈孔） | 狭窄距中切牙15/25/40cm |
| 胃 | 贲门、幽门（幽门瓣），分为底、体、窦 | 幽门螺杆菌好发部位 |
| 十二指肠 | 25-30cm，分为球部、降部、水平部、升部 | 十二指肠大乳头（胆总管/胰管开口） |
| 回肠 | 远端有集合淋巴结（Peyer斑） | Meckel憩室（先天畸形） |
| 阑尾 | 根部位子髂窝内，McBurney点 | 阑尾炎压痛点 |

**肝：**
- 脏面：H沟（肝门+横沟+左纵沟+右纵沟）
- 肝门：肝固有动脉左右支、肝左右管、肝门静脉左右支
- 分叶：肝左叶、肝右叶、尾状叶、方叶
- 肝血供：门静脉（75%）+肝动脉（25%）
- 胆汁分泌：肝细胞分泌→胆小管→小叶间胆管→左右肝管→肝总管

**胰：**
- 头、颈、体、尾四部分
- 胰管：与胆总管汇合成肝胰壶腹（Vater壶腹）
- 外分泌：胰液（含胰蛋白酶/胰脂肪酶/胰淀粉酶）
- 内分泌：胰岛（A细胞分泌胰高血糖素、B细胞分泌胰岛素）

**呼吸系统：**

| 结构 | 特点 |
|------|------|
| 鼻 | 外鼻、鼻腔（鼻甲）、鼻窦（上颌窦/额窦/筛窦/蝶窦） |
| 喉 | 软骨（甲状/环状/会厌）、声带、喉室 |
| 气管 | 16-20个C形软骨环，后壁为平滑肌 |
| 支气管 | 右主支气管（粗短垂直）、左主支气管（细长水平） |
| 肺 | 左2右3叶，肺门（含支气管/肺动脉/肺静脉/神经/淋巴） |

**肺下界：** 锁骨中线第6肋、腋中线第8肋、肩胛线第10肋

**泌尿系统：**

| 器官 | 位置/形态 | 考点 |
|------|-----------|------|
| 肾 | T12-L3，右肾略低，肾门约L1 | 肾门：肾动脉、肾静脉、肾盂、神经、淋巴 |
| 输尿管 | 3个狭窄（肾盂输尿管移行处、跨越髂血管处、膀胱壁内段） | 结石易嵌顿部位 |
| 膀胱 | 空虚时呈三棱锥体，充盈时呈球形 | 膀胱三角（肿瘤好发部位） |
| 尿道 | 男性16-22cm（前列腺部/膜部/海绵体部）；女性3-5cm | 女性易患尿路感染 |

### 记忆口诀

> **食管三狭窄**：环软、主支、穿膈处，结石嵌顿要注意
> **肝门结构**：左右肝管左右动脉，门静脉支最粗
> **肺下界**：锁六腋八肩胛十
> **输尿管三狭**：上中下三狭窄，上与肾盂连，下入膀胱壁
> **膀胱三角**：膀胱三角是肿瘤好发，输尿管口在两角

## 第三篇 脉管系统

### 核心概念

| 系统 | 组成 | 功能 |
|------|------|------|
| **心血管系统** | 心（4腔）、动脉、毛细血管、静脉 | 运输血液 |
| **淋巴系统** | 淋巴管道、淋巴器官（淋巴结/脾/胸腺/扁桃体）、淋巴组织 | 免疫、协助回流 |

### 高频考点

**心的位置与外形：**
- 位置：胸腔纵隔，约2/3在正中线左侧
- 外形：心尖（向左下）、心底（大部分由左心房构成）、两面（胸肋面+膈面）、三缘（左缘/右缘/下缘）
- 心腔：右心房（卵圆窝）、右心室（肺动脉瓣）、左心房（肺静脉入口）、左心室（主动脉瓣）

**心的传导系统：**
窦房结→结间束→房室结→希氏束→左/右束支→浦肯野纤维

**重要动脉：**

| 动脉 | 行程 | 重要分支/分布 |
|------|------|--------------|
| 颈总动脉 | 颈侧，甲状腺上动脉分叉处有颈动脉窦（压力感受器）和颈动脉小球（化学感受器） | 颈内动脉（颅内）、颈外动脉（面部） |
| 锁骨下动脉 | 穿斜角肌间隙 | 椎动脉（颅内）、胸廓内动脉 |
| 腋动脉 | 腋窝 | 旋肱前/后动脉、肱动脉 |
| 肱动脉 | 肱二头肌内侧 | 桡动脉、尺动脉 |
| 股动脉 | 股静脉外侧 | 股深动脉（旋股内/外动脉） |
| 腘动脉 | 膝关节后方 | 胫前/后动脉 |

**腹主动脉不成对脏支：**
- 腹腔干（腹腔干）：胃左动脉、肝总动脉、脾动脉
- 肠系膜上动脉：空肠/回肠动脉、右结肠动脉、中结肠动脉
- 肠系膜下动脉：左结肠动脉、乙状结肠动脉、直肠上动脉

**肝门静脉组成：** 脾静脉+肠系膜上静脉（有时加肠系膜下静脉）
- 属支：胃左静脉、胃右静脉、脐周静脉、胆囊静脉
- 侧支循环：食管下段、直肠、脐周、前腹壁

**静脉：**

| 静脉 | 收集范围 | 特点 |
|------|----------|------|
| 头臂静脉 | 头颈上肢 | 奇静脉汇入 |
| 上腔静脉 | 头颈上肢胸壁 | 由左右头臂静脉合成 |
| 下腔静脉 | 下肢盆部腹部 | 由左右髂总静脉合成 |
| 大隐静脉 | 下肢浅层 | 内踝前方上行，注入股静脉 |
| 小隐静脉 | 下肢浅层 | 外踝后方上行，注入腘静脉 |

**淋巴系统：**
- 胸导管：左腰干+左支气管干+左锁骨下干+左颈干（L1水平）
- 右淋巴导管：右支气管干+右锁骨下干+右颈干

### 记忆口诀

> **心传导**：窦房结是起搏点，结间束到房室结，希氏束分左右束，浦肯野纤维到心肌
> **腹腔干分支**：肝胃脾，腹腔干三分支
> **肝门静脉侧支**：食管下段直肠脐周，前腹壁三途径
> **大隐静脉**：内踝前方上行，注入股静脉大隐

## 第四篇 感觉器

### 核心概念

| 感觉器 | 结构 | 功能 |
|--------|------|------|
| **视器** | 眼球（外中内纤维膜/血管膜/视网膜）、眼副器 | 视觉 |
| **前庭蜗器** | 外耳（耳廓/外耳道/鼓膜）、中耳（鼓室/咽鼓管/乳突窦/乳突）、内耳（半规管/前庭/耳蜗） | 听觉+平衡觉 |

### 高频考点

**眼球结构：**

| 层次 | 结构 |
|------|------|
| 纤维膜 | 角膜（透明，无血管，营养来自房水）、巩膜（白色） |
| 血管膜 | 虹膜（有瞳孔，括约肌/开大肌）、睫状体（分泌房水）、脉络膜 |
| 视网膜 | 视部（感光：视锥细胞辨色、视杆细胞暗视觉）、盲部、视神经盘（生理盲点） |

**房水循环：** 睫状体产生→后房→瞳孔→前房→虹膜角膜角→小梁网→Schlemm管→眼静脉

**中耳：**
- 鼓室：鼓膜内侧含3块听小骨（锤骨/砧骨/镫骨）
- 咽鼓管：连通鼓室与鼻咽部，维持鼓膜两侧气压平衡
- 乳突窦：与鼓室相通

**内耳迷路：**
- 骨迷路：半规管（3个，互成直角）、前庭、耳蜗（2.5圈）
- 膜迷路：壶腹嵴（旋转加速度）、椭圆囊斑/球囊斑（直线加速度）、蜗管（柯蒂器/螺旋器，感声）

### 记忆口诀

> **眼球三层**：外纤中血内感光，角膜透明巩白完
> **房水循环**：睫状体产房水，后房瞳孔前房角，小梁网入Schlemm管
> **听觉传导**：声波入耳鼓膜振，锤砧镫骨放大传，蜗管柯蒂感声音

## 第五篇 神经系统

### 核心概念

| 分类 | 组成 | 功能 |
|------|------|------|
| **中枢神经** | 脑（端脑/间脑/中脑/脑桥/延髓/小脑）、脊髓 | 传导/整合/反射 |
| **周围神经** | 脑神经（12对）、脊神经（31对）、植物神经（交感/副交感） | 传入/传出 |

### 高频考点

**脑：**

| 部位 | 组成 | 功能 |
|------|------|------|
| 端脑 | 左右大脑半球，胼胝体连接 | 躯体/内脏运动/感觉/语言/情绪 |
| 间脑 | 丘脑（感觉传导中继）、下丘脑（内脏/内分泌/体温调节） | 感觉中继/内脏调节 |
| 中脑 | 大脑脚/四叠体/导水管 | 瞳孔对光反射/视听力反射 |
| 脑桥 | 脑桥基底部/被盖部 | 呼吸调节/锥体系/面神经 |
| 延髓 | 锥体/橄榄/薄束结节/楔束结节 | 生命中枢（心跳/呼吸/血管运动） |
| 小脑 | 蚓部/半球 | 维持平衡/调节肌张力/协调运动 |

**脊髓：**
- 位置：椎管内，T12-L1成人
- 31个节段：颈8、胸12、腰5、骶5、尾1
- 灰质：H形，前角（运动神经元）、后角（感觉神经元）、侧角（交感节前神经元）
- 白质：上行（薄束/楔束/脊髓丘脑束）、下行（皮质脊髓束/红核脊髓束）

**脊神经：**

| 神经 | 组成 | 分布 |
|------|------|------|
| 颈丛 | C1-4 | 膈神经（膈肌）、枕小/耳大/颈横/锁骨上神经 |
| 臂丛 | C5-8、T1 | 正中/尺/桡/肌皮/腋神经 |
| 胸神经 | T1-12 | 肋间神经 |
| 腰丛 | T12、L1-4 | 股/闭孔/股外侧皮神经 |
| 骶丛 | L4-5、S1-5 | 坐骨神经（腓总/胫神经） |

**臂丛主要分支：**

| 神经 | 损伤表现 |
|------|----------|
| 正中神经 | 猿手（鱼际肌萎缩+拇指对掌障碍） |
| 尺神经 | 爪形手（小鱼际肌+骨间肌萎缩） |
| 桡神经 | 垂腕（伸腕伸指障碍） |
| 肌皮神经 | 屈肘障碍（肱二头肌） |
| 腋神经 | 方肩（三角肌萎缩） |

**脑神经：**

| 脑神经 | 性质 | 核团 | 分布/功能 |
|--------|------|------|------------|
| Ⅰ嗅神经 | 感觉性 | 嗅球 | 鼻腔嗅黏膜 |
| Ⅱ视神经 | 感觉性 | 外侧膝状体 | 视网膜 |
| Ⅲ动眼神经 | 运动性 | 动眼神经核/副交感核 | 眼外肌（上/内/下直肌、下斜肌、上睑提肌）、瞳孔括约肌/睫状肌 |
| Ⅳ滑车神经 | 运动性 | 滑车神经核 | 上斜肌 |
| Ⅴ三叉神经 | 混合性 | 三叉神经感觉核/运动核 | 面部感觉、咀嚼肌 |
| Ⅵ展神经 | 运动性 | 展神经核 | 外直肌 |
| Ⅶ面神经 | 混合性 | 面神经核/上泌涎核/孤束核 | 面部表情肌、味觉（舌前2/3）、泪腺/下颌下腺/舌下腺分泌 |
| Ⅷ前庭蜗神经 | 感觉性 | 前庭神经核/蜗神经核 | 前庭（平衡）、耳蜗（听觉） |
| Ⅸ舌咽神经 | 混合性 | 疑核/下泌涎核/孤束核 | 咽肌、咽部感觉、腮腺分泌 |
| Ⅹ迷走神经 | 混合性 | 迷走神经背核/疑核/孤束核 | 喉肌/咽喉感觉、内脏运动（副交感） |
| Ⅺ副神经 | 运动性 | 副神经核 | 胸锁乳突肌/斜方肌 |
| Ⅻ舌下神经 | 运动性 | 舌下神经核 | 舌肌 |

**内脏神经（植物神经）：**

| 类型 | 起源 | 节前神经元 | 节后神经元 | 主要功能 |
|------|------|-----------|-----------|----------|
| 交感神经 | 胸髓（T1-L3）侧角 | 短轴突 | 长轴突 | 应激状态：瞳孔散大/心跳加快/胃肠蠕动减慢/支气管扩张/血糖↑ |
| 副交感神经 | 脑干（S2-4） | 长轴突 | 短轴突 | 静息状态：瞳孔缩小/心跳减慢/胃肠蠕动↑/支气管收缩/唾液分泌↑ |

**锥体系与锥体外系：**
- 锥体系：皮质脊髓束（支配四肢肌）+皮质核束（支配头面部）
- 锥体外系：纹状体（帕金森病）、小脑（共济失调）

### 记忆口诀

> **臂丛分支**：正中神经猿人手，尺神经来爪形手，桡神经是垂腕症，肌皮屈肘不能，腋神经方肩
> **脑神经记忆**：一嗅二视三动眼，四滑五叉六外展，七面八听九舌咽，迷副舌下十二全
> **交感神经**：交感兴奋应激态，瞳孔散大心跳快，胃肠抑制支气管缩，血糖升高应急
> **副交感神经**：副交感静息态，瞳孔缩小心跳慢，胃肠活跃支气管缩，唾液分泌旺盛

## 第六篇 内分泌系统

### 核心概念

| 内分泌腺 | 位置 | 分泌激素 |
|----------|------|----------|
| **垂体** | 蝶鞍垂体窝，分为腺垂体（远侧部+结节部+中间部）和神经垂体（神经部+漏斗） | GH、PRL、TSH、ACTH、FSH、LH（腺垂体）；ADH、OT（神经垂体） |
| **甲状腺** | 喉前外侧，C2-7 | T3、T4、CT |
| **甲状旁腺** | 甲状腺背面，4个 | PTH（升血钙） |
| **肾上腺** | 肾上级，分为皮质（球状带/束状带/网状带）和髓质 | 皮质：醛固酮/皮质醇/性激素；髓质：Ad/NE |
| **胰岛** | 胰腺内 | α细胞（胰高血糖素）；β细胞（胰岛素） |
| **松果体** | 间脑顶部 | 褪黑素 |
| **胸腺** | 前纵隔 | 胸腺素 |

### 高频考点

**垂体分泌的激素：**

| 激素 | 作用 |
|------|------|
| 生长激素（GH） | 促进生长发育（尤其骨骼），升血糖 |
| 催乳素（PRL） | 促进乳腺发育和乳汁分泌 |
| 促甲状腺激素（TSH） | 促进甲状腺分泌T3/T4 |
| 促肾上腺皮质激素（ACTH） | 促进肾上腺皮质分泌皮质醇 |
| 促性腺激素（FSH/LH） | 促进性腺发育和性激素分泌 |

**甲状腺激素：**
- T4（甲状腺素）：量大、活性弱
- T3：量小、活性强
- 功能：促进新陈代谢、生长发育、提高神经系统兴奋性
- 临床：甲亢（代谢↑、心悸、多汗、消瘦）；甲减（代谢↓、嗜睡、粘液性水肿）

**甲状旁腺激素（PTH）：** 升血钙、降血磷

**肾上腺皮质激素：**

| 皮质区 | 分泌激素 | 主要功能 |
|--------|----------|----------|
| 球状带 | 醛固酮（盐皮质激素） | 保Na+排K+、水潴留 |
| 束状带 | 皮质醇（糖皮质激素） | 升血糖、抗炎、抗过敏、免疫抑制 |
| 网状带 | 性激素（雄激素） | 影响性征 |

**肾上腺髓质激素：** 肾上腺素（Ad）+去甲肾上腺素（NE）

**胰岛素与胰高血糖素：**

| 激素 | 来源 | 作用 |
|------|------|------|
| 胰岛素 | β细胞 | 降血糖、促进糖原/脂肪/蛋白质合成 |
| 胰高血糖素 | α细胞 | 升血糖、促进糖原分解/糖异生 |

### 记忆口诀

> **垂体激素**：促甲促肾促性腺，生长催乳各自忙
> **甲状腺激素**：T3活性强T4量多，甲亢高代谢甲减低代谢
> **肾上腺皮质**：球状带醛固酮，束状带皮质醇，网状带是性激素
> **PTH作用**：升钙降磷保骨骼
