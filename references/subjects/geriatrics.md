# 老年医学速查表

## 老年疾病特点

| 特点 | 说明 |
|------|------|
| 多病共存 | 老年人同时患有2种或以上慢性疾病 |
| 临床表现不典型 | 症状轻微或不典型，阳性体征少 |
| 多器官功能减退 | 肝、肾、心、肺等功能储备下降 |
| 用药特殊性 | 药代动力学改变，多药相互作用 |
| 老年人用药原则 | 小剂量开始、简化用药、定期评估 |

### 老年人用药原则

1. **小剂量开始** - 从最低有效剂量的1/3~1/2开始
2. **简化用药** - 优先选择长效制剂，减少服药次数
3. **个体化给药** - 根据肝肾功能调整剂量
4. **定期评估** - 定期审查用药方案，撤除不必要的药物
5. **注意相互作用** - 警惕多药联用的相互作用

## 心血管系统

### 老年高血压

**诊断要点：**
- 单纯收缩期高血压多见（收缩压≥140mmHg，舒张压<90mmHg）
- 脉压差增大
- 血压波动大，易发生体位性低血压
- 假性高血压常见（肱动脉硬化致袖带测压偏高）

**治疗原则：**
- 目标血压<150/90mmHg（适度宽松）
- 优先选用长效CCB（如氨氯地平）
- ACEI/ARB可用于合并糖尿病或心衰者
- 避免过度降压，警惕体位性低血压

### 冠心病

**诊断要点：**
- 无症状心肌缺血多见（无痛性心梗）
- 症状不典型，可仅表现为呼吸困难、乏力
- 心电图不典型，酶学变化延迟

**治疗原则：**
- 抗血小板聚集（阿司匹林）
- 调脂治疗（他汀类，目标LDL-C<1.8mmol/L）
- 硝酸酯类缓解心绞痛
- 必要时血运重建

### 心力衰竭

**诊断要点：**
- 射血分数保留的心衰（HFpEF）多见
- 劳力性呼吸困难、夜间阵发性呼吸困难
- 双下肢水肿、肺部湿啰音
- NT-proBNP升高

**治疗原则：**
- 利尿剂消除水肿
- ACEI/ARB/ARNI改善心室重构
- β受体阻滞剂（稳定后使用）
- 醛固酮受体拮抗剂
- HFpEF：治疗基础病（高血压、糖尿病、房颤）

### 心房颤动

**诊断要点：**
- 心悸、乏力、头晕、胸闷
- 脉搏短绌、心音强弱不等
- 心电图：P波消失，f波代之，RR间期不等

**治疗原则：**
- **抗凝治疗**（CHA₂DS₂-VASc评分≥2分需抗凝）
- 率控制：β受体阻滞剂/地尔硫卓/地高辛
- 律控制：胺碘酮（必要时电复律）
- 抗凝药选择：华法林、达比加群、利伐沙班

## 呼吸系统

### 老年肺炎

**诊断要点：**
- 症状不典型，咳痰不明显
- 可仅表现为精神萎靡、食欲不振
- 发热可不明显，白细胞升高不显著
- 肺部体征少，但影像学重

**治疗原则：**
- 经验性抗感染（覆盖常见病原体）
- 止咳、祛痰、对症支持
- 注意营养支持和补液
- 警惕呼吸衰竭

### 慢性阻塞性肺疾病（COPD）

**诊断要点：**
- 慢性咳嗽、咳痰、进行性呼吸困难
- 吸烟史
- 肺功能：FEV1/FVC<0.70

**治疗原则：**
- 支气管扩张剂（LAMA/LABA）
- ICS联合LABA/LAMA（频繁急性加重）
- 戒烟、氧疗、肺康复
- 疫苗接种（流感、肺炎疫苗）

### 睡眠呼吸暂停综合征

**诊断要点：**
- 夜间打鼾、呼吸暂停
- 白天嗜睡、头痛
- 多导睡眠图确诊（AHI≥5）

**治疗原则：**
- 减肥、侧卧位睡眠
- 持续正压通气（CPAP）
- 口腔矫治器
- 外科手术（部分病例）

## 内分泌系统

### 2型糖尿病

**诊断要点：**
- 餐后血糖升高为主（空腹血糖可正常）
- "三多一少"症状不典型
- 常合并肥胖、高血压、血脂异常

**治疗原则：**
- 二甲双胍为一线用药
- 个体化HbA1c目标（一般<7%，老年人可放宽至<8%）
- 注意低血糖风险（优选不易引起低血糖的药物）
- 关注血糖监测，避免血糖波动过大

### 甲状腺功能异常

**诊断要点：**

| 类型 | 特点 |
|------|------|
| 临床甲减 | 怕冷、嗜睡、便秘、胫前黏液水肿 |
| 亚临床甲减 | T4正常，仅TSH升高 |
| 甲亢 | 可表现为房颤、心悸（老年淡漠型甲亢） |

**治疗原则：**
- **甲减**：左甲状腺素钠替代，从小剂量开始
- **甲亢**：ATD治疗或放射性碘治疗

### 骨质疏松症

**诊断要点：**
- 腰背疼痛、身高缩短、驼背
- 易发生脆性骨折（椎体、髋部、腕部）
- 骨密度测定：T值≤-2.5

**治疗原则：**
- 钙剂+维生素D补充
- 抗骨吸收药：双膦酸盐、选择性雌激素受体调节剂
- 促骨形成药：特立帕肽（严重病例）
- 防跌倒宣教

## 神经系统

### 阿尔茨海默病

**诊断要点：**
- 进行性认知功能下降
- 记忆力障碍（近记忆受损为主）
- 生活能力下降、精神行为症状
- 排除其他原因

**治疗原则：**
- 胆碱酯酶抑制剂：多奈哌齐、卡巴拉汀
- NMDA受体拮抗剂：美金刚
- 精神行为症状：非典型抗精神病药（小剂量）
- 认知训练、社会支持

### 帕金森病

**诊断要点：**
- 静止性震颤（"搓丸样"动作）
- 肌强直（铅管样/齿轮样）
- 运动迟缓
- 姿势平衡障碍

**治疗原则：**
- **首选左旋多巴制剂**（最有效）
- 多巴胺受体激动剂：普拉克索
- MAO-B抑制剂：司来吉兰
- 金刚烷胺（改善症状）
- 康复训练

### 脑血管病

**诊断要点：**
- 缺血性：突然起病，局灶性神经功能缺损
- 出血性：剧烈头痛、呕吐、意识障碍
- 头颅CT/MRI确诊

**治疗原则：**
- 缺血性：rt-PA静脉溶栓（4.5h内）、血管内治疗
- 抗血小板聚集：阿司匹林、氯吡格雷
- 脑保护剂、调控血压
- 康复训练

### 谵妄

**诊断要点：**
- 急性意识障碍，波动性病程
- 注意力障碍
- 可有幻觉、妄想
- 睡眠-觉醒周期紊乱

**治疗原则：**
- 识别和治疗诱因（感染、药物、代谢）
- 环境管理：安静、光线适中、定向力支持
- 必要时小剂量氟哌啶醇或奥氮平
- 避免使用苯二氮卓类（除非戒断）

## 泌尿系统

### 前列腺增生

**诊断要点：**
- 尿频、尿急、夜尿增多
- 排尿等待、尿线变细、排尿不尽
- 直肠指诊：前列腺增大
- 超声：前列腺体积增大、残余尿增多

**治疗原则：**
- α受体阻滞剂：坦索罗辛（改善症状）
- 5α还原酶抑制剂：非那雄胺（缩小前列腺）
- 重度梗阻：手术治疗（经尿道前列腺切除术）

### 尿失禁

**诊断要点：**

| 类型 | 特点 |
|------|------|
| 压力性 | 咳嗽、打喷嚏时漏尿 |
| 急迫性 | 尿急后漏尿，常伴尿频 |
| 充溢性 | 膀胱充盈后漏尿 |
| 混合性 | 两种或以上混合 |

**治疗原则：**
- 生活方式干预（减肥、戒烟）
- 盆底肌训练（压力性）
- 抗胆碱能药（急迫性）：索利那新
- 手术（压力性）：尿道悬吊术

### 慢性肾脏病

**诊断要点：**
- 血肌酐升高、GFR下降（>3个月）
- 蛋白尿、血尿
- 肾脏结构异常

**治疗原则：**
- 控制原发病
- 低蛋白饮食（0.6-0.8g/kg/d）
- 控制血压（ACEI/ARB，降至<130/80mmHg）
- 纠正贫血、钙磷代谢紊乱
- 避免肾毒性药物

## 老年综合征

| 综合征 | 核心表现 | 处理原则 |
|--------|----------|----------|
| **衰弱** | 体重下降、乏力、活动减少 | 运动锻炼、营养支持、多学科干预 |
| **跌倒** | 不明原因跌倒 | 平衡训练、环境改造、药物调整 |
| **尿便失禁** | 大小便失控 | 盆底训练、药物、护理 |
| **认知障碍** | 记忆力下降、定向力障碍 | 认知训练、药物、照护 |
| **压疮** | 皮肤破损 | 翻身、减压、伤口护理 |

### 衰弱评估（Fried标准）

满足3项以上：
1. 体重下降（1年内>5%）
2. 乏力（握力下降）
3. 行走速度减慢
4. 体力活动量降低
5. 疲乏

## 记忆口诀

### 老年用药原则
> "小单定个" — 小剂量、简化用药、定期评估、个体化

### 老年肺炎特点
> "症征分离" — 症状轻但体征少，影像学重

### 老年甲减表现
> "慢冷淡安" — 动作缓慢、怕冷、淡漠、面部黏液水肿

### 老年糖尿病特点
> "餐后为王" — 餐后血糖升高为主

### 老年高血压特点
> "单纯收" — 单纯收缩期高血压多见

### 老年谵妄特点
> "急波动" — 急性起病、波动性病程

### 老年用药警示
> "体位三低" — 体位性低血压、低血糖、低钠血症
