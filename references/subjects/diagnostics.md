# 诊断学速查表

> 306考纲覆盖，约占总内科33%

---

## 第一章 症状学

### 核心概念

| 症状 | 要点 |
|------|------|
| **发热** | 热型：稽留热（39-40℃，24h波动<1℃，见于大叶性肺炎）、弛张热（39℃以上，24h波动>2℃，见于败血症）、间歇热（高热期与无热期交替，见于疟疾）、波状热（逐渐上升逐渐下降，见于布氏杆菌病）、回归热（急升急降，见于回归热）、不规则热（无规律，见于结核病） |
| **咳嗽与咳痰** | 干咳（咽炎/哮喘）、湿咳（支气管炎/肺炎）、痰液性状（铁锈色痰=肺炎链球菌、黄脓痰=金葡菌、绿脓杆菌=铜绿假单胞菌、砖红色胶冻痰=克雷伯杆菌） |
| **咯血** | 少量<100ml/日、中量100-500ml/日、大量>500ml/日或一次>300ml。咯血与呕血鉴别：咯血有肺部症状/黑便/碱性，呕血有胃部症状/黑便/酸性 |
| **胸痛** | 胸壁（带状疱疹/肋软骨炎）、肺（肺炎累及胸膜）、心血管（心绞痛/心梗/肺栓塞）、纵隔（纵隔肿瘤） |
| **呼吸困难** | 吸气性（喉梗阻/哮吼）、呼气性（COPD/哮喘）、混合性（肺炎/心衰）、夜间阵发性（心源性哮喘） |
| **腹痛** | 急性腹痛：阑尾炎（转移性右下腹痛）、胆囊炎（右上腹/墨菲征）、胰腺炎（左上腹/腰背部带状疼痛）、肠梗阻（腹痛+腹胀+呕吐+肛门停止排气）、胃十二指肠穿孔（板状腹/膈下游离气体） |
| **呕血与黑便** | 病因：消化性溃疡（最常见）、食管胃底静脉曲张破裂、急性胃粘膜病变、胃癌。黑便：上消化道出血>50ml/日，柏油便：>250ml/日 |
| **黄疸** | 溶血性（UCB↑↑/CB正常/尿胆原↑）、肝细胞性（CB↑/UCB↑/尿胆红素+/尿胆原±）、阻塞性（CB↑↑/UCB↑/尿胆红素++/尿胆原↓）、先结合后非结合 |
| **意识障碍** | 觉醒度↓：嗜睡→昏睡→昏迷；意识内容↓：意识模糊→谵妄。脑疝：瞳孔不等大+对光反射消失+去脑强直 |

### 高频考点

- 热型鉴别：稽留热=大叶性肺炎/斑疹伤寒；弛张热=败血症/风湿热；间歇热=疟疾/急性肾盂肾炎；波状热=布氏杆菌病；回归热=回归热；不规则热=结核病/风湿热
-咯血量估计：大咯血窒息风险，一次>300ml或24h>500ml需紧急处理
- 腹膜刺激征=压痛+反跳痛+肌紧张，提示腹膜炎
- 黄疸鉴别三部曲：病史（溶血史/肝病史/用药史）→体检（肝脾/淋巴结）→化验（胆红素分类/肝酶/凝血）
- 意识障碍 Glasgow评分：睁眼4+言语5+运动6=最高15分，<8分需气管插管

### 记忆口诀

> **热型记忆**：稽留大叶斑，伤寒弛张败，风湿间歇疟，波状布氏回，不规结缔核
> **黄疸分类**：溶血UCK高，阻塞CB高，肝细胞都升高

## 第二章 体格检查

### 生命体征

| 项目 | 正常值 | 异常意义 |
|------|--------|----------|
| **体温** | 36.3-37.2℃ | >37.3℃为发热，>39℃为高热，>41℃为超高热 |
| **脉搏** | 60-100次/分 | <60=心动过缓，>100=心动过速，交替脉=心衰，奇脉=心脏压塞 |
| **呼吸** | 12-20次/分 | >24=呼吸急促，肺泡性呼吸困难；节律异常（潮式/间停=中枢病变） |
| **血压** | <120/80mmHg | 高血压≥140/90mmHg，直立性低血压（卧位转立位↓20/10mmHg） |

### 头颈部检查

| 项目 | 要点 |
|------|------|
| **瞳孔** | 大小（正常3-4mm）、对光反射（直接/间接）、近反射、Horner综合征（睑垂/缩瞳/陷落） |
| **扁桃体** | 0度不可见，1度可见，2度超出舌腭弓，3度超出咽后壁正中线 |
| **甲状腺** | 弥漫性（甲亢/桥本）、结节性（腺瘤/癌）、吹风样杂音（甲亢） |
| **颈动脉** | 颈动脉狭窄杂音（收缩期），颈静脉怒张（右心衰/心脏压塞） |

### 肺部检查

| 类型 | 正常音 | 异常音 |
|------|--------|--------|
| **呼吸音** | 支气管呼吸音（喉/胸骨上窝）、支气管肺泡呼吸音（肩胛间区/胸骨旁）、肺泡呼吸音（大部分肺野） |减弱=气胸/胸腔积液/肺气肿，增强=发热/代谢性酸中毒 |
| **啰音** | 干啰音（哮鸣音/鼾音=气道狭窄）、湿啰音（水泡音=支气管/肺泡液体） | 细湿啰音（肺泡）、中湿啰音（细支气管）、粗湿啰音（气管） |
| **语音共振** | 正常不可闻 | 增强=肺实变，减弱=胸腔积液/气胸 |

### 心脏检查

| 项目 | 要点 |
|------|------|
| **杂音** | 收缩期（房室瓣反流/主动脉/肺动脉狭窄）、舒张期（房室瓣狭窄/主/肺动脉反流）、连续性（动脉导管未闭） |
| **心律** | 窦性心律（HR 60-100，PR 0.12-0.20s，QRS<0.12s）、房性早搏、室性早搏、房颤（心律绝对不齐+第一心音强弱不等+脉搏短绌） |
| **心包摩擦音** | 搔抓样/摩擦样，坐位前倾屏气时明显，提示心包炎 |

### 腹部检查

| 项目 | 要点 |
|------|------|
| **视诊** | 腹部膨隆（腹水/肠梗阻/肿块）、腹式呼吸减弱（腹膜炎）、胃型/肠型/蠕动波 |
| **触诊** | 压痛部位、反跳痛（腹膜刺激）、腹壁紧张（板状腹=胃穿孔）、肝脾肿大（测量锁骨中线）、腹部肿块（性质/活动度/压痛） |
| **叩诊** | 移动性浊音（>1000ml腹水）、肝浊音界（缩小=胃肠穿孔）、肾区叩痛 |
| **听诊** | 肠鸣音（正常4-5次/分，亢进=机械性肠梗阻，减低=腹膜炎/低钾） |

### 神经系统检查

| 类型 | 内容 |
|------|------|
| **颅神经** | 12对颅神经检查，视/动眼/滑车/三叉/面/听/舌咽/迷走/副/舌下神经 |
| **运动系统** | 肌力（0-5级）、肌张力（折刀样↑=上运动神经元，铅管样↑=帕金森，减低=下运动神经元） |
| **感觉系统** | 浅感觉（痛/温/触）、深感觉（位置/振动）、皮层感觉（实体/图形） |
| **反射** | 生理反射（浅反射+深反射）、病理反射（Babinski/Chaddock/Gordon/Hoffmann）、脑膜刺激征（颈强/Kernig/Brudzinski） |

### 高频考点

- 扁桃体分度：3度为肿大达咽后壁正中线
- 干湿啰音鉴别：干啰音为气流通过狭窄气道，湿啰音为气流通过液体
- 心脏杂音分级：1/6最轻，2/6轻，3/6中等，4/6响亮，5/6很响，6/6极响/可触及震颤
- 腹水移动性浊音：>1000ml可叩出阳性
- Kernig征：仰卧屈髋屈膝90°，伸膝<135°为阳性
- 协调运动：跟膝胫试验阳性=小脑病变

### 记忆口诀

> **甲状腺触诊**：吞咽时上下来滑动，弥漫甲亢结节瘤，压痛常为亚甲炎
> **脑膜刺激征**：颈强摆头颈不弯，Kernig摆腿膝不伸，Brudzinski屈颈腿也弯

## 第三章 辅助检查

### 心电图

| 项目 | 诊断要点 |
|------|----------|
| **心梗** | ST弓背向上抬高（定位：V1-V3前壁、V3-V5前壁、II/III/aVF下壁、I/aVL侧壁），对应导联压低=对应性改变，Q波（梗死后出现） |
| **心律失常** | 房早（早发P波/PR间期正常/不完全代偿）、室早（宽大畸形QRS/完全代偿）、房颤（f波/R-R不等）、室颤（无QRS/等电位线）、房室传导阻滞（I度PR>0.20s/II度I型PR渐长直至QRS脱落/II度II型PR固定/QRS脱落/III度P波与QRS无关） |
| **电解质紊乱** | 高钾（T波高尖/QT延长/QRS增宽）、低钾（U波/T波低平/ST压低）、高钙（QT缩短） |

### X线检查

| 项目 | 诊断要点 |
|------|----------|
| **肺炎** | 大叶性肺炎（均匀一致密度增高影/支气管充气征）、支气管肺炎（斑片状阴影）、间质性肺炎（网格状/条索状） |
| **肠梗阻** | 站立位X线：肠管扩张>3cm（空肠=鱼骨刺/回肠=光滑），气液平面（阶梯状） |
| **骨折** | 首选X线，隐匿性骨折需CT/MRI |
| **气胸** | 无肺纹理区域，压缩边缘可见脏层胸膜线 |

### CT/MRI

| 项目 | 诊断要点 |
|------|----------|
| **脑卒中** | 急性期CT（48h内高密度影/等密度），MRI-DWI（急性期高信号） |
| **脑肿瘤** | CT等/低密度+强化，MRI T1低T2高信号+强化 |
| **肺栓塞** | CT肺动脉造影（充盈缺损） |
| **腹部疾病** | 肝血管瘤（MRI T2灯泡征）、胰腺炎（胰周渗出/坏死） |

### 实验室检查

| 项目 | 正常值 | 异常意义 |
|------|--------|----------|
| **血常规** | WBC 4-10×10^9/L，RBC男4.0-5.5×10^12/L女3.5-5.0×10^12/L，Hb男120-160g/L女110-150g/L，PLT 100-300×10^9/L | WBC↑感染/WBC↓病毒感染/粒细胞缺乏，Hb↓贫血，PLT↓血小板减少 |
| **肝功能** | ALT/AST 0-40U/L，ALP 40-150U/L，GGT 0-50U/L，TBIL 3.4-17.1μmol/L，CB 0-6.8μmol/L | ALT↑>AST↑/酒精肝AST>ALT，ALP↑/GGT↑胆道疾病 |
| **肾功能** | BUN 2.6-7.5mmol/L，Cr 44-133μmol/L，UA 90-420μmol/L | BUN/Cr↑肾功能损害，UA↑痛风 |
| **凝血功能** | PT 11-14s，APTT 25-35s，TT 12-16s，Fg 2-4g/L | PT↑外源性凝血（肝素/华法林/VitK缺乏），APTT↑内源性凝血 |
| **心肌标志物** | cTnI 0-0.04ng/ml，CK-MB 0-5ng/ml，Myoglobin 0-100ng/ml | cTnI升高=心梗（3-4h升高，11-24h达峰，7-14d恢复正常） |

### 高频考点

- 心梗定位：V1-V3前壁，V3-V5前壁，II/III/aVF下壁，I/aVL侧壁
- 心梗时间演变：超急性期（数分钟）ST斜型抬高→急性期（数小时-数天）ST弓背向上→近期（数周）ST恢复→陈旧期（数月后）T波倒置/Q波
- 肠梗阻X线：站立位见阶梯状气液平面，空肠鱼骨刺影
- PT正常11-14s，延长>3s有意义，INR 0.8-1.2
- 心肌酶：Myoglobin最早（1-2h），CK-MB次之（4-6h），cTnI/cTnT最特异（3-4h升高）

### 记忆口诀

> **心梗心电图演变**：超急抬高斜，急性弓背抬，近期ST复，陈旧倒置Q
> **心肌酶时间**：肌红1-2升，激酶4-6峰，肌钙7-14复

## 第四章 影像学与内镜

### 超声检查

| 项目 | 适应证 |
|------|--------|
| **腹部B超** | 肝/胆/胰/脾/肾疾病，胆囊结石（强回声伴声影），腹水（肝肾间隙无回声） |
| **心脏超声** | 心功能（EF正常>55%），瓣膜病（狭窄/反流），先心病（分流） |
| **血管超声** | 动脉粥样硬化（斑块），深静脉血栓（不可压缩），夹层（真假腔） |
| **浅表超声** | 甲状腺结节（TI-RADS分级），乳腺肿块（BI-RADS分级） |

### 内镜检查

| 项目 | 适应证 |
|------|--------|
| **胃镜** | 上消化道出血（呕血/黑便），胃癌早期诊断，胃息肉切除 |
| **结肠镜** | 结肠癌筛查，肠息肉，溃疡性结肠炎，克罗恩病 |
| **ERCP** | 胆管结石（取石/支架），胰头癌（胆管梗阻） |
| **支气管镜** | 中心型肺癌（活检），肺部感染（肺泡灌洗） |

### 高频考点

- 超声：胆囊结石=强回声光团+声影，可随体位移动
- 心脏EF正常>55%，轻度下降40-54%，中度30-39%，重度<30%
- 胃镜活检：胃癌确诊依靠病理，弥漫型胃癌（印戒细胞癌）预后差
- 结肠癌：便潜血阳性→肠镜确诊，多发于直肠/乙状结肠

### 记忆口诀

> **ERCP适应证**：胆管结石胰腺癌，梗阻性黄疸金标准
> **超声分级**：TI-RADS二级单，囊性为主多良性，三级实性需观察，四级以上要活检

## 第五章 诊断思路与鉴别诊断

### 发热鉴别诊断

| 病因 | 特点 |
|------|------|
| **感染性** | 最常见，细菌/病毒/真菌/寄生虫 |
| **非感染性** | 结缔组织病（发热+皮疹+关节痛）、恶性肿瘤（淋巴瘤/白血病）、中枢性发热（下丘脑病变）、药物热（用药后发热） |

### 呼吸困难鉴别

| 类型 | 特点 | 代表疾病 |
|------|------|----------|
| **肺源性** | 吸气性（三凹征/喉鸣=上气道梗阻）、呼气性（呼气延长/哮鸣=COPD/哮喘）、混合性（劳力性呼吸困难） | 支气管哮喘/COPD/肺炎/胸腔积液 |
| **心源性** | 劳力性呼吸困难→端坐呼吸→夜间阵发性呼吸困难→急性肺水肿 | 急性左心衰 |
| **中毒性** | 深快呼吸（代谢性酸中毒）、潮式呼吸（中枢病变） | 糖尿病酮症酸中毒/尿毒症 |
| **血源性** | 呼吸急促+心悸 | 重度贫血/高铁血红蛋白血症 |

### 胸痛鉴别

| 疾病 | 疼痛特点 | 伴随症状 |
|------|----------|----------|
| **心绞痛** | 胸骨后压榨样，3-5分钟，硝酸甘油可缓解 | 出汗/恶心/放射至左肩臂 |
| **急性心梗** | 持续>30分钟，硝酸甘油不缓解 | 大汗/濒死感/休克 |
| **肺栓塞** | 突发胸痛+呼吸困难+咯血 | 低氧血症/心动过速 |
| **主动脉夹层** | 撕裂样剧痛，向背部放射 | 血压差异/神经系统症状 |
| **气胸** | 突发单侧胸痛+呼吸困难 | 患侧呼吸音减低/叩诊过清音 |
| **肺炎** | 发热+咳嗽+胸痛（胸膜受累） | 湿啰音/实变体征 |
| **胃食管反流** | 烧心+反酸，卧位加重 | 慢性咳嗽/哮喘 |

### 腹痛鉴别

| 疾病 | 疼痛特点 | 体检特点 |
|------|----------|----------|
| **急性阑尾炎** | 转移性右下腹痛（6-8h） | 右下腹压痛/反跳痛/McBurney点 |
| **急性胆囊炎** | 右上腹痛/墨菲征阳性 | 莫非征阳性/黄疸 |
| **急性胰腺炎** | 上腹痛向腰背部放射 | Grey-Turner征/Cullen征（出血坏死型） |
| **胃十二指肠穿孔** | 突发剧痛，板状腹 | 膈下游离气体 |
| **肠梗阻** | 腹痛+腹胀+呕吐+肛门停止排气 | 肠鸣音亢进/气过水声 |
| **宫外孕破裂** | 下腹痛+休克 | 失血表现/宫颈举痛 |
| **肾绞痛** | 绞痛向会阴部放射 | 肾区叩痛/血尿 |

### 黄疸鉴别

| 类型 | CB | UCB | 尿胆红素 | 尿胆原 | 粪胆原 |
|------|-----|-----|----------|--------|--------|
| **溶血性** | 正常/微↑ | ↑↑ | 阴性 | ↑↑ | ↑ |
| **肝细胞性** | ↑ | ↑ | 阳性 | ± | ↓ |
| **阻塞性** | ↑↑ | ↑ | 强阳性 | ↓/消失 | ↓/消失 |

### 高频考点

- 发热问诊：热型/诱因/缓解因素/伴随症状/用药史
- 呼吸困难夜间阵发性：左心衰典型表现，坐起后缓解
- 胸痛危险信号：主动脉夹层（撕裂样/血压差）、肺栓塞（突发/咯血）、心梗（持续/硝酸甘油不缓解）
- 急性阑尾炎：McBurney点（脐与右髂前上棘连线外1/3）压痛
- 黄疸最常用检查：胆红素分类+腹部超声

### 记忆口诀

> **腹痛鉴别**：右上墨菲黄疸见，左上胰腺腰背牵，下腹阑尾转移痛，腹穿不凝宫外孕
> **黄疸诊思路**：先问病史找诱因，再看胆红分类清，影像检查定位置，病因治疗才根本
