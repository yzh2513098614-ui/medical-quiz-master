# 康复医学速查表

## 康复评定

### 运动功能评定

| 评定项目 | 方法 | 标准 |
|----------|------|------|
| **肌力评定** | MMT（徒手肌力检查） | 0-5级 |
| | 等速肌力测试 | 峰力矩、功率 |
| **关节活动度（ROM）** | 量角器测量 | 正常参考值 |
| **平衡功能** | Berg平衡量表 | 0-56分，<40分跌倒风险高 |
| | Fugs平衡量表 | 0-24分 |
| **协调功能** | 指鼻试验、跟膝胫试验 | 正常/异常 |
| **步态分析** | 目测法/仪器分析 | 正常/异常步态 |

### 日常生活活动能力（ADL）

#### Barthel指数

| 项目 | 评分标准 |
|------|----------|
| 进食 | 0/5/10分 |
| 洗澡 | 0/5分 |
| 修饰（洗脸、刷牙） | 0/5分 |
| 穿衣 | 0/5/10分 |
| 控制大便 | 0/5/10分 |
| 控制小便 | 0/5/10分 |
| 如厕 | 0/5/10分 |
| 转移（床-椅） | 0/5/10/15分 |
| 步行 | 0/5/10/15分 |
| 上楼梯 | 0/5/10分 |

**评定标准：**
- 满分100分
- ≥60分：轻度功能障碍
- 41-59分：中度功能障碍
- ≤40分：重度功能障碍

#### 其他ADL量表

| 量表 | 用途 |
|------|------|
| IADL（工具性ADL） | 复杂日常活动能力 |
| FIM（功能独立性评定） | 更详细的ADL评定 |

### 言语功能评定

| 评定项目 | 内容 |
|----------|------|
| **失语症** | 听理解、命名、复述、阅读、书写 |
| **构音障碍** | 构音器官运动、语音清晰度 |
| **吞咽功能** | 饮水试验（洼田试验）、VFSS |

#### 洼田饮水试验

| 分级 | 表现 |
|------|------|
| Ⅰ级 | 5秒内一口喝完，无呛咳 |
| Ⅱ级 | 5秒内喝完，超过一口，有呛咳 |
| Ⅲ级 | 5秒以上喝完，超过一口，有呛咳 |
| Ⅳ级 | 每次均呛咳，全部喝完困难 |
| Ⅴ级 | 屡屡呛咳，不能喝完 |

### 认知功能评定

| 量表 | 用途 | 满分/筛查界值 |
|------|------|----------------|
| **MMSE** | 简易精神状态检查 | 30分，<27分异常 |
| MoCA | 蒙特利尔认知评估 | 30分，<26分异常 |
| 画钟试验 | 视空间功能 | 4分，<3分异常 |
| 数字广度 | 注意力/工作记忆 | 正序/倒序 |

#### MMSE评定项目

- 定向力（时间+地点）：10分
- 即刻记忆：3分
- 注意力和计算力：5分
- 延迟记忆：3分
- 语言（命名+复述+命令+阅读+书写）：9分
- 视空间（临摹图形）：1分

### 心肺功能评定

| 评定方法 | 指标 |
|----------|------|
| 6分钟步行试验 | 6MWD（正常>400m） |
| 心肺运动试验 | VO2max、AT |
| 代谢当量 | METs |
| Borg评分 | 主观呼吸困难程度 |

## 康复治疗技术

### 物理因子治疗

#### 电刺激疗法

| 类型 | 适应证 |
|------|--------|
| **神经肌肉电刺激（NMES）** | 废用性肌萎缩、周围神经损伤 |
| **功能性电刺激（FES）** | 偏瘫足下垂、脊髓损伤 |
| **经皮电神经刺激（TENS）** | 镇痛（Gate control理论） |
| **直流电离子导入** | 炎症、疤痕 |

#### 光疗

| 类型 | 适应证 |
|------|--------|
| 红外线 | 慢性炎症、疼痛、痉挛 |
| 紫外线 | 感染、创面、银屑病 |
| 激光（LLLT） | 促进创面愈合、镇痛、消炎 |

#### 超声波治疗

- 频率：0.8-3MHz
- 作用：机械效应、热效应
- 适应证：软组织损伤、瘢痕、骨折愈合

#### 温热疗法

| 类型 | 适应证 |
|------|--------|
| 蜡疗 | 瘢痕、关节挛缩 |
| 湿热敷 | 肌肉痉挛、慢性疼痛 |
| 短波/微波 | 深部热疗 |

### 运动治疗

#### 关节活动度训练

| 类型 | 适应证 |
|------|--------|
| 被动关节活动度训练 | 肌力0-2级、制动后 |
| 主动辅助关节活动度训练 | 肌力2-3级 |
| 主动关节活动度训练 | 肌力≥3级 |
| 牵伸训练 | 关节挛缩、肌肉紧张 |

#### 肌力训练

| 类型 | 原理 |
|------|------|
| 等长收缩训练 | 肌力2-3级，增强肌张力 |
| 等张收缩训练 | 增强肌力和肌肉体积 |
| 等速训练 | 训练效率和安全性高 |
| 渐进抗阻训练 | 肌肉适应性增强 |

#### 平衡训练

**训练顺序：**
1. 坐位平衡（静态→动态）
2. 跪位平衡
3. 站位平衡（静态→动态）
4. 行走中平衡

**训练方法：**
- 重心转移训练
- 平衡板训练
- 平衡垫训练
- 闭眼训练

#### 步态训练

| 训练内容 | 说明 |
|----------|------|
| 基础训练 | 站立位重心转移、原地踏步 |
| 辅助器具训练 | 平行杠内步行、助行器使用 |
| 实际步行训练 | 室内外步行、上下楼梯 |

## 康复常见疾病

### 脑卒中康复

#### 偏瘫康复

**康复时机：** 病情稳定后48小时开始

**康复目标：** 抑制异常模式，建立正常运动模式

| 时期 | 康复内容 |
|------|----------|
| 急性期 | 床上良肢位摆放、被动活动、坐位平衡 |
| 恢复期 | 坐站转移、站位平衡、步行训练、手功能训练 |
| 后遗症期 | 日常生活活动训练、社区康复 |

**Brunnstrom分期：**

| 分期 | 表现 |
|------|------|
| Ⅰ期 | 迟缓，无随意运动 |
| Ⅱ期 | 联合反应，共同运动 |
| Ⅲ期 | 随意出现共同运动 |
| Ⅳ期 | 共同运动模式打破，开始分离运动 |
| Ⅴ期 | 协调运动精细化 |
| Ⅵ期 | 独立运动，近乎正常 |
| Ⅶ期 | 正常随意运动 |

#### 言语康复

- 发病1-3个月是言语康复最佳时期
- 轻度失语：语言训练为主
- 重度失语：手势语、交流板等替代方式

#### 吞咽康复

| 训练方法 | 说明 |
|----------|------|
| 间接训练 | 口腔感觉训练、舌肌训练、Masako训练 |
| 直接训练 | 吞咽姿势、食物调整、点头样吞咽 |

### 脊髓损伤康复

#### 损伤平面与功能预后

| 损伤平面 | 关键功能 | 预期自理能力 |
|----------|----------|--------------|
| C1-4 | 呼吸、头控制 | 完全依赖 |
| C5 | 肩、肘屈曲 | 少量自理，大量依赖 |
| C6 | 腕背伸 | 部分自理，需大量辅助 |
| C7 | 肘伸展 | 基本自理，需少量辅助 |
| C8-T2 | 手指精细动作 | 自理，轮椅活动 |
| T3-12 | 躯干控制 | 自理，佩戴KAFO可站立 |
| L1-2 | 髋屈伸 | 自理，佩戴AFO可步行 |
| L3-5 | 膝背伸/跖屈 | 基本自理，社区内步行 |
| S1-2 | 踝/足功能 | 正常或接近正常 |

#### 康复重点

- **急性期：** 预防并发症（压疮、深静脉血栓、泌尿系感染）
- **恢复期：** 运动功能训练、日常生活活动训练
- **社会康复：** 职业训练、心理支持

---

### 骨折后康复

#### 康复原则

1. 骨折固定期：肌肉等长收缩、未固定关节活动
2. 骨折愈合期：关节活动度训练、渐进负重
3. 恢复期：肌力训练、平衡协调训练

#### 各部位骨折康复

| 部位 | 康复要点 |
|------|----------|
| 上肢 | 尽早进行手指、腕、肘、肩活动 |
| 下肢 | 渐进负重、步态训练、防关节挛缩 |
| 脊柱 | 腰背肌训练、姿势矫正 |

---

### 关节置换术后康复

#### 髋关节置换

| 时间 | 康复内容 |
|------|----------|
| 术后1-3天 | 踝泵、股四头肌等长收缩 |
| 术后4-7天 | 床上训练、坐位训练 |
| 术后2周 | 助行器步行、转移训练 |
| 术后3周后 | 渐进抗阻训练、上下楼梯 |

**注意事项：**
- 避免髋关节屈曲>90°、内收、内旋
- 避免坐矮凳、翘二郎腿
- 防脱位宣教

#### 膝关节置换

| 时间 | 康复内容 |
|------|----------|
| 术后当天 | 踝泵、股四头肌等长收缩 |
| 术后1-3天 | CPM训练（被动屈曲） |
| 术后1周 | 主动屈曲训练、床边坐位 |
| 术后2周 | 站立平衡、步行训练 |
| 术后3周后 | 关节活动度训练、渐进抗阻 |

---

### 腰背痛康复

#### 急性期处理

- 卧床休息（<48小时）
- 冰敷/热敷
- 非甾体抗炎药
- 柔韧性训练

#### 慢性腰背痛康复

| 治疗内容 | 说明 |
|----------|------|
| 核心稳定训练 | 腹横肌、多裂肌训练 |
| McKenzie疗法 | 姿势矫正、椎间盘突出康复 |
| 悬吊训练（SET） | 激活深层肌肉 |
| 麦肯基力学治疗 | 椎间盘突出康复 |

#### 腰背痛预防

- 正确姿势（站、坐、卧）
- 避免久坐久站
- 规律腰背肌训练
- 搬重物正确方式

## 记忆口诀

### 康复评定原则
> "力活言认心肺" — 肌力/ROM、ADL、言语、认知、心肺

### 骨折康复原则
> "动早负重晚" — 固定期活动、愈合期负重

### 脑卒中康复时机
> "48小时后" — 病情稳定48小时开始

### 脊髓损伤康复重点
> "压深尿" — 防压疮、防DVT、防泌尿系感染

### 关节置换后禁忌
> "90度内收内旋" — 屈髋<90°、不内收内旋

### 康复评定顺序
> "先动后静" — 动态平衡→静态平衡

### 物理因子治疗选择
> "急冷慢热" — 急性期用冷，慢性期用热

### 肌力训练原则
> "等长等张等速" — 循序渐进，从等长到等速

### 脑卒中Brunnstrom分期
> "123动分离协调正常" — 迟缓→共同运动→分离运动→协调→正常
