## 第一章 细胞和组织的损伤与适应

### 核心概念

| 概念 | 要点 |
|------|------|
| **适应性变化** | 萎缩（体积缩小+功能下降）、肥大（体积增大+功能增强）、增生（数量增多）、化生（类型转换） |
| **细胞变性** | 细胞水肿（常见，最早期病变）、脂肪变性（肝>心>肾）、玻璃样变性（结缔组织/血管壁/细胞内）、淀粉样变性、黏液样变性 |
| **坏死** | 凝固性坏死（心/肾/脾）、液化性坏死（脑）、坏疽（干性/湿性/气性）、纤维素样坏死（风湿/AON） |
| **凋亡** | 程序性死亡，无炎症，凋亡小体形成，基因调控（Bcl-2家族、Caspase） |

### 高频考点

- 细胞水肿：线粒体受损，ATP生成减少，Na+/K+泵功能障碍，细胞内Na+和水增多
- 脂肪变性：肝细胞最常见，常见于慢性肝/心/肾疾病
- 坏死标志：细胞核固缩→碎裂→溶解
- 坏疽：干性（四肢末端，动脉阻塞静脉通畅）、湿性（内脏，与细菌感染有关）、气性（深部肌肉，产气荚膜杆菌）

### 记忆口诀

> **萎缩**：萎缩就是小而弱，肥大有质也有量
> **坏死**：核固缩碎裂溶解，凝固液化纤维素
> **化生**：鳞化腺化上皮化，损伤长期刺激来

## 第二章 损伤的修复

### 核心概念

| 概念 | 要点 |
|------|------|
| **再生** | 干细胞分化增殖替代死亡细胞，完全性再生（结构和功能完全恢复）/不完全性再生（纤维组织增生） |
| **肉芽组织** | 由毛细血管+成纤维细胞+炎细胞组成，最后转化为瘢痕组织 |
| **创伤愈合** | 伤口愈合过程：止血→炎症→增殖→重塑，一期愈合（二期愈合） |
| **影响愈合因素** | 局部（感染/血供/神经/异物）、全身（年龄/营养/全身疾病/药物） |

### 高频考点

- 细胞再生能力：不稳定细胞（表皮/呼吸道/消化道上皮）>稳定细胞（肝/胰/肾/涎腺）>永久细胞（神经/心肌/骨骼肌）
- 肉芽组织作用：抗感染保护创面，填补伤口缺损，机化坏死物
- 瘢痕形成：胶原纤维沉积和交联，瘢痕疙瘩（过度增生）
- 骨折愈合：血肿形成→机化→骨痂形成→钙化→重塑

### 记忆口诀

> **再生能力**：不稳定细胞最活跃，永久细胞难再生，稳定细胞居中间
> **肉芽组织**：新鲜嫩红如肉芽，修复愈合功劳大，最后变成灰白疤

## 第三章 局部血液循环障碍

### 核心概念

| 概念 | 要点 |
|------|------|
| **充血** | 动脉血量增多（动脉性/主动性）、静脉血量增多（静脉性/被动性） |
| **出血** | 内出血（组织内）、外出血（体表），病因：破裂性出血（外伤/病变）、渗出性出血（血管壁损伤） |
| **血栓形成** | 条件：心血管内皮损伤、血流状态改变（涡流/缓慢）、血液凝固性增高 |
| **栓塞** | 血栓栓塞（最常见）、脂肪栓塞、气体栓塞、羊水栓塞、肿瘤细胞栓塞 |
| **梗死** | 贫血性梗死（白色梗死，心/肾/脾）、出血性梗死（红色梗死，肺/肠/脑） |

### 高频考点

- 血栓类型：白色血栓（血小板+纤维蛋白，头部）、混合血栓（血小板小梁+纤维蛋白+红细胞，体部）、红色血栓（纤维蛋白+红细胞，尾部）、透明血栓（微血栓，DIC）
- 血栓结局：软化（纤溶酶）、机化（肉芽组织）、钙化（静脉石/动脉石）、再通
- 梗死条件：动脉阻塞+侧支循环不充分+组织耐缺氧能力低
- 肺梗死：肺动脉栓塞+肺淤血（双重血供，所以肺梗死少见）
- 肠梗死：肠系膜动脉栓塞，出血性梗死（肠系膜上动脉为主干）

### 记忆口诀

> **血栓形成条件**：内皮损伤是首要，血流缓慢和涡流，凝血增高要记牢
> **梗死**：心肾脾白梗死，肺肠红梗死

## 第四章 炎症

### 核心概念

| 概念 | 要点 |
|------|------|
| **炎症基本病变** | 变质（变性和坏死）、渗出（血管反应：扩张/通透性↑/液体渗出/细胞渗出）、增生（实质/间质细胞） |
| **急性炎症** | 持续时间短（数小时-数天），以渗出为主，中性粒细胞为主 |
| **慢性炎症** | 持续时间长（数周-数月），以增生为主，淋巴细胞/浆细胞/巨噬细胞为主 |
| **炎症介质** | 血管活性胺（组胺/5-HT）、前列腺素、白细胞三烯、血小板激活因子、细胞因子（IL/TNF）、补体（C3a/C5a）、凝血系统 |

### 高频考点

- 渗出过程：血管扩张→血管通透性↑→白细胞附壁→白细胞游出→趋化
- 炎症分类：变质性炎（病毒性肝炎）、渗出性炎（浆液性/纤维素性/化脓性/出血性）、增生性炎（慢性炎症/肉芽肿）
- 化脓性炎：脓细胞（变性坏死的中性粒细胞），蜂窝织炎（弥漫性化脓，溶血性链球菌），脓肿（局限性，金黄色葡萄球菌）
- 肉芽肿：慢性炎症特征，由上皮样细胞+多核巨细胞组成（结核结节、伤寒小结、异物肉芽肿）

### 记忆口诀

> **炎症介质**：组胺五羟色胺，前列腺素白三烯，血小板激活因子
> **化脓菌**：金葡脓肿局限大，链球蜂窝弥散广

## 第五章 肿瘤

### 核心概念

| 概念 | 要点 |
|------|------|
| **肿瘤异型性** | 组织结构异型性、细胞异型性（核浆比↑/核深染/核分裂象↑） |
| **生长方式** | 膨胀性生长（良性）、浸润性生长（恶性）、外生性生长（乳头状/息肉状） |
| **扩散** | 直接蔓延、转移（淋巴道/血道/种植性转移） |
| **命名** | 良性：部位+组织+瘤，恶性：上皮→癌（鳞癌/腺癌/移行上皮癌），间叶→肉瘤 |

### 高频考点

- 良性肿瘤与恶性肿瘤区别：异型性（有无）、核分裂象（多少）、生长速度（快慢）、生长方式（膨胀/浸润）、转移（有无）、复发（易/难）
- 癌前病变：黏膜白斑（口腔/外阴）、慢性萎缩性胃炎、结肠息肉、乳腺囊性增生病
- 常见肿瘤：癌（肺癌/胃癌/肝癌/结直肠癌/乳腺癌/宫颈癌）、肉瘤（骨肉瘤/脂肪肉瘤/纤维肉瘤/平滑肌肉瘤）
- 副肿瘤综合征：异位内分泌（ACTH/ADH/胰岛素）、副蛋白血症（多发性骨髓瘤）、黑棘皮病

### 记忆口诀

> **良性恶性**：良性异型小，恶性异型大；良性膨胀长，恶性浸润长；良性不转移，恶性易转移
> **癌前病变**：白斑萎缩息肉增，乳腺囊性宫颈糜，慢性炎症要警惕

## 第六章 心血管系统疾病

### 核心概念

| 概念 | 要点 |
|------|------|
| **动脉粥样硬化** | 泡沫细胞形成（巨噬细胞吞噬Ox-LDL）、纤维斑块→粥样斑块→复合病变，主要累及大中等动脉 |
| **高血压** | 细动脉硬化（玻璃样变）、小动脉硬化（内膜胶原纤维增生）、主要累及全身细、小动脉 |
| **风湿病** | A组乙型溶血性链球菌感染，与M蛋白交叉反应，病变：风湿小结（Aschoff小体） |
| **心瓣膜病** | 二尖瓣狭窄（风心病最常见）、主动脉瓣狭窄/关闭不全 |

### 高频考点

- 粥样硬化斑块：纤维帽+脂质池+胆固醇结晶+钙化+新生血管
- 粥样硬化并发症：斑块破裂→血栓形成→栓塞、斑块内出血→动脉瘤形成、血管腔狭窄→缺血
- 风湿性心脏病：二尖瓣受累最常见（75%），其次为主动脉瓣
- 感染性心内膜炎：急性（化脓菌，金黄色葡萄球菌）、亚急性（草绿色链球菌）
- 心肌病：扩张型（EF↓）、肥厚型（流出道梗阻）、限制型（心室舒张障碍）

### 记忆口诀

> **粥样硬化**：大中等动脉内膜下，泡沫细胞是特征，纤维斑块来形成
> **风湿小结**：阿绍夫小结是特征，风湿热的皮肤表现

## 第七章 呼吸系统疾病

### 核心概念

| 概念 | 要点 |
|------|------|
| **慢性支气管炎** | 黏膜上皮纤毛倒伏脱落、黏液腺增生、软骨变性萎缩→管壁慢性炎症 |
| **肺气肿** | 肺泡扩张融合（阻塞性/老年性/代偿性），全腺泡型/腺泡周围型/隔旁型 |
| **肺心病** | 肺动脉高压→右心室肥厚扩张→右心衰竭 |
| **肺炎** | 大叶性肺炎（肺炎链球菌，稽留热，铁锈色痰）、小叶性肺炎（混合菌感染）、病毒性肺炎（间质性） |
| **肺癌** | 鳞癌（中央型，吸烟相关）、腺癌（周围型，女性多见）、小细胞肺癌（恶性最高） |

### 高频考点

- 大叶性肺炎分期：充血水肿期（1-2天）→红色肝样变期（3-4天）→灰色肝样变期（5-6天）→溶解消散期
- 小叶性肺炎：化脓性炎，支气管+肺泡受累，散在小灶性病变
- 硅肺：吸入SiO2→肺巨噬细胞吞噬→硅尘颗粒→Fibroblast增生→硅结节形成→弥散性纤维化
- 肺癌分类：鳞癌（最常见，角化珠/细胞间桥）、腺癌（腺腔/黏液分泌）、小细胞癌（神经内分泌颗粒，燕麦状细胞）

### 记忆口诀

> **大叶性肺炎**：充血水肿红色变，灰色溶解是四期，铁锈色痰肝样变
> **肺癌**：鳞癌中央吸烟相关，腺癌周围女性多见

## 第八章 消化系统疾病

### 核心概念

| 概念 | 要点 |
|------|------|
| **慢性胃炎** | 浅表性（非萎缩性）、萎缩性（自身免疫性/多灶萎缩性）、肥厚性，幽门螺杆菌感染（+） |
| **消化性溃疡** | 胃溃疡（GU多在胃小弯胃角附近）、十二指肠溃疡（DU多在球部前壁） |
| **病毒性肝炎** | 急性/慢性/重型（急性/亚急性/慢性重型），HAV/HBV/HCV/HDV/HEV |
| **肝硬化** | 假小叶形成（特征性病变），门脉高压+肝功能减退 |
| **消化道肿瘤** | 食管癌（鳞癌多见）、胃癌（腺癌多见）、结直肠癌（腺癌多见）、原发性肝癌（肝细胞癌最常见） |

### 高频考点

- 溃疡病特点：圆形/椭圆形，边缘整齐，底部平坦，底部依次为渗出层→肉芽组织→瘢痕组织
- 病毒性肝炎病理：肝细胞变性（细胞水肿/嗜酸性变）、肝细胞坏死（点状/碎片状/桥接/大片）、炎细胞浸润
- 肝硬化分类：门脉性（小结节性，早期乙肝/酒精）、坏死后性（大结节性，慢性肝炎进展）、胆汁性（梗阻性）、淤血性（慢性右心衰）
- 门脉高压表现：脾大、腹水、侧支循环形成（食管下段/直肠/脐周）

### 记忆口诀

> **溃疡特点**：圆或椭圆，边缘整齐，底平壁陡，疤痕层是愈合
> **肝硬化**：假小叶是特征，纤维间隔来分割，门脉高压肝衰

## 第九章 泌尿系统疾病

### 核心概念

| 概念 | 要点 |
|------|------|
| **肾小球肾炎** | 急性/急进性/慢性/膜性/微小病变型/系膜增生性/局灶节段性 |
| **肾盂肾炎** | 上尿路感染（肾盂），细菌感染（大肠杆菌最常见），急性/慢性 |
| **肾细胞癌** | 肾小管上皮来源，透明细胞癌最常见（脂质糖原丰富） |
| **膀胱癌** | 尿路上皮癌最常见（90%），无痛性血尿 |

### 高频考点

- 急性肾小球肾炎：弥漫性增生性肾炎（毛细血管内皮+系膜增生），上呼吸道感染史，补体C3↓
- 急进性肾小球肾炎：新月体性肾炎（壁层上皮细胞增生形成新月体），Goodpasture综合征（抗GBM抗体）
- 膜性肾病：基底膜钉突样增厚，"大白肾"，成人肾病综合征最常见原因
- 慢性肾小球肾炎：肾小球硬化→肾单位减少→继发性颗粒性固缩肾

### 记忆口诀

> **新月体**：壁层上皮增生，新月体来形成，急进肾炎特征
> **膜性肾病**：基底膜钉突样，大白肾来命名

## 第十章 乳腺及女性生殖系统疾病

### 核心概念

| 概念 | 要点 |
|------|------|
| **乳腺癌** | 非浸润性（导管内癌/小叶原位癌）、浸润性（非特殊型最常见/特殊型）、paget病 |
| **宫颈癌** | 鳞状细胞癌（90%多见，HPV感染）、腺癌，移行带区好发 |
| **子宫内膜疾病** | 子宫内膜增生（单纯性/复杂性/不典型增生）、子宫内膜癌（腺癌）、子宫平滑肌瘤（良性） |
| **卵巢肿瘤** | 上皮性（浆液性/黏液性/子宫内膜样）、性索间质性（颗粒细胞瘤/卵泡膜细胞瘤）、生殖细胞性（畸胎瘤） |

### 高频考点

- 乳腺癌分类：非浸润性（原位癌）、浸润性（非特殊型最常见60-70%、特殊型有各自特征）
- 乳腺癌转移：淋巴道转移（腋窝淋巴结最早），血道转移（肺/骨/肝/脑）
- 宫颈癌：鳞状上皮CIN（宫颈上皮内瘤变）→原位癌→浸润癌，HPV16/18相关
- 卵巢肿瘤：上皮性最常见（50-70%），性索间质分泌雌激素，生殖细胞可含三胚层组织

### 记忆口诀

> **乳腺癌**：非浸润原位癌，浸润非特最常见，特殊黏液小叶样
> **宫颈癌**：CIN一二三级，早期浸润和癌，HPV相关要记牢

## 第十一章 传染病及寄生虫病

### 核心概念

| 概念 | 要点 |
|------|------|
| **结核病** | 结核结节（特征性病变，上皮样细胞+朗格汉斯巨细胞），肺结核（I-IV型） |
| **伤寒** | 伤寒细胞（吞噬了淋巴细胞/红细胞/细胞碎片的巨噬细胞），伤寒小结 |
| **细菌性痢疾** | 假膜性大肠炎（纤维素渗出+坏死+炎细胞），福氏/宋内志贺菌 |
| **寄生虫病** | 阿米巴病（肠阿米巴/肝阿米巴）、血吸虫病（虫卵结节/干线型肝硬化） |

### 高频考点

- 结核病：基本病变（渗出/增生/坏死），好转（吸收/纤维化/钙化），恶化（溶解播散/扩大/空洞形成）
- 肺结核：I型原发性（儿童，哑铃状阴影）、II型血行播散型（粟粒性结核）、III型继发性（成人，浸润空洞）、IV型结核性胸膜炎、V型其他
- 伤寒：回肠下段集合淋巴小结/孤立淋巴小结肿胀→坏死→溃疡（长轴与肠管平行）→愈合
- 流行性脑脊髓膜炎：脑膜炎奈瑟菌，蛛网膜下腔脓性渗出，突发高热+脑膜刺激征+淤点瘀斑

### 记忆口诀

> **结核结节**：上皮样细胞朗格汉斯，渗出增生坏死三部曲
> **伤寒溃疡**：溃疡长轴平行肠，愈合不窄不粘连

## 第十二章 性传播疾病与免疫病理

### 核心概念

| 概念 | 要点 |
|------|------|
| **梅毒** | 梅毒螺旋体，硬下疳（I期）、梅毒疹（II期）、树胶肿（III期），闭塞性动脉内膜炎 |
| **艾滋病** | HIV感染，CD4+T细胞↓，Kaposi肉瘤（血管内皮来源），机会感染 |
| **免疫病理** | 变态反应（I-IV型）、自身免疫病、免疫缺陷病 |

### 高频考点

- I型超敏反应：IgE介导，肥大细胞/嗜碱粒细胞，速发型（过敏性休克/哮喘/荨麻疹）
- II型超敏反应：IgG/IgM介导，细胞溶解（输血反应/新生儿溶血/自身免疫性溶血性贫血）
- III型超敏反应：免疫复合物介导，Arthus反应（类Arthus反应）、血清病、肾小球肾炎（SLE）
- IV型超敏反应：T细胞介导，细胞免疫，接触性皮炎（染发剂/镍）/结核菌素试验/移植物排斥

### 记忆口诀

> **四型超敏**：一速发二细胞毒，三免复合四细胞，移植排斥属四型
> **自身免疫病**：SLE是二型相关，桥本甲减四型来
