## 第一章 药效学

### 核心概念

| 概念 | 要点 |
|------|------|
| **受体理论** | 占领学说（受体与配体1:1结合）、速率学说、亲和力与内在活性 |
| **受体类型** | G蛋白偶联受体（肾上腺素/乙酰胆碱M受体）、离子通道受体（尼古丁受体）、酪氨酸激酶受体（胰岛素受体）、核受体（糖皮质激素受体） |
| **激动剂** | 与受体高亲和力+高内在活性，产生效应（阿片受体激动剂=吗啡） |
| **拮抗剂** | 与受体高亲和力+无内在活性，阻断激动剂作用（竞争性/非竞争性） |
| **量效关系** | 最小有效量（阈剂量）、最大效应（Emax）、效价强度（产生50%效应的浓度） |
| **治疗指数** | TI=LD50/ED50，TI越大越安全（安全范围=TI） |
| **安全范围** | 最小有效量与最小中毒量之间的距离 |

### 高频考点

- 竞争性拮抗剂：与激动剂竞争同一受体，结合可逆，量效曲线平行右移
- 非竞争性拮抗剂：与受体形成共价键，结合不可逆，Emax降低
- 部分激动剂：内在活性<1，与激动剂合用呈拮抗作用，与受体亲和力存在
- 受体脱敏：长期使用激动剂后受体数目下调（向下调节）
- 受体增敏：长期使用拮抗剂后受体数目上调（向上调节）

### 记忆口诀

> **受体分类**：G蛋白偶联受体最常见，七次跨膜是特征
> **竞争vs非竞争**：竞争可逆平行移，非竞争不可逆，Emax降低是标志

## 第二章 药动学

### 核心概念

| 过程 | 关键参数 | 影响因素 |
|------|----------|----------|
| **吸收** | 生物利用度（F= AUCiv/AUCpo）、首关效应（肝脏代谢使口服生物利用度降低） | 吸收部位/给药途径/药物脂溶性/首关效应 |
| **分布** | 表观分布容积（Vd）、血浆蛋白结合率、血脑屏障、胎盘屏障 | 脂溶性/蛋白结合率/组织亲和力/特殊屏障 |
| **代谢** | 肝脏细胞色素P450酶系（CYP450）、首关效应、酶诱导/酶抑制 | 肝功能/遗传多态性/药物相互作用 |
| **排泄** | 肾小球滤过/肾小管分泌/肾小管重吸收、胆汁排泄、肠肝循环 | 肾功能/胆汁分泌/尿液pH |

### 药动学参数

| 参数 | 定义 | 临床意义 |
|------|------|----------|
| **半衰期（t1/2）** | 血药浓度下降50%所需时间 | 反映药物消除速度，决定给药间隔 |
| **清除率（CL）** | 单位时间内清除的血浆体积 | 肝/肾功能不全时需调整剂量 |
| **稳态血药浓度（Css）** | 给药速度=消除速度时的浓度 | 连续给药5个t1/2达稳态 |
| **负荷剂量** | 首次给药达到Css的剂量 | 缩短达稳态时间，DL= Css×Vd/C |

### 高频考点

- 首关效应：口服药物经肝脏代谢生物利用度降低（硝酸甘油、吗啡、普萘洛尔首关效应明显）
- 肝药酶诱导剂：苯巴比妥、利福平、卡马西平（加速自身及其他药物代谢）
- 肝药酶抑制剂：酮康唑、氯霉素、西咪替丁、红霉素（减慢其他药物代谢）
- 肝功能Child-Pugh分级：C级禁用主要经肝代谢药物（如利多卡因）
- 肾功能不全剂量调整：内生肌酐清除率（Ccr）<30ml/min需调整给药方案

### 记忆口诀

> **首关效应**：首关效应口服低，舌下含服避过肝
> **肝药酶诱导**：利福平、苯巴比妥，酶诱导加速代谢
> **肝药酶抑制**：酮康唑、氯霉素、红霉素，酶抑制代谢减慢

## 第三章 传出神经系统药

### 核心概念

| 递质/受体 | 分布 | 生理效应 |
|------------|------|----------|
| **胆碱能神经** | 副交感神经节后纤维、运动神经、交感神经节前纤维 | M样作用（心脏抑制/腺体分泌/平滑肌收缩/瞳孔缩小） |
| **去甲肾上腺素** | 交感神经节后纤维 | α1血管收缩/瞳孔散大/胃肠平滑肌松弛 |
| **多巴胺** | 中枢/外周 | D1肾血管扩张，D2中枢抑制 |

### 胆碱能系统药物

| 药物 | 机制 | 临床应用 | 不良反应 |
|------|------|----------|----------|
| **毛果芸香碱** | M受体激动剂（缩瞳） | 青光眼（闭角型）、阿托品中毒 | 视物模糊、出汗、流涎 |
| **新斯的明** | 抗胆碱酯酶（可逆性） | 重症肌无力、术后肠麻痹 | 胆碱能危象（过量） |
| **有机磷酸酯** | 抗胆碱酯酶（不可逆） | 杀虫剂（农药） | M样/N样症状（见中毒部分） |
| **阿托品** | M受体拮抗剂 | 缓慢型心律失常、有机磷解毒、M3受体阻断（麻醉前给药） | 抗胆碱：口干/视物模糊/尿潴留/便秘 |

### 肾上腺素能系统药物

| 药物 | 机制 | 临床应用 | 禁忌证 |
|------|------|----------|--------|
| **肾上腺素** | α+β受体激动 | 心脏骤停、过敏性休克、支气管痉挛 | 高血压/冠心病/甲亢 |
| **去甲肾上腺素** | α1>β1受体激动 | 休克（低灌注/高阻力） | 缺血性器官损害 |
| **异丙肾上腺素** | β1=β2受体激动 | 缓慢性心律失常、心脏骤停、支气管痉挛 | 冠心病/甲亢/心律失常 |
| **多巴胺** | D1=D2=β1>α1 | 休克（低剂量肾血管扩张） | 心律失常 |
| **酚妥拉明** | α受体拮抗剂（非选择性） | 嗜铬细胞瘤诊断/外周血管痉挛 | 低血压/心律失常 |
| **普萘洛尔** | β受体拮抗剂（非选择性） | 高血压/心绞痛/心律失常/甲亢 | 哮喘/AVB/心源性休克 |

### 高频考点

- 毛果芸香碱：缩瞳（兴奋瞳孔括约肌M受体），降低眼内压（开放Schlemm管）
- 阿托品：散瞳（阻断瞳孔括约肌M受体）、睫状肌麻痹（视近物不清）、解除迷走神经过度兴奋（加速心率）
- 肾上腺素：α受体收缩皮肤黏膜血管（止血），β2受体舒张支气管（平喘），β1受体强心（心脏骤停）
- 普萘洛尔禁忌：支气管哮喘（β2阻断诱发支气管痉挛）、重度房室传导阻滞、急性心衰
- β受体阻滞剂：美托洛尔（β1选择性）、比索洛尔（β1高选择性）、卡维地洛（α+β阻滞）

### 记忆口诀

> **M受体激动剂**：毛果缩瞳降眼压，出汗流涎肠蠕动
> **阿托品**：散瞳升压抗胆碱，心加速来腺抑制
> **β受体阻滞剂**：洛尔阻滞β受体，哮喘禁用要记牢
> **肾上腺素**：强心升压又平喘，过敏休克首选药

## 第六章 抗菌药

### 抗菌药分类与抗菌谱

| 药物 | 抗菌谱 | 临床应用 | 不良反应 |
|------|--------|----------|----------|
| **青霉素G** | 革兰阳性球菌（链球菌/肺炎球菌）、脑膜炎奈瑟菌、螺旋体 | 咽炎/肺炎/脑膜炎/梅毒/破伤风 | 过敏性休克（最严重）/皮疹/赫氏反应 |
| **阿莫西林** | 广谱青霉素（对革兰阴性菌作用增强） | 呼吸道/泌尿道感染/幽门螺杆菌根除 | 过敏/皮疹/腹泻 |
| **苯唑西林** | 耐酶青霉素（对甲氧西林敏感金葡菌MSSA） | 耐青霉素葡萄球菌感染 | 过敏/肝损 |
| **头孢唑林** | 第一代头孢（革兰阳性强，革兰阴性弱） | 围手术期预防/呼吸道感染 | 肾损/过敏 |
| **头孢曲松** | 第三代头孢（革兰阳性弱，革兰阴性强） | 淋病/脑膜炎/肺炎/腹腔感染 | 胆汁淤积/二重感染 |
| **头孢吡肟** | 第四代头孢（对β-内酰胺酶稳定） | 医院获得性肺炎/复杂性腹腔感染 | CNS不良反应（尤其是肾功能不全） |
| **阿奇霉素** | 大环内酯类（支原体/衣原体/军团菌） | 呼吸道感染/性病/社区获得性肺炎 | 胃肠道反应/肝损/QT延长 |
| **左氧氟沙星** | 氟喹诺酮类（广谱） | 呼吸道/泌尿道/肠道感染/骨关节感染 | 肌腱断裂/光毒性/CNS反应/QT延长 |
| **万古霉素** | 糖肽类（革兰阳性菌） | MRSA/艰难梭菌结肠炎 | 肾损（万古霉素肾）/红人综合征/耳损 |
| **替考拉宁** | 糖肽类（革兰阳性菌） | MRSA（与万古霉素类似） | 肾损较万古霉素轻 |

### 抗菌药作用机制

| 机制 | 代表药物 |
|------|----------|
| **抑制细胞壁合成** | 青霉素类/头孢菌素类/万古霉素/达托霉素 |
| **抑制蛋白质合成** | 氨基糖苷类（30S）/四环素类（30S）/大环内酯类（50S）/氯霉素（50S）/林可霉素（50S） |
| **抑制核酸合成** | 氟喹诺酮类（DNA回旋酶）/利福平（RNA聚合酶）/甲氧苄啶（叶酸合成） |
| **破坏细胞膜完整性** | 多黏菌素B/两性霉素B/制霉菌素/达托霉素 |

### 抗菌药耐药机制

| 耐药机制 | 代表菌种 |
|----------|----------|
| **β-内酰胺酶水解** | 葡萄球菌（青霉素酶）、肠杆菌（ESBLs超广谱β-内酰胺酶） |
| **靶位改变（PBP亲和力降低）** | 肺炎球菌（青霉素耐药）、MRSA（ mecA基因编码PBP2a） |
| **外排泵增强/膜通透性降低** | 铜绿假单胞菌（对多种抗菌药天然耐药） |
| **靶位酶突变** | 结核分枝杆菌（利福平耐药rpoB突变） |
| **代谢途径改变** | 肠杆菌（对磺胺类耐药） |
| **氨基糖苷类钝化酶** | 肠杆菌科（乙酰转移酶/核苷转移酶使氨基糖苷类失活） |

### 抗菌药分级管理

| 级别 | 药物 | 使用原则 |
|------|------|----------|
| **非限制使用** | 阿莫西林/头孢氨苄/红霉素/左氧氟沙星（口服） | 一线用药，可自行使用 |
| **限制使用** | 第三代头孢/氟喹诺酮（注射）/万古霉素 | 需病原学检查支持 |
| **特殊使用** | 碳青霉烯类/万古霉素（注射）/利福平 | 需专家会诊/药敏结果/特殊审批 |

### 高频考点

- 青霉素G过敏性休克抢救：肾上腺素（首选）+抗组胺药+糖皮质激素+补充血容量
- 抗菌谱记忆：草绿色链球菌（青霉素+庆大霉素）、肠球菌（氨苄西林+庆大霉素协同）
- MRSA首选：万古霉素/替考拉宁/利奈唑胺
- 梅毒首选：青霉素G（苄星青霉素）
- 军团菌肺炎首选：阿奇霉素/左氧氟沙星
- 幽门螺杆菌根除：铋剂+PPI+阿莫西林+甲硝唑（四联疗法14天）

### 记忆口诀

> **抗菌药抗菌谱**：阳性球菌青霉强，阴性杆菌头孢帮，支原体衣原大环用，厌氧甲硝唑来帮
> **β-内酰胺类**：青头碳青霉烯类，细胞壁合成抑制
> **MRSA治疗**：万古替考利奈唑，糖肽类药是首选
> **青霉素过敏**：过敏性休克最严重，喉头水肿呼吸难，肾上腺素是首选
> **氨基糖苷类**：氨基糖苷杀菌剂，耳毒肾毒是特点，肌肉松弛呼吸停
> **大环内酯类**：红霉素阿奇霉素，支原体衣原体特效

## 第七章 抗菌药临床应用

### 常见感染的经验性治疗

| 感染类型 | 常见病原体 | 经验性用药 |
|----------|------------|------------|
| **社区获得性肺炎** | 肺炎链球菌/支原体/衣原体/流感嗜血杆菌 | 青霉素/阿莫西林/大环内酯（轻中度）；三代头孢+大环内酯/呼吸氟喹诺酮（重度） |
| **医院获得性肺炎** | 铜绿假单胞菌/肠杆菌/MRSA | 抗假单胞菌β-内酰胺+抗 MRSA（万古霉素） |
| **细菌性脑膜炎** | 肺炎链球菌/脑膜炎奈瑟菌/流感嗜血杆菌/李斯特菌 | 成人：头孢曲松+万古霉素+氨苄西林（>50岁）；儿童：头孢噻肟+头孢霉素 |
| **细菌性心内膜炎** | 草绿色链球菌/肠球菌/金葡菌 | 青霉素+庆大霉素（草绿色链球菌）；万古霉素+庆大霉素（肠球菌/金葡菌） |
| **复杂性腹腔感染** | 肠杆菌/脆弱拟杆菌/肠球菌 | 三代头孢+甲硝唑/碳青霉烯类/哌拉西林他唑巴坦 |
| **泌尿道感染** | 大肠埃希菌/克雷伯菌/肠球菌 | 呋喃妥因（单纯膀胱炎）/磷霉素/三代头孢/氟喹诺酮 |
| **细菌性痢疾** | 志贺菌/沙门菌 | 氟喹诺酮（首选）/阿奇霉素 |

### 抗菌药不良反应汇总

| 药物 | 严重不良反应 | 常见不良反应 |
|------|--------------|--------------|
| **青霉素** | 过敏性休克（I型超敏） | 皮疹/赫氏反应（梅毒/回归热） |
| **氨基糖苷类** | 耳毒性（不可逆）/肾毒性（可逆） | 麻木/肌肉无力 |
| **氟喹诺酮类** | 肌腱断裂/心脏QT延长/CNS毒性 | 胃肠道反应/光毒性/失眠 |
| **四环素类** | 牙齿黄染（<8岁）/骨发育障碍 | 胃肠道反应/二重感染（难辨梭菌） |
| **氯霉素** | 骨髓抑制（再生障碍性贫血） | 灰婴综合征（新生儿） |
| **磺胺类** | Stevens-Johnson综合征/骨髓抑制 | 肾损（结晶尿）/过敏（皮疹） |
| **万古霉素** | 红人综合征（非IgE介导）/肾损/耳损 | 发热/皮疹 |
| **两性霉素B** | 肾损（几乎100%） | 寒战发热/低钾血症 |

### 记忆口诀

> **脑膜炎常见病原**：脑膜双球肺链球，流感杆菌脑膜奈，老年幼儿要加李斯特
> **心内膜炎常见病原**：草绿链球菌肠球菌，金葡心内是凶手
> **氨基糖苷类**：氨基糖苷耳肾毒，耳鸣耳聋肾损害，呼吸抑制肌肉停

## 第八章 其他重要药物

### 糖皮质激素

| 项目 | 内容 |
|------|------|
| **抗炎机制** | 抑制磷脂酶A2（减少PG/LT合成）/抑制COX-2表达/抑制NF-κB |
| **免疫抑制机制** | 抑制Th细胞/增强Treg/抑制细胞因子合成 |
| **临床应用** | 自身免疫病（系统性红斑狼疮/类风湿关节炎）/哮喘持续状态/过敏性休克/器官移植排斥 |
| **不良反应** | 感染/糖尿病/高血压/骨质疏松/消化性溃疡/股骨头坏死/肾上腺皮质功能不全（撤药过快） |
| **禁忌证** | 。活动性结核/消化性溃疡/严重精神病/未能控制的感染 |
| **常用方案** | 短效：氢化可的松（应激/急症）；中效：泼尼松（自身免疫病）；长效：地塞米松（脑水肿/过敏） |

### 抗炎药

| 药物 | 机制 | 临床应用 | 不良反应 |
|------|------|----------|----------|
| **阿司匹林** | 非选择性COX-1/COX-2抑制剂 | 心脑血管病二级预防（75-100mg/d）/解热镇痛/抗炎 | 胃肠道出血（COX-1抑制）/出血倾向/Reye综合征（儿童禁用） |
| **对乙酰氨基酚** | 选择性COX-3（中枢） | 解热镇痛（无抗炎作用）/儿童首选 | 肝损（过量，N-乙酰对苯醌亚胺蓄积） |
| **布洛芬** | 非选择性COX抑制剂 | 风湿性疾病/痛经/解热 | 胃肠道反应/肾损/心血管事件 |
| **塞来昔布** | 选择性COX-2抑制剂 | 骨关节炎/类风湿关节炎 | 心血管事件（塞来昔布未证实CV安全性优于非选择性） |

### 记忆口诀

> **糖皮质激素**：激素抗炎又免疫，发热休克哮喘好，感染扩散胃溃疡，骨质疏松骨坏死
> **糖皮质激素撤药**：撤药过快肾上腺皮质功能不全，分次减量是原则
> **阿司匹林**：百年老药阿司匹，解热镇痛抗血小板，胃肠出血要警惕
