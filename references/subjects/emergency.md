# 急诊医学速查表

---

## 第一章 急救技能

### 一、心肺脑复苏（CPCR）

**成人BLS流程**（CABCDE）：

| 步骤 | 内容 | 要点 |
|------|------|------|
| C - Circulation | 胸外心脏按压 | 频率100-120次/分，深度5-6cm，按压:放松=1:1 |
| A - Airway | 开放气道 | 仰头抬颏法，疑颈椎损伤用双手托颌法 |
| B - Breathing | 人工呼吸 | 10-12次/分，避免过度通气 |
| C - Circulation | 维持循环 | 继续胸外按压 |
| D - Drugs | 药物治疗 | 肾上腺素、胺碘酮等 |
| E - Electricity | 电除颤 | 单向波360J，双向波120-200J |

**心脏按压要点**：
- 部位：胸骨中下1/3交界处
- 姿势：肘关节伸直，垂直下压
- 周期：30:2（成人），15:2（儿童双人）
- 尽量减少中断

**药物选择**：
| 药物 | 适应证 | 剂量 |
|------|--------|------|
| 肾上腺素 | 室颤/无脉室速/心脏停搏 | 1mg IV q3-5min |
| 胺碘酮 | 室颤/无脉室速（除颤后） | 首剂300mg，第二剂150mg |
| 利多卡因 | 室性心律失常 | 1-1.5mg/kg |
| 硫酸镁 | 尖端扭转型室速 | 1-2g IV |

> **记忆口诀**：肾上腺素首选，胺碘酮次之，利多卡因亦可
> **按压深度**：5-6cm（成人），至少胸廓前后径1/3（儿童）
> **按压位置**：胸骨下半段，不可剑突下

### 二、电除颤与复律

**适应证**：
- 室颤、无脉性室速 → 非同步电除颤
- 室上性心动过速、房颤 → 同步电复律

**能量选择**：

| 类型 | 单向波 | 双向波 |
|------|--------|--------|
| 心室除颤 | 360J | 120-200J |
| 室上速/房颤 | 100J | 50-100J |
| 房扑 | 50J | 25-50J |

**电极位置**：
- 胸骨电极：胸骨右缘第2肋间
- 剑突电极：左乳头线下方

> **关键**：室颤时不要等待药物，先除颤！尽早除颤是提高存活率的关键！

### 三、气管插管

**适应证**：
- 呼吸心跳骤停
- 呼吸衰竭（PaO₂＜60mmHg或PaCO₂＞50mmHg）
- 气道保护反射消失
- 需要机械通气

**插管路径**：经口插管（急诊首选）

**判断插管成功**：
- 听诊双肺呼吸音对称
- 胃区无气过水声
- 呼气末CO₂监测（黄金标准）
- 胸片确认位置

**并发症**：
- 牙齿损伤、口腔出血
- 喉痉挛、支气管痉挛
- 误入食管（最危险）
- 气胸

### 四、深静脉穿刺

**常用部位**：
| 部位 | 定位 | 并发症 |
|------|------|--------|
| 颈内静脉 | 胸锁乳突肌内侧缘 | 气胸、血胸 |
| 锁骨下静脉 | 锁骨中点下方 | 气胸（多见） |
| 股静脉 | 股动脉内侧0.5-1cm | 动静脉瘘、血肿 |

**并发症预防**：
- 穿刺后复查胸片（锁骨下/颈内）
- 严格无菌操作
- 避免长期留置（＜2周）

## 第二章 急症处理

### 一、急性冠脉综合征（ACS）

**分型**：

| 类型 | 特征 | 心电图 | 标记物 |
|------|------|--------|--------|
| STEMI | 持续性胸痛≥30min | ST弓背向上抬高 | cTn升高达峰 |
| NSTEMI | 不稳定性胸痛 | ST压低/T波倒置 | cTn升高 |
| UA | 不稳定性胸痛 | ST压低/T波倒置 | cTn正常 |

**STEMI再灌注时间窗**：

| 指标 | 时间窗 |
|------|--------|
| 门-球时间（D2B） | **≤90分钟**（PCI） |
| 溶栓适应证 | 发病＜**12小时**，年龄＜75岁 |
| 溶栓-球囊时间 | **≤30分钟** |

> **记忆口诀**：黄金120分，白金12小时
> 90分钟PCI，12小时内可溶栓

**STEMI处理流程**：
1. 吸氧、止痛（吗啡2-4mg IV）
2. 阿司匹林300mg嚼服
3. 氯吡格雷300-600mg或替格瑞洛180mg
4. 肝素（PCI前）
5. 急诊PCI（首选）或溶栓
6. β受体阻滞剂、ACEI、他汀

**NSTEMI/UA处理**：
- 危险分层（TIMI评分）
- 抗凝+抗血小板
- 择期造影（高危48小时内）

### 二、急性心力衰竭

**Killip分级**（急性心梗心功能）：

| 分级 | 表现 | 住院死亡率 |
|------|------|------------|
| I | 无心衰表现 | 6% |
| II | 肺底湿啰音、S3奔马律 | 17% |
| III | 急性肺水肿 | 38% |
| IV | 心源性休克 | 80% |

**急性左心衰处理**（抢救措施）：
| 措施 | 药物/方法 |
|------|-----------|
| 体位 | 半卧位或端坐位，双腿下垂 |
| 吸氧 | 高流量6-8L/min，湿化瓶加酒精 |
| 镇静 | 吗啡3-5mg IV（减低前后负荷） |
| 利尿 | 呋塞米20-40mg IV |
| 扩血管 | 硝酸甘油0.3-0.6mg舌下或硝普钠 |
| 正性肌力 | 多巴酚丁胺、米力农 |
| 正压通气 | CPAP/BiPAP，必要时气管插管 |

> **记忆口诀**：端坐位腿下垂，吸氧镇痛吗啡配
> 利尿扩管强心记，关键时刻呼吸机

### 三、高血压危象

**定义**：BP＞180/120mmHg，伴靶器官损害

**分类**：

| 类型 | 特点 | 处理 |
|------|------|------|
| 高血压急诊 | 急性靶器官损害 | 立即降压（1小时内） |
| 高血压亚急症 | 无靶器官损害 | 24-48小时内降压 |

**降压目标**：
- 脑血管意外：BP≥200/130mmHg才降压，目标24小时降低＜25%
- 主动脉夹层：收缩压100-120mmHg，心率＜60次/分
- 其他：1小时内降低25%，后逐步降至目标

**常用药物**：

| 药物 | 起效 | 适应证 |
|------|------|--------|
| 硝普钠 | 即刻 | 各种高血压急诊 |
| 硝酸甘油 | 2-5min | ACS、心衰 |
| 拉贝洛尔 | 5-10min | 脑血管意外、妊高症 |
| 乌拉地尔 | 3-5min | 多种急诊 |

> **记忆口诀**：高血压危象不慌，硝普纳硝酸甘油是良方
> 硝普纳0.5-3μg/kg/min，注意避光

### 四、脑血管意外

**快速鉴别**：

| 类型 | 起病 | 症状 | 影像 |
|------|------|------|------|
| 脑梗死 | 安静/睡眠中 | 偏瘫、失语 | CT低密度（24h后） |
| 脑出血 | 活动中 | 头痛、呕吐 | CT高密度（立即） |
| SAH | 突发 | 雷击样头痛 | CT脑池高密度 |

**急性脑梗死溶栓时间窗**：

| 指标 | 时间窗 |
|------|--------|
| rt-PA静脉溶栓 | 发病＜**4.5小时** |
| 动脉溶栓 | 发病＜6小时 |
| 机械取栓 | 发病＜**24小时**（前循环） |

> **关键**：时间就是大脑！每延误1分钟，丢失190万神经元！

**GCS评分**（颅脑损伤评估）：

| 项目 | 1分 | 2分 | 3分 | 4分 | 5分 | 6分 |
|------|-----|-----|-----|-----|-----|-----|
| 睁眼 | 无 | 疼痛 | 语言 | 自发 | - | - |
| 语言 | 无 | 异常 | 含糊 | 正确 | - | - |
| 运动 | 无 | 伸展 | 屈曲 | 躲避 | 定位 | 遵命 |

**GCS＜8分**提示重型颅脑损伤，需气管插管保护气道

### 五、哮喘持续状态

**定义**：严重哮喘发作，对常规治疗无效

**诊断标准**：
- 呼吸困难持续
- 说话不能成句
- 意识障碍
- 奇脉＞25mmHg
- PEF＜33%预计值

**处理**：

| 措施 | 用法 |
|------|------|
| 吸氧 | 100%纯氧，维持SpO₂≥93% |
| 雾化β2激动剂 | 沙丁胺醇2.5-5mg q20min×3 |
| 抗胆碱能 | 异丙托溴铵0.5mg q20min×3 |
| 糖皮质激素 | 甲泼尼龙40-80mg IV |
| 氨茶碱 | 负荷5mg/kg，维持0.5-0.8mg/kg/h |
| 镁硫酸 | 2g IV（20分钟内） |
| 机械通气 | 呼吸机首选SIMV+PEEP |

> **记忆口诀**：沙丁胺醇雾化全，激素静滴要当先
> 氨茶碱配合使用，危重之时呼吸机

### 六、糖尿病酮症酸中毒（DKA）

**诊断标准**：
- 血糖＞13.9mmol/L
- 血酮＞3.0mmol/L或尿酮+
- 动脉血pH＜7.3
- HCO₃⁻＜18mmol/L
- AG↑（阴离子间隙）

**治疗流程**：

| 阶段 | 目标 | 措施 |
|------|------|------|
| 补液 | 恢复血容量 | 0.9%NS 1-2L（首1小时） |
| 胰岛素 | 降血糖 | 0.1U/kg/h静滴 |
| 补钾 | 维持电解质 | 血K⁺＜5.2时补钾 |
| 补碱 | pH＜6.9时补 | 5%NaHCO₃ 100mL |

**补液量**：8-10L/24h（前8小时补充约一半）

> **记忆口诀**：DKA治疗三步走，扩容补钾胰岛素
> 血糖降到13.9，延长滴注减剂量

### 七、急性中毒

#### 有机磷农药中毒

**中毒机制**：抑制胆碱酯酶 → 乙酰胆碱蓄积

**胆碱酯酶复活剂**：碘解磷定（PAM）或氯解磷定（PAM-Cl）

**阿托品化指标**（记忆口诀）：
> 瞳孔散大颜面红，肺部啰音消失，皮肤干燥心率快

**阿托品用量**：
- 轻度：2-4mg q15-30min
- 中度：4-8mg q15min
- 重度：8-12mg q10-15min

**肟类复能剂**：首剂PAM 1-2g IV（30分钟内），后1g/h维持

#### 一氧化碳（CO）中毒

**诊断**：COHb水平
| COHb% | 症状 |
|-------|------|
| 10-20% | 轻度头痛 |
| 30-40% | 中度头晕、恶心 |
| 50-60% | 昏迷、抽搐 |
| ＞60% | 死亡 |

**治疗**：
- 高流量高浓度吸氧（100% O₂）
- 高压氧舱（首选，重症）
- 防治脑水肿（甘露醇+地塞米松）

> **记忆口诀**：煤气中毒不用慌，高压氧疗是宝藏
> 碳氧血红蛋白高，100%纯氧来赶跑

## 第三章 创伤处理

### 一、多发伤处理原则（ATLS）

**初级评估**（ABCDE）：

| 步骤 | 内容 | 致命性 |
|------|------|--------|
| A - Airway | 气道（保护颈椎） | 最优先 |
| B - Breathing | 呼吸 | 可立即致死 |
| C - Circulation | 循环（控制出血） | 早期可致死 |
| D - Disability | 神经状态 | |
| E - Exposure | 暴露/环境控制 | |

**控制出血**：
- 直接压迫
- 填塞止血
- 止血带（四肢大出血）
- 手术止血

### 二、颅脑损伤

**GCS评分**（见上文）

**颅内血肿手术指征**：

| 血肿类型 | 手术指征 |
|----------|----------|
| 硬膜外 | 幕上＞30mL，幕下＞10mL，中线移位＞5mm |
| 硬膜下 | 厚度＞10mm，中线移位＞5mm |
| 脑内 | 直径＞3cm，伴意识恶化 |

**颅内压监测**：
- GCS≤8分，有CT异常
- 维持ICP＜22mmHg
- 维持CPP（脑灌注压）＞60mmHg

### 三、脊柱损伤

**制动原则**：
- 现场：硬板平移，整体翻转
- 影像学：CT为主，MRI评估脊髓
- 固定：颈托+脊柱板

**手术指征**：
- 骨折脱位不稳定
- 神经功能进行性加重
- 脊柱压缩超过50%

### 四、骨盆骨折

**失血量估算**：
| 骨折类型 | 估计失血量 |
|----------|------------|
| 稳定型 | 500-1500mL |
| 不稳定型 | 1500-3000mL+ |

**失血性休克是骨盆骨折的主要死因**

**急救处理**：
- 骨盆带/床单固定
- 大量输血（O型红细胞）
- 介入栓塞（出血血管）
- 外固定架稳定骨盆

> **记忆口诀**：骨盆骨折休克凶，床单固定是首冲
> 大量输血保容量，介入栓塞显神通

## 第四章 急诊流程图

### 成人心脏骤停流程

```
开始CPR
    ↓
心电图判断 ↓
    ├─ 室颤/无脉室速 → 非同步除颤 → CPR 2分钟 → 肾上腺素1mg q3-5min → 除颤 → CPR
    ├─ 无脉电活动 → CPR + 肾上腺素 → 查找可逆原因
    └─ 心脏停搏 → CPR + 肾上腺素 → 查找可逆原因
```

### ACS处理流程

```
胸痛患者 → 10分钟内ECG → 判读
    ↓
    ├─ ST抬高 → STEMI → 评估PCI/溶栓
    │    → 门-球90分钟内PCI 或 30分钟内溶栓
    ↓
    └─ ST压低/T波改变 → NSTEMI/UA
         → 危险分层 → 抗凝+抗血小板
         → 高危48h内造影，低危择期造影
```

### 急性心衰处理流程

```
急性呼吸困难 → 评估 → BNP/TNT
    ↓
    ├─ 心源性 → 确立心衰诊断
    │    → Killip分级 → 相应处理
    │    → I-II级：利尿+血管扩张剂
    │    → III-IV级：利尿+正性肌力+血管扩张剂
    ↓
    └─ 非心源性 → 寻找其他病因
```
