## 麻醉前评估

### ASA分级（美国麻醉医师协会）

| 分级 | 定义 | 示例 |
|------|------|------|
| ASA I | 正常健康 | 健康、无吸烟、无饮酒 |
| ASA II | 轻度系统性疾病 | 轻微吸烟、轻度肥胖 |
| ASA III | 重度系统性疾病 | 糖尿病、高血压控制良好 |
| ASA IV | 危及生命的严重疾病 | 心绞痛、心肌梗死病史 |
| ASA V | 濒死状态，不手术必死亡 | 主动脉瘤破裂、腹主动脉瘤 |
| ASA VI | 脑死亡，器官捐献 | - |

> **口诀**：一正二轻三重四死五濒六捐

### Mallampati气道分级

| 分级 | 可见结构 | 插管难度 |
|------|----------|----------|
| I级 | 腭垂、腭弓、软腭、硬腭 | 容易 |
| II级 | 腭垂、腭弓、软腭 | 一般 |
| III级 | 软腭、基底部 | 困难 |
| IV级 | 仅硬腭 | 非常困难 |

### 麻醉前准备

| 项目 | 内容 |
|------|------|
| **禁食** | 清液体2h、固体6-8h |
| **胃排空** | 饱胃患者清醒插管/快速序贯诱导 |
| **血压** | 高血压患者术前血压控制在160/100mmHg以下 |
| **糖尿病** | 手术当日停用口服降糖药和长效胰岛素 |
| **抗凝药** | 华法林停用5d、阿司匹林停用7d |
| **急救药品** | 阿托品、麻黄碱、去氧肾上腺素 |

## 全身麻醉

### 静脉麻醉药

| 药物 | 机制 | 适应症 | 特点 |
|------|------|--------|------|
| **丙泊酚** | GABA受体激动 | 全麻诱导/维持 | 起效快(30s)、苏醒快、再分布、无蓄积 |
| **依托咪酯** | GABA受体激动 | 心血管不稳定患者 | 血流动力学稳定、抑制肾上腺皮质 |
| **氯胺酮** | NMDA受体拮抗 | 短小手术、烧伤 | 分离麻醉、强镇痛、升高颅内压/眼压 |
| **咪达唑仑** | GABA受体激动 | 镇静/遗忘 | 可逆性遗忘、呼吸抑制轻 |
| **芬太尼** | μ受体激动 | 镇痛 | 强效镇痛、呼吸抑制、心动过缓 |

**丙泊酚 vs 依托咪酯对比**：
| 特征 | 丙泊酚 | 依托咪酯 |
|------|--------|----------|
| 循环系统 | 血压↓、心率↓ | 血流动力学稳定 |
| 呼吸系统 | 呼吸抑制强 | 呼吸抑制轻 |
| 肾上腺皮质 | 无影响 | 抑制（长时间使用） |
| 注射痛 | 常见 | 少见 |
| 适用 | 成人、日间手术 | 心血管不稳定、老年 |

### 吸入麻醉药

| 药物 | MAC（肺泡最低有效浓度） | 特点 |
|------|------------------------|------|
| **七氟烷** | 1.8% | 最常用、气道刺激小、可控性好 |
| **异氟烷** | 1.15% | 肌松好、心律失常少 |
| **恩氟烷** | 1.7% | 不增加肌松药作用 |
| **地氟烷** | 6% | 醒快、刺激气道 |
| **氧化亚氮** | 105% | 镇痛强、苏醒快、弥散性缺氧 |

> **口诀**：七氟最常用、异氟肌松好、恩氟不增效、氧化亚氮笑

**吸入麻醉药苏醒时间排序**（从快到慢）：地氟烷 > 氧化亚氮 > 七氟烷 > 异氟烷 > 恩氟烷

### 肌松药

#### 去极化肌松药

| 药物 | 起效 | 持续时间 | 特点 |
|------|------|----------|------|
| **琥珀胆碱** | 30-60s | 6-10min | 唯一去极化肌松药；高钾血症、恶性高热、II相阻滞 |

> ⚠️ 琥珀胆碱**绝对禁忌**：大面积烧伤、创伤、截瘫患者（高钾血症风险）

#### 非去极化肌松药

| 药物 | 起效 | 持续时间 | 代谢方式 | 拮抗 |
|------|------|----------|----------|------|
| **维库溴铵** | 2-3min | 30-45min | 肝脏代谢 | 新斯的明/舒更葡萄糖 |
| **罗库溴铵** | 1min | 45-75min | 肝脏代谢为主 | 舒更葡萄糖（首选） |
| **顺阿曲库铵** | 2-3min | 45-60min | Hofmann消除 | 不需拮抗 |
| **阿曲库铵** | 2-3min | 30-45min | Hofmann消除 | 不需拮抗 |

> **口诀**：维库罗库要拮抗，顺阿阿曲Hofmann自消失

#### 肌松药拮抗

| 拮抗药 | 适应症 | 用法 |
|--------|--------|------|
| **新斯的明** | 轻度-中度阻滞 | 0.04-0.07mg/kg + 阿托品0.015mg/kg |
| **舒更葡萄糖** | 重度阻滞、琥珀胆碱II相阻滞 | 2-4mg/kg |

## 局部麻醉与神经阻滞

### 常用局麻药

| 药物 | 浓度 | 持续时间 | 特点 |
|------|------|----------|------|
| **利多卡因** | 0.5%-2% | 45-60min | 中效、渗透力强、过敏少 |
| **丁卡因** | 0.25%-1% | 2-3h | 长效、表面麻醉首选 |
| **罗哌卡因** | 0.2%-1% | 2-4h | 运动-感觉分离、心脏毒性低 |
| **布比卡因** | 0.25%-0.75% | 3-6h | 长效、心脏毒性大 |
| **氯普鲁卡因** | 1%-3% | 30-60min | 快效、酯类、孕妇可用 |

**局麻药作用时间排序**（从短到长）：利多卡因 < 氯普鲁卡因 < 罗哌卡因 < 丁卡因 < 布比卡因

**心脏毒性**：布比卡因 > 左旋布比卡因 > 罗哌卡因 > 利多卡因 > 丙泊酚

### 局麻方法

| 方法 | 适应症 | 常用药物 |
|------|--------|----------|
| **表面麻醉** | 眼、鼻、口腔、尿道 | 丁卡因、利多卡因 |
| **局部浸润麻醉** | 浅表手术 | 利多卡因 |
| **神经阻滞** | 四肢、牙科 | 利多卡因、罗哌卡因 |
| **椎管内麻醉** | 下腹部、下肢 | 罗哌卡因、布比卡因 |
| **硬膜外麻醉** | 胸/腹/下肢手术 | 罗哌卡因、布比卡因 |

### 局麻药毒性反应

| 项目 | 内容 |
|------|------|
| **早期症状** | 头晕、耳鸣、口舌麻木、视力模糊 |
| **中期症状** | 肌肉震颤、抽搐、意识丧失 |
| **晚期症状** | 呼吸抑制、心律失常、心脏骤停 |
| **治疗** | 停药、氧疗、咪达唑仑/丙泊酚抗惊厥、气管插管 |

## 椎管内麻醉

### 腰麻（蛛网膜下腔阻滞）

| 项目 | 内容 |
|------|------|
| **穿刺部位** | L3-L4（成人）、L4-L5（儿童） |
| **穿刺针** | 25G/27G斜面针 |
| **常用药物** | 布比卡因10-15mg + 葡萄糖50-100mg |
| **麻醉平面** | 感觉平面高于运动平面 |
| **血压下降** | 麻黄碱15mg iv |
| **恶心呕吐** | 昂丹司琼4mg iv |

### 硬膜外麻醉

| 项目 | 内容 |
|------|------|
| **穿刺部位** | 颈/胸/腰段 |
| **常用药物** | 罗哌卡因0.5%-1%、布比卡因0.5% |
| **试验量** | 3ml利多卡因（排除蛛网膜下腔） |
| **镇痛浓度** | 罗哌卡因0.1%-0.2% |
| **手术浓度** | 罗哌卡因0.5%-0.75% |

### 联合腰麻-硬膜外麻醉（CSEA）

| 特点 | 说明 |
|------|------|
| **优点** | 起效快、镇痛完善、可延长时间 |
| **适应症** | 剖宫产、下肢手术 |
| **穿刺点** | L3-L4 |
| **腰麻剂量** | 布比卡因7.5-10mg |
| **硬膜外** | 用于术后镇痛或手术延长 |

### 椎管内麻醉并发症

| 并发症 | 特点 | 处理 |
|--------|------|------|
| **低血压** | 最常见，平面过高、交感神经阻滞 | 麻黄碱、补液、左侧卧位 |
| **全脊麻** | 全部脊神经被阻滞 | 呼吸心跳停止，立即抢救 |
| **呼吸抑制** | 肋间肌/膈肌麻痹 | 面罩给氧，气管插管 |
| **恶心呕吐** | 迷走神经亢进、低血压 | 昂丹司琼、补液 |
| **尿潴留** | 骶神经阻滞 | 导尿 |
| **头痛** | 脑脊液丢失（腰麻后） | 卧床、补液、硬膜外血补丁 |
| **神经损伤** | 穿刺直接损伤 | 激素、神经营养 |

> **口诀**：全脊麻最危险，呼吸心跳全停摆

## 麻醉并发症

### 全麻并发症

| 并发症 | 原因 | 处理 |
|--------|------|------|
| **呼吸抑制** | 麻醉药/肌松药残余 | 纳洛酮、新斯的明 |
| **恶心呕吐** | 吸入麻醉药、阿片类 | 昂丹司琼、托烷司琼 |
| **舌后坠** | 全麻后肌松弛 | 口咽/鼻咽通气道 |
| **喉痉挛** | 分泌物刺激 | 加压给氧、琥珀胆碱 |
| **支气管痉挛** | 哮喘病史 | 氨茶碱、糖皮质激素 |
| **肺误吸** | 饱胃 | 半卧位、吸引、气管插管 |
| **苏醒期躁动** | 疼痛、不适、缺氧 | 镇痛、约束 |
| **恶性高热** | 挥发性麻醉药、琥珀胆碱 | 丹曲林、降温 |

> **口诀**：吐舌后坠加喉痉挛，支气管痉挛肺误吸

### 局麻并发症

| 并发症 | 原因 | 处理 |
|--------|------|------|
| **局麻药毒性** | 剂量过大、血管内注射 | 抗惊厥、脂质溶液 |
| **过敏反应** | 酯类局麻药 | 肾上腺素、糖皮质激素 |
| **高铁血红蛋白血症** | 丁卡因/苯佐卡因 | 亚甲蓝 |

## 疼痛治疗

### 患者自控镇痛（PCA）

| 参数 | 成人常用值 |
|------|------------|
| **背景剂量** | 0-2ml/h |
| **单次剂量** | 1-3ml |
| **锁定时间** | 5-15min |
| **1小时限量** | 4-6ml |
| **4小时限量** | 20-30ml |

**PCA常用配方**：
| 配方 | 药物组合 |
|------|----------|
| 单纯阿片类 | 芬太尼1-2mg + 生理盐水100ml |
| 复合配方 | 芬太尼0.8-1.6mg + 曲马多400-600mg + 生理盐水100ml |
| 静脉PCA | 吗啡10-20mg + 托烷司琼10mg + 生理盐水100ml |

### 多模式镇痛

| 药物类型 | 代表药物 | 作用机制 |
|----------|----------|----------|
| **NSAIDs** | 帕瑞昔布、氟比洛芬酯 | 抑制外周COX |
| **对乙酰氨基酚** | - | 抑制中枢COX |
| **阿片类** | 吗啡、芬太尼 | μ受体激动 |
| **局麻药** | 罗哌卡因 | 阻断Na通道 |
| **抗惊厥药** | 加巴喷丁、普瑞巴林 | 神经病理性疼痛 |
| **抗抑郁药** | 文拉法辛、度洛西汀 | 5-HT/NE再摄取 |

> **口诀**：多模式镇痛是王道，NSAIDs加对乙，阿片兜底加局麻

### 常用镇痛药物剂量

| 药物 | 镇痛剂量 | 峰效时间 | 持续时间 |
|------|----------|----------|----------|
| **吗啡** | 10mg im/iv | 30-60min | 4h |
| **芬太尼** | 50-100μg iv | 3-5min | 30-60min |
| **舒芬太尼** | 0.1-0.2μg/kg | 2-3min | 15-30min |
| **曲马多** | 50-100mg po/im | 2-3h | 6-8h |
| **帕瑞昔布** | 40mg iv | 30min | 12h |
| **氟比洛芬酯** | 50mg iv | 6min | 4-6h |

## 记忆口诀

1. **ASA分级**：「一正二轻三重四死五濒六捐」
   - I正常、II轻疾、III重疾、IV危及生命、V濒死、VI器官捐献

2. **吸入麻醉药选择**：「七氟最常用、异氟肌松好」
   - 七氟烷最常用、异氟烷肌松效果好

3. **肌松药拮抗**：「维库罗库要拮抗，顺阿阿曲Hofmann自消失」
   - 维库溴铵/罗库溴铵需拮抗、顺阿曲库铵/阿曲库铵Hofmann消除

4. **局麻药毒性症状**：「头晕耳鸣嘴里麻，视力模糊眼发花」
   - 早期表现

5. **椎管内麻醉并发症**：「全脊麻最危险，呼吸心跳全停摆」
   - 全脊麻是最危险的并发症

6. **全麻后舌后坠处理**：「托下颌、放口咽、鼻咽通气」
   - 解除舌后坠三步曲

7. **PCA参数**：「背景锁定一小时限量」
   - 背景剂量、锁定时间、1小时限量
