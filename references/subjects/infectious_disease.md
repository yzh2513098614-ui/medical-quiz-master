# 传染病学速查表

> 适用：执业医师考试 / 传染病学考核

---

## 一、法定传染病分类

### 甲类（2种）— 强制管理

| 病名 | 病原体 | 报告时限 |
|------|--------|----------|
| **鼠疫** | 鼠疫杆菌 | 2小时内网络直报 |
| **霍乱** | 霍乱弧菌 | 2小时内网络直报 |

> **记忆口诀**：鼠霍（书活）—— 老鼠活得不安稳，要强制管理

### 乙类（27种）— 严格管理

| 病名 | 病原体 | 潜伏期 | 隔离期 |
|------|--------|--------|--------|
| **艾滋病** | HIV | 6个月~10年 | 首诊后24小时内上报 |
| **病毒性肝炎（甲/乙/丙/丁/戊）** | HAV/HBV/HCV/HDV/HEV | 甲肝:2-6周；乙肝:6周~6月 | 甲肝:隔离至病后3周；乙肝/丙肝:抗病毒治疗 |
| **肺结核** | 结核分枝杆菌 | 2周~数月 | 痰菌阴性后2周 |
| **麻疹** | 麻疹病毒 | 10-14天 | 出疹后5天 |
| **百日咳** | 百日咳杆菌 | 7-10天 | 发病后21天或阵咳后14天 |
| **流行性出血热** | 汉坦病毒 | 7-14天 | 隔离至急性期症状消失 |
| **狂犬病** | 狂犬病毒 | 1-3个月 | 疑似病例即隔离 |
| **乙脑** | 乙型脑炎病毒 | 10-14天 | 隔离至体温正常 |
| **登革热** | 登革病毒 | 5-8天 | 病程超过5天且热退24小时 |
| **细菌性痢疾** | 志贺菌 | 1-3天 | 症状消失后连续2次阴性 |
| **伤寒/副伤寒** | 伤寒杆菌 | 7-14天 | 体温正常15天或2次阴性 |
| **流脑** | 脑膜炎奈瑟菌 | 2-3天 | 症状消失后3天 |
| **猩红热** | A组β溶血链球菌 | 1-7天 | 咽拭子培养阴性或病后7天 |
| **淋病** | 淋病奈瑟菌 | 2-5天 | 性伴侣需同时治疗 |
| **梅毒** | 苍白螺旋体 | 10-90天 | 治疗期间避免性接触 |
| **血吸虫病** | 血吸虫 | 30-60天 | 无需隔离（不人传人） |
| **疟疾** | 疟原虫 | 12-30天 | 疟疾发作停止且原虫消失 |
| **人感染H7N9禽流感** | H7N9病毒 | 7天内 | 体温正常、临床症状消失 |

> **记忆口诀**：乙肝流麻痢伤寒，乙脑出血热狂犬；登革梅毒猩红热，流脑淋病加流感

### 丙类（11种）— 监测管理

| 病名 | 病原体 | 潜伏期 | 隔离期 |
|------|--------|--------|--------|
| **流行性感冒** | 流感病毒 | 1-3天 | 热退后48小时 |
| **手足口病** | EV71/柯萨奇病毒 | 3-5天 | 体温正常、皮疹消退 |
| **感染性腹泻** | 轮状/诺如/大肠杆菌 | 2-7天 | 症状消失 |
| **急性出血性结膜炎** | 腺病毒/肠道病毒70型 | 1-2天 | 隔离至症状消失 |
| **风疹** | 风疹病毒 | 10-21天 | 出疹后5天 |
| **流行性腮腺炎** | 腮腺炎病毒 | 14-21天 | 腮腺消肿或病后21天 |
| **包虫病** | 棘球绦虫幼虫 | 数月~数年 | 不需要隔离 |

> **记忆口诀**：流感腹泻手足口，风疹腮腺监测管

## 二、常见传染病详解

### 病毒性肝炎

#### 甲型肝炎（HAV）

| 项目 | 内容 |
|------|------|
| **病原体** | 甲型肝炎病毒（RNA病毒） |
| **传染源** | 急性期患者+隐性感染者 |
| **传播途径** | 粪-口途径（食物/水/密切接触） |
| **好发人群** | 儿童/青少年（多为亚临床） |
| **临床表现** | 急性黄疸型：发热→黄疸（尿黄/巩膜黄/皮肤黄）ALT显著升高 |
| **诊断** | 抗-HAV IgM阳性 |
| **治疗** | 保肝降酶+对症支持，无需抗病毒 |
| **预防** | 甲肝疫苗（灭活疫苗）；注意饮食卫生 |

> **记忆口诀**：甲肝急黄不慢急，粪口传播好防治

#### 乙型肝炎（HBV）

| 项目 | 内容 |
|------|------|
| **病原体** | 乙型肝炎病毒（DNA病毒） |
| **传染源** | 慢性HBV感染者/急性患者 |
| **传播途径** | 母婴垂直传播、血液传播、性传播 |
| **临床表现** | 急性/慢性/淤胆型；慢性化率与年龄相关（婴幼儿>90%） |
| **诊断标志物** | HBsAg/HBeAg/抗-HBc IgM/HBV DNA |
| **治疗** | 恩替卡韦/替诺福韦抗病毒；干扰素 |
| **预防** | 乙肝疫苗（10μg/0/1/6月）+ HBIG（暴露后） |

**乙肝两对半解读**：

| 模式 | HBsAg | 抗-HBs | HBeAg | 抗-HBe | 抗-HBc | 意义 |
|------|-------|--------|-------|--------|--------|------|
| 大三阳 | + | - | + | - | + | 病毒复制活跃，传染性强 |
| 小三阳 | + | - | - | + | + | 病毒低复制，传染性低 |
| 抗HBs阳性 | - | + | - | - | - | 接种疫苗或既往感染已康复 |

> **记忆口诀**：大三阳HBV复制狂，小三阳复制不张狂

#### 丙型肝炎（HCV）

| 项目 | 内容 |
|------|------|
| **病原体** | 丙型肝炎病毒（RNA病毒） |
| **传播途径** | 血液传播为主（输血/注射/纹身） |
| **特点** | 慢性化率高（>70%），易肝硬化 |
| **诊断** | 抗-HCV IgG阳性 + HCV RNA阳性 |
| **治疗** | DAAs（直接抗病毒药物）：索磷布韦/维帕他韦 |

#### 丁型肝炎（HDV）

- **需依赖HBV才能复制**，与HBV混合/重叠感染会加重病情
- **诊断**：抗-HDV IgG/IgM阳性

#### 戊型肝炎（HEV）

- 类似甲肝，粪-口途径，**孕妇感染后病死率高**
- **诊断**：抗-HEV IgM阳性

### 艾滋病（AIDS/HIV）

| 项目 | 内容 |
|------|------|
| **病原体** | 人类免疫缺陷病毒（HIV-1/2） |
| **传染源** | HIV感染者/AIDS患者（血液/精液/阴道分泌物） |
| **传播途径** | 性接触/血液/母婴垂直 |
| **临床表现** | 急性期（发热/淋巴结肿大）→ 无症状期（8-10年）→ AIDS期 |
| **AIDS期标准** | CD4<200/μL 或机会性感染/肿瘤 |
| **诊断** | 抗-HIV初筛（ELISA）+ 确证（WB）；HIV RNA定量 |
| **治疗** | HAART（鸡尾酒疗法）：2种NRTI + 1种INSTR/NNRTI |
| **预防** | 安全性行为/无偿献血/美沙酮替代/母婴阻断（PEP） |

> **记忆口诀**：艾A——CD4到200以下，或出现机会性肿瘤

**HIV职业暴露处理**：
1. 立即上报 + 伤口处理
2. 24小时内启动PEP
3. 随访监测12周

---

### 肺结核

| 项目 | 内容 |
|------|------|
| **病原体** | 结核分枝杆菌（抗酸染色阳性） |
| **传染源** | 开放性肺结核患者（痰菌阳性） |
| **传播途径** | 飞沫传播 |
| **临床类型** | 原发性肺结核（I型）/ 继发性肺结核（II型）/ 结核性胸膜炎（IV型） |

**继发性肺结核（最常见）**：
- 好发部位：肺尖（第1-2肋间）、上叶后段、下叶背段
- **临床表现**：咳嗽咳痰≥2周、痰血、低热盗汗、消瘦
- **诊断**：痰菌阳性（涂片/培养）+ 胸部X线/CT

**PPD试验（结核菌素试验）**：
| 结果 | 判断标准 |
|------|----------|
| 阴性 | <5mm |
| 阳性 | ≥5mm（一般阳性5-9mm/中度阳性10-19mm/强阳性≥20mm或水泡） |

**隔离期**：痰菌阴性后2周

**治疗原则**：
- 早期/联合/适量/规律/全程
- **标准方案**：2HRZE/4HR（H异烟肼/R利福平/Z吡嗪酰胺/E乙胺丁醇）

> **记忆口诀**：肺结核治疗要趁早，联合适量规律全

---

### 细菌性痢疾（菌痢）

| 项目 | 内容 |
|------|------|
| **病原体** | 志贺菌（福氏/宋内氏） |
| **传染源** | 急性/慢性患者+带菌者 |
| **传播途径** | 粪-口途径（食物/水/接触 |
| **潜伏期** | 1-3天 |
| **临床表现** | 急性：腹痛/腹泻/粘液脓血便/里急后重； 慢性：病程>2月 |
| **诊断** | 大便常规：粘液脓血便；大便培养志贺菌阳性 |
| **治疗** | 喹诺酮类（诺氟沙星/左氧氟沙星）；补液纠酸 |
| **预防** | 隔离患者至连续2次阴性；注意饮食卫生 |

> **记忆口诀**：菌痢三联征——腹痛腹泻脓血便

---

### 伤寒与副伤寒

| 项目 | 内容 |
|------|------|
| **病原体** | 伤寒杆菌/副伤寒甲/乙/丙杆菌 |
| **传染源** | 患者/带菌者 |
| **传播途径** | 粪-口（食物/水） |
| **潜伏期** | 伤寒7-14天；副伤寒1-3天 |
| **临床表现** | 持续高热（稽留热）/表情淡漠/相对缓脉/玫瑰疹/肝脾大/腹胀 |
| **诊断** | 血/骨髓/大便培养伤寒杆菌阳性；肥达反应（O≥1:80，H≥1:160） |
| **并发症** | 肠出血/肠穿孔（最严重，第3周） |
| **治疗** | 氟喹诺酮（首选）/三代头孢；禁用退热剂（易诱发出血） |
| **隔离期** | 体温正常15天或连续2-3次阴性 |

> **记忆口诀**：伤寒稽留高热脉缓，玫瑰疹肝脾大，第三周小心肠穿孔

---

### 流行性脑脊髓膜炎（流脑）

| 项目 | 内容 |
|------|------|
| **病原体** | 脑膜炎奈瑟菌（革兰阴性双球菌） |
| **传染源** | 隐性感染者/患者 |
| **传播途径** | 飞沫传播 |
| **潜伏期** | 2-3天 |
| **好发季节** | 冬春季（1-3月） |
| **临床表现** | 突发高热/剧烈头痛/喷射性呕吐/脑膜刺激征（+）；可有瘀点/瘀斑 |
| **临床类型** | 普通型（90%）/暴发型（休克型/脑膜脑炎型/混合型） |
| **诊断** | 脑脊液：米汤样混浊（化脓性）；细菌培养（+）；皮肤瘀点涂片（+） |
| **暴发型治疗** | 抗菌+糖皮质激素+扩容纠酸+血管活性药物 |
| **预防** | A群流脑疫苗；密切接触者预防性服用利福平 |
| **隔离期** | 症状消失后3天 |

> **记忆口诀**：流脑冬春突发高热头痛吐，瘀点瘀斑脑膜征阳

---

### 流行性乙型脑炎（乙脑）

| 项目 | 内容 |
|------|------|
| **病原体** | 乙型脑炎病毒（虫媒病毒，蚊传播） |
| **传染源** | 猪（主要传染源）/鸟类 |
| **传播途径** | 三带喙库蚊叮咬 |
| **潜伏期** | 10-14天 |
| **好发季节** | 夏秋季（7-9月） |
| **临床表现** | 高热/意识障碍/抽搐/呼吸衰竭/脑膜刺激征；严重者出现病理反射 |
| **诊断** | 乙脑IgM抗体阳性（发病3-5天后）；脑脊液非化脓性改变 |
| **治疗** | 对症支持：降温/止痉/甘露醇降颅压；无特效抗病毒药 |
| **预防** | 乙脑疫苗（减毒活疫苗）；灭蚊 |

> **记忆口诀**：乙脑夏秋高热抽搐加意识，三大后遗症（失语/瘫痪/精神异常）

---

### 肾综合征出血热（HFRS）

| 项目 | 内容 |
|------|------|
| **病原体** | 汉坦病毒（RNA病毒） |
| **传染源** | 鼠类（黑线姬鼠/褐家鼠） |
| **传播途径** | 接触鼠类排泄物/气溶胶/污染食物 |
| **潜伏期** | 7-14天 |
| **临床表现** | 发热/三痛（头痛/腰痛/眼眶痛）/三红（面红/颈红/胸红）/球结膜充血 |
| **病程五期** | 发热期→低血压休克期→少尿期→多尿期→恢复期 |
| **诊断** | 抗汉坦病毒IgM阳性；汉坦病毒RNA阳性 |
| **治疗** | 利巴韦林；扩容纠酸；透析（少尿期） |
| **预防** | 灭鼠；疫苗接种 |

> **记忆口诀**：出血热三红三痛五期过，少尿多尿要透析

---

### 狂犬病

| 项目 | 内容 |
|------|------|
| **病原体** | 狂犬病毒（弹状病毒科） |
| **传染源** | 带毒动物（犬/猫/蝙蝠/浣熊） |
| **传播途径** | 病兽咬伤/抓伤/舔舐破损皮肤 |
| **潜伏期** | 1-3个月（可短至1周，长至1年） |
| **临床表现** | 恐水/怕风/咽肌痉挛/进行性瘫痪/意识模糊 |
| **诊断** | 狂犬病IgM/IgG抗体阳性；脑组织切片Negri小体（+） |
| **治疗** | 暴露后处理：伤口清洗+疫苗+免疫球蛋白；无治愈方法 |
| **预防** | 犬类疫苗接种；暴露前免疫 |

> **潜伏期记忆**：头面部咬伤潜伏期短（1周）；远端咬伤潜伏期长（1年）

---

### 疟疾

| 项目 | 内容 |
|------|------|
| **病原体** | 疟原虫（间日/三日/恶性/卵圆疟原虫） |
| **传染源** | 疟疾患者/带虫者 |
| **传播途径** | 按蚊叮咬；输血；母婴垂直传播 |
| **潜伏期** | 间日疟14天/恶性疟12天/三日疟30天 |
| **典型表现** | 寒战高热出汗（三寒三热）：寒战期→高热期→大汗期；周期性发热 |
| **凶险类型** | 脑型疟（昏迷/抽搐/脑膜刺激征）→死亡率高 |
| **诊断** | 厚薄血膜涂片查见疟原虫；RDT快速诊断 |
| **治疗** | 氯喹（敏感株）/青蒿素衍生物（恶性疟）；伯氨喹（防复发，用于间日疟） |
| **预防** | 防蚊灭蚊；预防性用药（氯喹/多西环素）；疫苗（Bavarian Nordic疟疾疫苗） |

> **记忆口诀**：疟疾寒战高热出汗三寒热，恶性最重脑型疟

## 三、传染病防控总览

### 传播途径与预防措施

| 传播途径 | 典型疾病 | 预防措施 |
|----------|----------|----------|
| **飞沫传播** | 流感/流脑/麻疹/肺结核 | 口罩/通风/疫苗 |
| **空气传播** | 肺结核/水痘/麻疹 | N95口罩/负压病房 |
| **接触传播** | 菌痢/手足口/急性结膜炎 | 洗手/消毒/隔离 |
| **血液/体液传播** | 乙肝/丙肝/艾滋病 | 安全注射/性防护/血制品筛查 |
| **虫媒传播** | 乙脑/疟疾/登革热 | 灭蚊/防蚊 |
| **粪-口传播** | 甲肝/伤寒/菌痢 | 饮食卫生/疫苗 |
| **垂直传播** | 乙肝/艾滋病/梅毒 | 母婴阻断（PEP） |

### 隔离期总表（重要数据）

| 疾病 | 隔离期 |
|------|--------|
| **甲肝** | 病后3周 |
| **乙肝/丙肝** | 抗病毒治疗期间 |
| **肺结核** | 痰菌阴性后2周 |
| **菌痢** | 症状消失+2次阴性 |
| **伤寒** | 体温正常15天或2次阴性 |
| **流脑** | 症状消失后3天 |
| **乙脑** | 体温正常 |
| **麻疹** | 出疹后5天 |
| **猩红热** | 病后7天或培养阴性 |
| **流感** | 热退后48小时 |
| **手足口病** | 体温正常+皮疹消退 |

> **记忆口诀**：呼麻流腮2天热，手足水痘疹消止；菌痢伤寒两阴性，肺结痰阴2周止

### 突发公共卫生事件

**分级**：特别重大（I级）/ 重大（II级）/ 较大（III级）/ 一般（IV级）

**上报要求**：
- 甲类传染病：2小时内
- 乙类传染病：24小时内
- 丙类传染病：监测区24小时内

**需要报告的聚集性**：
- 同一学校/单位/社区出现≥3例相同症状
- 短时间内出现≥5例食源性疾病
