## 第一章 细胞的基本功能

### 核心概念

| 概念 | 要点 |
|------|------|
| **跨膜物质转运** | 单纯扩散、易化扩散（通道/载体）、主动转运（原发/继发）、膜泡运输 |
| **信号转导** | 受体类型：离子通道型、G蛋白耦联型、酶联型 |
| **细胞电活动** | 静息电位（K+外流）、动作电位（Na+内流为主）、局部电位 |
| **神经-肌接头** | 1:1传递、Ca2+触发ACh释放、终板电位 |

### 高频考点

- 动作电位特点：全或无、不衰减传导、有不应期
- 局部电位特点：等级性、可总和、衰减传导
- 钠钾泵功能：维持细胞内外Na+/K+梯度
- 受体分类及代表：N2-AChR（离子通道）、M受体（G蛋白）、胰岛素受体（酶联）

### 记忆口诀

> **动作电位**：全或无、不衰减，有不应期，不能总和
> **局部电位**：等级性、可总和，衰减传导，无不应期
> **三大受体**：离子通道直接开，G蛋白靠第二信使，酶联自带催化剂

## 第二章 血液

### 核心概念

| 概念 | 要点 |
|------|------|
| **血浆** | 晶体渗透压（维持细胞内外水平衡）、胶体渗透压（维持血管内外水平衡） |
| **血细胞** | 红细胞（携氧）、白细胞（免疫）、血小板（凝血） |
| **凝血系统** | 内源性（XII-XI-IX-VIII）、外源性（III-VII）、共同途径 |
| **抗凝系统** | 抗凝血酶III、肝素、蛋白质C/S |
| **纤溶系统** | 纤溶酶原→纤溶酶、tPA、uPA、α2-抗纤溶酶 |

### 高频考点

- 血细胞比容：男性约40-50%，女性约37-48%
- 凝血因子：只有III（组织因子）不存在于血浆中
- 血型：ABO血型（IgM天然抗体）、Rh血型（IgG免疫抗体）
- 交叉配血：主侧供者红细胞+受者血清，次侧相反

### 记忆口诀

> **凝血因子**：唯一没有III，Ca倒卵磷脂
> **血型抗体**：ABO有IgM天然，Rh有IgG免疫
> **晶体胶体**：晶体维持水平衡，胶体维持血浆容量

## 第三章 血液循环

### 核心概念

| 概念 | 要点 |
|------|------|
| **心肌电生理** | 静息电位、动作电位（0/1/2/3/4期）、有效不应期 |
| **心脏泵血** | 每搏输出量、心输出量=HR×CO、射血分数（正常≥55%） |
| **动脉血压** | 收缩压（心室收缩期）、舒张压（心室舒张期）、脉压=收缩压-舒张压 |
| **调节机制** | 神经（交感/副交感）、体液（肾素-血管紧张素-醛固酮系统） |

### 高频考点

- 心室肌动作电位：0期Na+内流，1期K+外流，2期（平台期）Ca2+内流，3期K+外流，4期静息期
- 影响心输出量因素：心率、前负荷、后负荷、心肌收缩力
- 冠脉循环特点：主要在舒张期供血，左心室压力高
- 脑循环特点：血流量稳定，脑屏障保护

### 记忆口诀

> **心室动作电位**：零钠一钾二钙平台，三钾四静一辈子
> **心输出量公式**：心输出量等于心率乘每搏，心率一定量每搏

## 第四章 呼吸

### 核心概念

| 概念 | 要点 |
|------|------|
| **肺通气** | 潮气量、补吸气量、补呼气量、残气量、肺活量用力肺活量 |
| **肺换气** | 气体分压差、通气/血流比值（正常0.84）、肺扩散容量 |
| **气体运输** | O2（Hb结合，98.5%）、CO2（HCO3-，88%） |
| **呼吸调节** | 中枢化学感受器（延髓）、外周化学感受器（颈动脉体/主动脉体） |

### 高频考点

- 肺总量=TLC=VC+RC，肺活量(VC)=补吸气量+潮气量+补呼气量，用力肺活量(FVC)≠肺活量(VC)
- V/Q比值正常=0.84，比值↑提示通气过度（死腔），比值↓提示血流过剩（动静脉短路）
- Hb氧解离曲线：左移（Hb对O2亲和力↑），右移（体温↑/PCO2↑/[H+]↑/2,3-DPG↑）
- 低O2对呼吸的刺激：外周感受器为主，急性低O2通过外周化学感受器兴奋呼吸

### 记忆口诀

> **V/Q比值**：V高Q低通气多，V低Q高血流多，比例正常0.84
> **氧解离曲线**：右移降温降PCO2，酸多DPG来帮忙

## 第五章 消化和吸收

### 核心概念

| 概念 | 要点 |
|------|------|
| **消化液** | 唾液（α-淀粉酶）、胃液（盐酸+胃蛋白酶+内因子）、胰液（所有消化酶）、胆汁（胆盐） |
| **胃肠运动** | 蠕动、容受性舒张、紧张性收缩、分节运动 |
| **吸收** | 小肠是主要吸收部位，糖/蛋白质的吸收为继发性主动转运 |

### 高频考点

- 胰液分泌调节：促胰液素（ HCl → S细胞 → 水盐多酶少）和CCK（蛋白/脂肪 → I细胞 → 酶多水盐少）
- 胃液分泌三期：头期（20-30%）、胃期（60-70%最多）、肠期（10%最少）
- 胆汁：不含脂肪酶，胆盐是吸收脂肪的必需物质
- 铁的吸收：Fe3+→Fe2+（维生素C/胃酸还原），吸收部位在十二指肠和空肠上段

### 记忆口诀

> **胆汁成分**：胆盐胆固醇胆色素，脂肪酶没有胰酶有
> **胃液三期**：头期神经胃期神经体液，肠期最少体液为主
> **胰腺分泌**：CCK酶多水盐少，促胰液素水盐多酶少

## 第六章 能量代谢与体温

### 核心概念

| 概念 | 要点 |
|------|------|
| **能量代谢** | 食物卡价、氧热价（糖21/脂肪19.7/蛋白18.9 kJ/L）、呼吸商（RQ） |
| **基础代谢** | 基础代谢率BMR（%），受甲状腺激素、体温、年龄、性别影响 |
| **体温** | 核心体温（37℃±0.5℃）、体表温度、散热方式（辐射/传导/对流/蒸发） |

### 高频考点

- 产热：骨骼肌收缩战栗、甲状腺激素、肾上腺素
- 散热：辐射（60%）、传导+对流（15%）、蒸发（15%发汗）、肺呼吸道散热
- 体温调节中枢：视前区-下丘脑前部（PO/AH）是中枢温度感受器所在
- 致热原：IL-1、IL-6、TNF-α、PGE2作用于下丘脑体温调定点

### 记忆口诀

> **能量计算**：氧热价糖21脂19.7，蛋白18.9最常用
> **散热方式**：辐射散热最主要，蒸发散热要发汗

## 第七章 尿的生成和排出

### 核心概念

| 概念 | 要点 |
|------|------|
| **肾小球滤过** | 肾小球滤过率GFR（正常125ml/min）、滤过分数（FF=18-20%） |
| **肾小管重吸收** | 近端小管（65-70%等渗性重吸收）、髓袢（25-30%）、远曲小管+集合管（调节性） |
| **尿液浓缩** | 逆流倍增机制、髓质高渗区、ADH调节 |
| **清除率** | 肾清除率=Ux×V/Px，菊粉清除率=Cin=GFR，物质清除率反映肾小球/小管功能 |

### 高频考点

- 肾小球有效滤过压=毛细血管血压-血浆胶体渗透压-囊内压
- 葡萄糖重吸收：100%在近端小管主动重吸收（继发性主动转运）
- 渗透性利尿：甘露醇、山梨醇、糖尿病（溶质在肾小管）
- 肾糖阈：血糖180-200mg/dl，达到此阈值葡萄糖开始从尿中排出

### 记忆口诀

> **肾小球滤过**：滤过压等于血胶囊
> **葡萄糖重吸收**：葡萄糖全在近端回，肾糖阈180-200

## 第八章 感觉器官

### 核心概念

| 概念 | 要点 |
|------|------|
| **感受器** | 感受器电位（局部电位）、发生器电位、适应现象（快/慢适应） |
| **视觉** | 眼的折光系统（成像）、视网膜感光系统（视锥/视杆细胞）、暗适应明适应 |
| **听觉** | 耳蜗微音器电位、行波理论、基底膜震动与音频定位 |
| **前庭** | 半规管（旋转加速度）、椭圆囊/球囊（直线加速度/头位变化） |

### 高频考点

- 视锥细胞：明视觉、色觉，中央凹只有视锥细胞
- 视杆细胞：暗视觉，只有一类视杆蛋白，对明暗无分辨力
- 视紫红质：视杆细胞感光色素，维生素A参与合成
- 耳蜗微音器电位：与声波频率和强度呈线性关系

### 记忆口诀

> **视锥视杆**：锥细胞明视觉色觉，杆细胞暗视觉无色
> **适应类型**：快适应触觉压觉，慢适应肌梭痛觉

## 第九章 神经系统

### 核心概念

| 概念 | 要点 |
|------|------|
| **神经元** | 神经元功能、神经纤维分类（有髓/无髓、A/B/C型）、轴突运输 |
| **突触传递** | 化学突触（定向/非定向）、电突触、神经递质（乙酰胆碱/NE/多巴胺/5-HT/GABA/Glu） |
| **反射** | 反射弧（5部分）、中枢突触传递特征（单向传播/中枢延搁/总和/兴奋节律改变/后放/敏感性） |
| **神经系统** | 感觉传入（特异性/非特异性投射）、运动传出（锥体系/锥体外系） |

### 高频考点

- 兴奋性突触后电位（EPSP）：Na+内流为主，去极化
- 抑制性突触后电位（IPSP）：Cl-内流为主，超极化
- 神经递质受体：N型（烟碱型）=离子通道，M型（毒蕈碱型）=G蛋白
- 内脏痛：定位模糊、持续时间长、可有牵涉痛

### 记忆口诀

> **EPSP/IPSP**：兴奋钠内流去极化，抑制氯内流超极化
> **锥体系锥体外系**：锥体系随意精细运动，锥体外系肌张力姿势

## 第十章 内分泌

### 核心概念

| 概念 | 要点 |
|------|------|
| **激素** | 激素分类（蛋白肽类/类固醇/胺类）、作用机制（受体后信号通路） |
| **下丘脑-垂体轴** | 下丘脑调节肽（TRH/CRH/GnRH/PRL/SS等）、垂体前叶（促激素）、垂体后叶（ADH/缩宫素） |
| **甲状腺激素** | T3/T4、甲状腺功能亢进/减退、呆小症 |
| **肾上腺** | 皮质（糖皮质激素/盐皮质激素/性激素）、髓质（肾上腺素/去甲肾上腺素） |
| **胰岛** | α细胞（分泌胰高血糖素）、β细胞（分泌胰岛素） |

### 高频考点

- 甲状腺激素：T4为主（量大），T3活性强（效价高）
- 糖皮质激素：升高血糖、抗炎、抗过敏、抗休克、免疫抑制
- 胰岛素：降低血糖、促进糖原/脂肪/蛋白质合成
- 应激反应：下丘脑-垂体-肾上腺皮质轴激活，ACTH和糖皮质激素升高

### 记忆口诀

> **甲状腺激素**：T3活性强T4量多，甲亢高代谢甲减低代谢
> **胰岛素**：胰岛素降血糖，胰高血糖素升血糖
> **应激**：应激时下丘脑垂体肾上腺，ACTH皮质醇一起升

## 第十一章 生殖

### 核心概念

| 概念 | 要点 |
|------|------|
| **男性生殖** | 睾丸功能（精子生成+睾酮分泌）、下丘脑-垂体-性腺轴 |
| **女性生殖** | 卵巢功能（卵子生成+雌激素/孕激素分泌）、月经周期（增生期/分泌期/月经期） |
| **妊娠** | 受精、着床、胎盘内分泌、临产启动 |

### 高频考点

- 月经周期：增生期（雌激素）、分泌期（孕激素）、月经期
- 雌激素：促进子宫内膜增生、促进乳腺导管发育、促排卵前LH峰
- 孕激素：促进子宫内膜分泌、抑制子宫收缩、升高体温
- 人绒毛膜促性腺激素（hCG）：维持妊娠黄体，诊断早孕

### 记忆口诀

> **月经周期**：增生期雌激素，分泌期孕激素，月经期激素撤退
> **雌激素作用**：子增乳长促排卵，促排促黄体生成
