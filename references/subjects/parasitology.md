# 寄生虫学速查表

## 一、原虫 (Protozoa)

---

### 1. 肠道原虫

#### 溶组织内阿米巴 (Entamoeba histolytica) — 阿米巴痢疾

| 项目 | 内容 |
|------|------|
| **病原体** | 溶组织内阿米巴（组织型/肠腔型滋养体） |
| **传染源** | 慢性患者/带包囊者 |
| **传播途径** | **粪-口传播**，食入被包囊污染的食物/水 |
| **潜伏期** | **1~4周** |
| **临床表现** | 阿米巴痢疾（果酱样脓血便）、阿米巴肝脓肿（右下肺野） |
| **病理特征** | 烧瓶样溃疡（黏膜下层蔓延） |
| **诊断** | 粪便涂片找滋养体/包囊，ELISA查抗原，PCR |
| **治疗** | 首选**甲硝唑**（灭滴灵），后续用巴龙霉素/双碘喹啉清除肠腔 |

> **记忆口诀**："阿米巴，粪口传，果酱便，肝脏单（脓肿），甲硝唑有效"

#### 蓝氏贾第鞭毛虫 (Giardia lamblia) — 贾第鞭毛虫病/旅游者腹泻

| 项目 | 内容 |
|------|------|
| **病原体** | 蓝氏贾第鞭毛虫（梨形鞭毛虫） |
| **传染源** | 人（主要）、牛、狗 |
| **传播途径** | **粪-口传播**，水源性爆发（包囊在水中存活数周） |
| **潜伏期** | **1~3周** |
| **临床表现** | 旅游者腹泻（水样便/脂肪泻）、腹胀/嗳气/恶臭便，儿童慢性感染可致发育迟缓 |
| **诊断** | 粪便涂片找包囊/滋养体，ELISA/IFA查抗原 |
| **治疗** | 首选**甲硝唑**，也可用替硝唑或呋喃唑酮 |

> **记忆口诀**："贾第鞭毛，梨形成双，旅游腹泻，水源难防，甲硝唑帮"

#### 隐孢子虫 (Cryptosporidium spp.) — 隐孢子虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 微小隐孢子虫 (*C. parvum*) 等 |
| **传染源** | 感染的人和动物（牛为主） |
| **传播途径** | **粪-口传播**，水源性爆发（包囊对氯耐受） |
| **潜伏期** | **1~12天** |
| **临床表现** | 免疫正常：自限性水样腹泻；**免疫缺陷者**（HIV/AIDS）：严重持续性腹泻（水样便>10次/天） |
| **诊断** | 粪便抗酸染色（红色卵圆形包囊），免疫荧光，PCR |
| **治疗** | 无特效药；免疫重建（HIV患者用ART）是关键；硝唑尼特有一定疗效 |

> **记忆口诀**："隐孢子虫免疫败，自限腹泻免疫在，艾滋病者水样便，硝唑尼特试试看"

### 2. 血液/组织原虫

#### 疟原虫 (Plasmodium spp.) — 疟疾

| 项目 | 内容 |
|------|------|
| **病原体** | 恶性疟原虫(*P. falciparum*)、间日疟原虫(*P. vivax*)、三日疟原虫(*P. malariae*)、卵形疟原虫(*P. ovale*) |
| **传染源** | 疟疾患者/带虫者 |
| **传播途径** | **按蚊叮咬**（中华按蚊/嗜人按蚊→间日疟；恶性按蚊→恶性疟） |
| **潜伏期** | 间日疟**13~15天**，恶性疟**9~14天**，三日疟**25~28天** |
| **临床表现** | **寒热往来**（冷→热→汗退热呈周期性）： <br>• 间日疟：48h周期 <br>• 三日疟：72h周期 <br>• 恶性疟：不规则热，可致脑疟（意识障碍/抽搐）、肾功能衰竭 |
| **诊断** | 厚薄血膜染色（吉姆萨）找疟原虫，RDT快速检测，PCR |
| **治疗** | 氯喹（敏感株）；**青蒿素类**（恶性疟首选）；伯氨喹（间日疟/卵形疟防复发）；恶性疟用青蒿素联合疗法(ACT) |

> **记忆口诀**："疟原虫，按蚊咬，寒热战栗周期性；间日三天各不同，脑疟昏迷最危急；青蒿素类治恶性，伯氨喹防间日复"

#### 杜氏利什曼原虫 (Leishmania donovani) — 黑热病

| 项目 | 内容 |
|------|------|
| **病原体** | 杜氏利什曼原虫（前鞭毛体/无鞭毛体） |
| **传染源** | 患者/病犬（内脏利什曼病） |
| **传播途径** | **白蛉叮咬** |
| **潜伏期** | **10天~1年**（通常2~4月） |
| **临床表现** | 黑热病三联征：**发热+肝脾肿大+贫血**；全血细胞减少，球蛋白升高（白蛋白倒置）；长期不愈可致**黑化病**（皮肤色素沉着） |
| **诊断** | 骨髓/脾/淋巴结穿刺找无鞭毛体，rK39试纸条，PCR |
| **治疗** | 首选**葡萄糖酸锑钠**（斯锑黑克）；两性霉素B（耐药/复发者） |

> **记忆口诀**："黑热病，白蛉咬，发热脾大又贫血，骨髓穿刺找原虫，锑剂治疗是正道"

## 二、蠕虫 (Helminths)

---

### 1. 线虫 (Nematodes)

#### 似蚓蛔线虫 (Ascaris lumbricoides) — 蛔虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 似蚓蛔线虫（成虫寄生小肠） |
| **传染源** | 感染者（雌虫日产卵20万枚） |
| **传播途径** | **粪-口传播**，食入污染土壤的蔬菜/水源 |
| **潜伏期** | **2个月左右** |
| **临床表现** | 腹痛（脐周为主）、厌食/异食癖；儿童可致营养不良； **并发症**：胆道蛔虫症、肠梗阻、阑尾炎、胰腺炎 |
| **诊断** | 粪便涂片找蛔虫卵（受精/未受精）；吐出/排出成虫可确诊 |
| **治疗** | 首选**阿苯达唑**（2岁以上）或**甲苯达唑**；并发症需内镜/手术 |

> **记忆口诀**："蛔虫经口入肠中，脐周腹痛营养差；胆道梗阻最常见，阿苯达唑效最佳"

#### 十二指肠钩虫/美洲钩虫 (Ancylostoma duodenale / Necator americanus) — 钩虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 十二指肠钩口线虫，美洲板口线虫（成虫寄生小肠） |
| **传染源** | 感染者 |
| **传播途径** | **泥土传播**：丝状蚴经皮肤侵入（赤足下地），也可垂直传播 |
| **潜伏期** | 皮肤侵入后**5~6周** |
| **临床表现** | 皮炎（钩蚴移行，局部红斑/瘙痒/"着土痒"）；**贫血**（缺铁性，慢性失血）；儿童可致发育迟缓 |
| **诊断** | 粪便涂片/饱和盐水漂浮法找虫卵 |
| **治疗** | 阿苯达唑/甲苯达唑；同时补充铁剂纠正贫血 |

> **记忆口诀**："钩虫经皮侵入体，皮炎贫血来就诊，泥土里头丝状蚴，铁剂补血是正理"

#### 蠕形住肠线虫 (Enterobius vermicularis) — 蛲虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 蠕形住肠线虫（蛲虫） |
| **传染源** | 感染者（自身感染+相互感染） |
| **传播途径** | **肛门-手-口直接感染**（雌虫夜间在肛周产卵→瘙痒→手指污染→自感染） |
| **潜伏期** | **2~6周** |
| **临床表现** | **肛门瘙痒**（夜间加重，可致失眠）；**幼儿园儿童**集体感染；女孩可致阴道炎 |
| **诊断** | 夜间肛门拭子法（透明胶带法）找虫卵 |
| **治疗** | 阿苯达唑（2周后重复1次防复发）；局部止痒（高锰酸钾坐浴） |

> **记忆口诀**："蛲虫夜行肛门痒，幼儿园里互相传，阿苯达唑两周五，高锰酸钾坐浴安"

#### 毛首鞭形线虫 (Trichuris trichiura) — 鞭虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 毛首鞭形线虫（鞭虫，成虫寄生盲肠） |
| **传染源** | 感染者 |
| **传播途径** | **粪-口传播**，食入污染土壤的虫卵 |
| **潜伏期** | **2~3个月** |
| **临床表现** | 轻症：腹痛/腹泻；重度感染（>1000条）：儿童可致直肠脱垂/营养不良 |
| **诊断** | 粪便涂片找鞭虫卵（腰鼓形，棕黄色） |
| **治疗** | 阿苯达唑/甲苯达唑 |

> **记忆口诀**："鞭虫盲肠来寄生，腰鼓虫卵棕黄形，重度感染脱肛见，阿苯达唑把虫清"

#### 旋毛形线虫 (Trichinella spiralis) — 旋毛虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 旋毛形线虫（旋毛虫） |
| **传染源** | 猪（主要）、野猪、熊等 |
| **传播途径** | **生食或半生食含幼虫囊包的肉**（野猪烤肉/腌肉/香肠最常见） |
| **潜伏期** | **1~4周** |
| **临床表现** | 发热+眼睑/面部水肿+**肌肉疼痛**（腓肠肌/咬肌/呼吸肌）+嗜酸性粒细胞↑ |
| **诊断** | 肌肉活检找幼虫（旋涡状）；血清学（ELISA/IFA） |
| **治疗** | 阿苯达唑（杀死肠内成虫+移行期幼虫）；严重者加用糖皮质激素 |

> **记忆口诀**："旋毛生肉把虫得，眼睑水肿肌肉痛，嗜酸粒细胞升高，阿苯达唑是主将"

### 2. 吸虫 (Trematodes)

#### 华支睾吸虫 (Clonorchis sinensis) — 肝吸虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 华支睾吸虫（肝吸虫） |
| **传染源** | 感染的人/猫/狗 |
| **传播途径** | **生食/半生食含囊蚴的淡水鱼/虾**（广东/广西食鱼生习俗） |
| **潜伏期** | **1~2月** |
| **临床表现** | 上腹痛/腹胀/厌食/腹泻；肝胆管阻塞→胆管炎/胆囊炎/胆石症；长期可致**胆管上皮癌** |
| **诊断** | 粪便/十二指肠液找虫卵；ELISA查抗体 |
| **治疗** | 首选**吡喹酮**；阿苯达唑也有效 |

> **记忆口诀**："肝吸虫，爱鱼生，胆管阻塞肝区痛，长期感染胆管癌，吡喹酮服立大功"

#### 卫氏并殖吸虫 (Paragonimus westermani) — 肺吸虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 卫氏并殖吸虫（肺吸虫） |
| **传染源** | 感染的人/肉食哺乳动物 |
| **传播途径** | **生食/半生食含囊蚴的蟹/蝲蛄**（江浙沪/东北） |
| **潜伏期** | **1~3月** |
| **临床表现** | 咳嗽+胸痛+**铁锈色或棕褐色痰**（肺部成虫）；可游走至**脑**（头痛/抽搐/偏瘫）；皮下结节 |
| **诊断** | 痰/粪便找虫卵；血清学；胸部CT（空洞/结节） |
| **治疗** | 首选**吡喹酮**；脑型需脱水/手术；硫氯酚也有效 |

> **记忆口诀**："肺吸虫，蟹里藏，铁锈色痰肺部长，脑型癫痫又偏瘫，吡喹酮来帮忙"

#### 日本血吸虫/曼氏血吸虫/埃及血吸虫 — 血吸虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 日本血吸虫（*Schistosoma japonicum*，中国流行）；曼氏血吸虫（*S. mansoni*）；埃及血吸虫（*S. haematobium*，非洲） |
| **传染源** | 感染者（曼/日→人；埃→人/灵长类） |
| **传播途径** | **钉螺**为中间宿主，尾蚴经皮肤侵入（湖水/河水接触史） |
| **潜伏期** | **4~8周** |
| **临床表现** | 急性：发热+荨麻疹+肝脾大+腹泻（"伤寒样"）；慢性：门脉高压（肝脾大/腹水/食管静脉曲张）/ **Egyptian urination**（血尿/膀胱钙化） |
| **诊断** | 粪便/尿液沉渣找虫卵（尾蚴孵化法）；血清学；肝脾超声 |
| **治疗** | 首选**吡喹酮**（单剂口服） |

> **记忆口诀**："血吸虫，钉螺帮，湖水洗手皮内钻，急性发热慢性硬化，腹水静脉曲张张；Egyptian血尿膀胱伤，吡喹酮一剂安康"

### 3. 绦虫 (Cestodes)

#### 猪带绦虫 (Taenia solium) — 猪带绦虫病/囊虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 猪带绦虫（成虫寄生小肠）；**猪囊尾蚴**（幼虫寄生组织） |
| **传染源** | 感染者（自体感染）；猪（中间宿主） |
| **传播途径** | **粪-口**：食入污染的虫卵（自体感染/异体感染）→囊虫病；食入**米猪肉**（含囊尾蚴）→绦虫病 |
| **潜伏期** | 绦虫病2~3月；囊虫病数月~数年 |
| **临床表现** | 绦虫病：腹痛/肛门瘙痒/粪便排节片；囊虫病：**脑囊虫病**（癫痫/颅压↑/精神障碍）、皮下结节、眼的玻璃体内囊虫 |
| **诊断** | 粪便找绦虫卵/节片；CT/MRI（脑内钙化灶）；血清/CSF ELISA |
| **治疗** | 绦虫病：吡喹酮/阿苯达唑；**脑囊虫病**：阿苯达唑+糖皮质激素；癫痫加抗癫痫药；眼囊虫病需手术 |

> **记忆口诀**："猪带绦虫米猪藏，囊虫入脑癫痫发，节片排出肛门痒，阿苯达唑压全场"

#### 牛带绦虫 (Taenia saginata) — 牛带绦虫病

| 项目 | 内容 |
|------|------|
| **病原体** | 牛带绦虫（无钩绦虫） |
| **传染源** | 感染者 |
| **传播途径** | **粪-口**：食入含囊尾蚴的未熟牛肉 |
| **潜伏期** | **2~3月** |
| **临床表现** | 腹痛/肛门瘙痒/排节片（主动逸出肛门）；症状较猪带绦虫轻 |
| **诊断** | 粪便找孕节（子宫侧支>15~20支）；肛周拭子 |
| **治疗** | 吡喹酮或阿苯达唑；南瓜籽+槟榔合剂（传统） |

> **记忆口诀**："牛带绦虫无钩强，牛肉未熟把虫长，节片主动逸肛门，吡喹酮服保安康"

#### 细粒棘球绦虫 (Echinococcus granulosus) — 包虫病/棘球蚴病

| 项目 | 内容 |
|------|------|
| **病原体** | 细粒棘球绦虫（成虫寄生犬/狼小肠）；**棘球蚴**（幼虫寄生肝/肺等） |
| **传染源** | 犬（终宿主）；羊/牛（中间宿主） |
| **传播途径** | **犬粪**中的虫卵污染食物/手，经口侵入人体 |
| **潜伏期** | **数月~数十年** |
| **临床表现** | 肝囊肿（右上腹痛/肝大/黄疸）；肺囊肿（咳嗽/胸痛/咯血）；骨囊肿（病理性骨折）；过敏性休克（囊肿破裂时） |
| **诊断** | 影像（超声/CT/MRI）+血清学（IHA/ELISA）；Casoni皮试（不特异） |
| **治疗** | 手术切除（首选）；阿苯达唑（术前/无法手术者）；PAIR（穿刺抽吸+杀虫剂） |

> **记忆口诀**："包虫病，狗传羊，肝肺囊肿潜伏长，手术切除是首选，阿苯达唑来帮忙"

## 三、蠕虫四大天王对比表

| 特征 | 蛔虫 | 钩虫 | 蛲虫 | 鞭虫 |
|------|------|------|------|------|
| **学名** | 似蚓蛔线虫 | 十二指肠钩虫/美洲钩虫 | 蠕形住肠线虫 | 毛首鞭形线虫 |
| **传播途径** | 粪-口（污染蔬菜） | **泥土**（经皮肤） | **肛门-手-口** | 粪-口 |
| **主要症状** | 脐周痛/营养不良/胆道梗阻 | **贫血**/皮炎 | **肛门瘙痒**（夜间） | 腹泻/直肠脱垂 |
| **好发人群** | 儿童 | 农民（赤足） | **幼儿园儿童** | 儿童 |

## 七、10大记忆口诀汇总

1. **"阿米巴，粪口传，果酱便，肝脏单（脓肿），甲硝唑有效"** — 溶组织内阿米巴
2. **"贾第鞭毛，梨形成双，旅游腹泻，水源难防，甲硝唑帮"** — 蓝氏贾第鞭毛虫
3. **"隐孢子虫免疫败，自限腹泻免疫在，艾滋病者水样便，硝唑尼特试试看"** — 隐孢子虫
4. **"疟原虫，按蚊咬，寒热战栗周期性；间日三天各不同，脑疟昏迷最危急；青蒿素类治恶性，伯氨喹防间日复"** — 疟原虫
5. **"黑热病，白蛉咬，发热脾大又贫血，骨髓穿刺找原虫，锑剂治疗是正道"** — 杜氏利什曼原虫
6. **"蛔虫经口入肠中，脐周腹痛营养差；胆道梗阻最常见，阿苯达唑效最佳"** — 似蚓蛔线虫
7. **"钩虫经皮侵入体，皮炎贫血来就诊，泥土里头丝状蚴，铁剂补血是正理"** — 钩虫
8. **"蛲虫夜行肛门痒，幼儿园里互相传，阿苯达唑两周五，高锰酸钾坐浴安"** — 蠕形住肠线虫
9. **"肝吸虫，爱鱼生，胆管阻塞肝区痛，长期感染胆管癌，吡喹酮服立大功"** — 华支睾吸虫
