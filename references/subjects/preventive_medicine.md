## 第一章 流行病学原理与方法

### 核心概念

| 概念 | 要点 |
|------|------|
| **流行病学** | 研究疾病和健康状态在人群中的分布及影响因素，用于预防控制疾病的学科 |
| **发病率** | 某时期内新发病例数/同期暴露人口数（反映疾病风险） |
| **患病率** | 某时点现患病例数/同期总人口数（与病程有关） |
| **死亡率** | 某时期内死亡人数/同期总人口数（反映人群死亡水平） |
| **病死率** | 某时期内死亡人数/同期发病人数（反映疾病严重程度） |

### 流行病学研究方法

| 方法 | 设计 | 方向 | 暴露因素 | 优点 | 缺点 |
|------|------|------|----------|------|------|
| **队列研究** | 从暴露→结局 | 前瞻性/回顾性 | 已知暴露分组 | 可计算RR/AR，能确立因果 | 耗时/耗资/失访偏倚 |
| **病例对照研究** | 从结局→暴露 | 回顾性 | 病例组vs对照组 | 快速/经济/适合罕见病 | 只能算OR，易产生回忆偏倚 |
| **随机对照试验(RCT)** | 随机分组 | 前瞻性 | 干预vs对照 | 证据等级最高 | 伦理限制/外部效度 |
| **横断面研究** | 同时调查 | 现况 | 无时间方向 | 快速/经济 | 不能确立因果 |
| **病例报告** | 单个病例 | 描述性 | - | 发现新病/罕见病 | 无对照，证据弱 |

### 高频考点

**队列研究指标：**
- RR（相对危险度）= I₁/I₀，RR>1表示危险因素，RR<1表示保护因素
- AR（归因危险度）= I₁-I₀，表示暴露导致的额外风险
- AR% = (I₁-I₀)/I₁ × 100%，表示暴露人群中的归因比例

**病例对照研究指标：**
- OR（比值比）= ad/bc，当发病率低时OR≈RR
- OR>1表示危险因素，OR<1表示保护因素

**灵敏度与特异度：**

| 指标 | 定义 | 计算公式 |
|------|------|----------|
| **灵敏度（真阳性率）** | 患者中正确检出阳性 | Se = TP/(TP+FN) × 100% |
| **特异度（真阴性率）** | 非患者中正确检出阴性 | Sp = TN/(TN+FP) × 100% |
| **假阳性率** | 非患者中错误检出阳性 | FP/(TN+FP) |
| **Youden指数** | 综合评价 | Se+Sp-1 |
| **阳性预测值** | 阳性中真患者比例 | PV+ = TP/(TP+FP) |
| **阴性预测值** | 阴性中真非患者比例 | PV- = TN/(TN+FN) |

### 记忆口诀

> **队列研究**：从因到果是队列，前瞻回顾都可以，计算RR和AR
> **病例对照**：从果到因病例对照，只能计算OR值
> **灵敏度特异度**：灵敏度的分子是患者阳性，特异度的分子是非患者阴性
> **发病率与患病率**：发病是新的，患病是累计，发病率低时患病率≈发病率×病程

## 第二章 疾病预防与控制

### 核心概念

| 预防级别 | 目标 | 举例 |
|----------|------|------|
| **一级预防** | 病因预防，减少发病 | 健康教育/疫苗接种/改善环境 |
| **二级预防** | 早发现早诊断早治疗 | 筛查/体检/高危人群定期监测 |
| **三级预防** | 防止伤残促进康复 | 治疗/康复/心理干预 |

### 高频考点

**疾病自然史：**
- 潜伏期→前驱期→临床期→康复/后遗症/死亡
- 潜伏期是流行病学调查研究的重要时期

**三级预防内容：**

| 级别 | 预防内容 | 具体措施 |
|------|----------|----------|
| **一级** | 病因预防 | 健康宣教、养成良好生活习惯、预防接种、劳动保护 |
| **二级** | 三早预防 | 定期体检、高危人群筛查、做到三早（早发现/早诊断/早治疗） |
| **三级** | 临床预防 | 治疗原发病、防止并发症、康复治疗、临终关怀 |

### 记忆口诀

> **三级预防**：一级病因在苗头，二级三早来护航，三级康复减伤残

## 第三章 预防接种

### 核心概念

| 疫苗类型 | 特点 | 举例 |
|----------|------|------|
| **活疫苗** | 免疫效果好，持久，免费 | 卡介苗、脊髓灰质炎糖丸、麻疹 |
| **灭活疫苗** | 安全，免疫效果较弱需多次 | 乙肝、流感、百白破 |
| **类毒素** | 细菌外毒素甲醛处理 | 白喉、破伤风 |
| **亚单位疫苗** | 提取有效抗原成分 | 乙肝CHO疫苗 |
| **多糖疫苗** | 细菌荚膜多糖 | 肺炎球菌、脑膜炎球菌 |

### 计划免疫程序

| 疫苗 | 接种时间 | 接种途径 | 加强时间 |
|------|----------|----------|----------|
| **卡介苗** | 出生时 | 皮内注射 | 无需加强 |
| **乙肝疫苗** | 0、1、6月 | 肌内注射 | 无需加强（高危可加强） |
| **脊髓灰质炎糖丸** | 2、3、4月，4岁 | 口服 | 4岁加强 |
| **百白破** | 3、4、5月，18月 | 肌内注射 | 6岁加强白破 |
| **麻疹疫苗** | 8月，18月 | 皮下注射 | 6岁加强麻腮风 |
| **乙脑疫苗** | 8月，2岁 | 皮下注射 | 6岁加强 |
| **A群流脑** | 6月，9月 | 皮下注射 | 3岁、6岁加强A+C |

### 高频考点

- 活疫苗免疫效果好但有禁忌（免疫缺陷者禁用，孕妇禁用）
- 灭活疫苗安全性好但需多次接种
- 乙肝疫苗：0-1-6月程序，国产CHO/酵母表达，高危人群需加强
- 脊髓灰质炎：2-3-4月龄，4岁加强；发热/急性病延缓接种
- 冷链管理：疫苗运输储存需2-8℃，保证效价

### 记忆口诀

> **活疫苗**：活疫苗效果好，免疫缺陷要禁用，孕妇禁用要记清
> **乙肝免疫程序**：零一六月要记牢，三针接种保护好
> **卡介苗**：出生第一针，皮内注射不要揉

## 第四章 社区卫生

### 核心概念

| 概念 | 要点 |
|------|------|
| **社区诊断** | 了解社区人群健康状况、资源、问题，为制定社区卫生计划提供依据 |
| **健康教育** | 通过信息传播和行为干预，促进人们自愿采纳健康生活方式 |
| **健康促进** | 包括健康教育+社会支持+环境改善+政策改变 |
| **社区卫生服务** | 以社区为范围，以人群为对象，提供综合性、连续性、可及性卫生服务 |

### 社区诊断内容

| 组成部分 | 主要内容 |
|----------|----------|
| **社会诊断** | 社区人群健康状况、生活质量、社会环境 |
| **流行病学诊断** | 疾病分布、主要健康问题及危险因素 |
| **行为诊断** | 找出与健康问题相关的行为因素 |
| **环境诊断** | 识别影响健康的环境因素 |

### 高频考点

**社区诊断步骤：**
1. 确定社区范围和人群
2. 收集社区健康信息
3. 识别社区主要健康问题
4. 确定优先解决问题的顺序
5. 制定社区卫生计划

**健康教育知信行模式（KAP）：**
- Knowledge（知识）→ Attitude（态度）→ Practice（行为）
- 只有掌握知识，才能转变态度，最终改变行为

**健康促进五大活动领域（Ottawa宪章）：**
1. 建立促进健康的公共政策
2. 创造支持性环境
3. 强化社区行动
4. 发展个人技能
5. 调整卫生服务方向

### 记忆口诀

> **社区诊断**：社会诊断定背景，流行病学找问题，行为诊断找原因，环境诊断找外因
> **知信行模式**：知识先行转态度，行为改变是目标
> **健康促进五大领域**：政策环境社区动，个人服务调整

## 第五章 营养与食品卫生

### 核心概念

| 营养素 | 主要功能 | 缺乏/过量 |
|--------|----------|-----------|
| **蛋白质** | 构成组织、酶、激素，免疫功能 | 缺乏：水肿、贫血、免疫力↓；过量：增加肾负担 |
| **碳水化合物** | 主要供能来源 | 缺乏：能量不足；过量：肥胖、糖尿病 |
| **脂肪** | 储能、保温、脂溶性维生素吸收 | 缺乏：脂溶性维生素吸收障碍；过量：肥胖、心血管病 |
| **维生素A** | 视紫红质合成，免疫，皮肤上皮 | 缺乏：夜盲症、干眼症、毛囊角化 |
| **维生素D** | 促进钙吸收、骨骼矿化 | 缺乏：佝偻病/骨软化；过量：高钙血症 |
| **维生素C** | 抗氧化、胶原合成、铁吸收 | 缺乏：坏血病（牙龈出血/皮下出血） |
| **钙** | 骨骼牙齿、神经肌肉 | 缺乏：佝偻病/骨质疏松；过量：肾结石 |
| **铁** | 血红蛋白、肌红蛋白 | 缺乏：缺铁性贫血（小红低色素） |
| **碘** | 甲状腺激素合成 | 缺乏：地方性甲状腺肿/克汀病 |

### 膳食指南（中国居民膳食指南2022）

| 内容 | 要点 |
|------|------|
| **食物多样** | 每天≥12种，每周≥25种 |
| **谷类为主** | 谷薯类200-400g/d，粗细搭配 |
| **多吃蔬果** | 蔬菜300-500g/d，水果200-350g/d |
| **适量鱼禽蛋瘦肉** | 120-200g/d，优选鱼和禽 |
| **少盐少油** | 盐<5g/d，油25-30g/d |
| **控糖限酒** | 添加糖<25g/d，酒精<15g/d |
| **规律进餐** | 三餐定时定量，≥3次/d |

### 食品安全

| 概念 | 要点 |
|------|------|
| **食物中毒** | 摄入有毒有害食物引起的急性亚急性疾病 |
| **细菌性食物中毒** | 沙门菌（蛋肉）、副溶血弧菌（海产品）、金葡菌（剩饭），多为胃肠炎型 |
| **化学性食物中毒** | 亚硝酸盐（腌制菜）、有机磷（农药），发病快无季节性 |
| **有毒动植物中毒** | 河豚鱼（神经毒）、毒蘑菇（肝肾毒）、发芽土豆（龙葵素） |
| **真菌毒素中毒** | 黄曲霉毒素（霉变粮食→肝癌）、赤霉病麦（呕吐毒素） |

### 高频考点

**膳食宝塔分层：**
- 第一层：谷薯类
- 第二层：蔬菜水果
- 第三层：鱼禽肉蛋
- 第四层：奶豆坚果
- 第五层：油盐

**食物中毒处理原则：**
1. 立即停止可疑食物
2. 催吐/洗胃/导泻
3. 对症治疗
4. 及时上报卫生部门
5. 保留可疑食物和患者标本

### 记忆口诀

> **维生素A缺乏**：夜盲干眼毛囊化，皮肤干燥要记牢
> **维生素C缺乏**：牙龈出血皮下斑，坏血病名记心间
> **铁缺乏**：小细胞低色素，补铁治疗要记住
> **食物中毒**：上吐下泻腹痛急，细菌化学要分清
> **膳食指南**：谷类为主蔬果多，少盐少油控糖酒

## 第六章 环境卫生

### 核心概念

| 环境因素 | 对健康的影响 |
|----------|--------------|
| **大气污染** | 呼吸道疾病、心血管疾病、肺癌（PM2.5/PM10/SO₂/NOₓ/CO/O₃） |
| **水污染** | 介水传染病（霍乱/伤寒/痢疾）、地方性氟中毒、砷中毒 |
| **土壤污染** | 重金属（铅/砷/镉）→痛痛病（水俣病） |
| **噪声污染** | 听力下降、神经衰弱、心血管损害 |
| **辐射** | 白血病、癌症、胎儿畸形 |

### 饮水卫生

| 指标 | 意义 |
|------|------|
| **总大肠菌群** | 粪便污染指示菌 |
| **菌落总数** | 反映水体清洁程度 |
| **余氯** | 反映消毒效果，≥0.3mg/L（管网末梢） |
| **浊度** | 反映水的清澈程度 |
| **pH值** | 6.5-8.5 |

### 职业病

| 职业病 | 危害因素 | 主要表现 | 预防措施 |
|--------|----------|----------|----------|
| **矽肺** | 二氧化硅粉尘 | 咳嗽、胸闷、呼吸困难，X线结节影 | 湿式作业/戴口罩/定期体检 |
| **煤工尘肺** | 煤尘 | 慢阻肺+纤维化 | 同上 |
| **铅中毒** | 铅 | 腹绞痛、贫血、周围神经炎 | 作业环境通风/禁止手接触 |
| **苯中毒** | 苯 | 急性： CNS抑制；慢性：再障、白血病 | 通风/个体防护/监测 |
| **汞中毒** | 汞 | 急性：腐蚀性胃肠炎；慢性：汞毒性震颤（意向性） | 同上 |
| **噪声聋** | 噪声 | 听力下降、耳鸣，早期高频听力丧失 | 隔声/消声/耳防护 |

### 高频考点

**职业病特点：**
-病因明确
- 有剂量-反应关系
- 群发性
- 多数可预防
- 需要特异性诊断

**职业病诊断依据：**
1. 职业史
2. 现场劳动卫生学调查
3. 临床表现为主要依据
4. 排除其他类似疾病

### 记忆口诀

> **职业病诊断**：职业史是前提，现场调查来证实，临床表现作依据，排除其他类似病
> **矽肺**：咳嗽胸闷呼吸难，X线结节来诊断
> **铅中毒**：腹绞痛来贫血伴，神经炎来周围端

## 第七章 统计与研究设计

### 核心概念

| 统计指标 | 公式 | 应用 |
|----------|------|------|
| **均数** | 所有观测值之和/观测数 | 正态分布资料集中趋势 |
| **中位数** | 排序后位于中间的数值 | 偏态分布资料 |
| **标准差** | √(Σ(x-x̄)²/n) | 表示数据离散程度 |
| **率** | 某现象发生数/可能发生总数×比例基数 | 表示某现象发生频率或强度 |
| **构成比** | 某一组成部分数/各部分总数×100% | 表示事物内部结构 |

### 假设检验

| 检验方法 | 适用资料 | 用途 |
|----------|----------|------|
| **t检验** | 正态分布的计量资料 | 两组均数比较 |
| **方差分析** | 正态分布的计量资料 | 多组均数比较 |
| **χ²检验** | 计数资料 | 率的比较/关联性分析 |
| **秩和检验** | 非正态分布资料 | 两组/多组比较 |

### 记忆口诀

> **均数中位数**：正态分布用均数，偏态分布用中位
> **t检验**：两组比较用t，多组比较用方差
> **χ²检验**：分类资料用卡方，构成比和率
