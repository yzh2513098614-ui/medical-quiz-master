## 第一章 蛋白质的结构与功能

### 核心概念

| 概念 | 要点 |
|------|------|
| **氨基酸** | 20种常见氨基酸，分为非极性疏水性、极性中性、酸性（Asp/Glu）、碱性（Lys/Arg/His） |
| **蛋白质结构** | 一级结构（肽键）、二级结构（α螺旋/β折叠/β转角）、三级结构（整条链空间排布）、四级结构（亚基） |
| **蛋白质变性** | 空间结构破坏，一级结构不变，理化性质改变，生物活性丧失 |
| **分离纯化** | 盐析、电泳、层析、超速离心、透析 |

### 高频考点

- 20种氨基酸：酸性氨基酸（Asp、Glu），碱性氨基酸（Lys、Arg、His）
- 蛋白质变性因素：热、酸、碱、有机溶剂、重金属盐、紫外线
- 呈色反应：茚三酮（蓝紫色，Pro黄色）、双缩脲（检测肽键）、Folin-酚（检测酪氨酸/色氨酸）
- 等电点pI：蛋白质净电荷为0时的pH

### 记忆口诀

> **氨基酸分类**：酸 Asp Glu，碱 Lys Arg His
> **蛋白质变性**：变性不变一级，肽键不断理化改

## 第二章 核酸的结构与功能

### 核心概念

| 概念 | 要点 |
|------|------|
| **核苷酸组成** | 碱基（嘌呤A/G、嘧啶C/T/U）、戊糖（核糖/脱氧核糖）、磷酸 |
| **DNA结构** | 双螺旋（右手B-DNA，直径2nm，螺距3.4nm，每圈10.5对碱基）、碱基配对（A=T 2氢键，G≡C 3氢键） |
| **RNA类型** | mRNA（模板，5'帽3'尾）、tRNA（三叶草形，转运氨基酸）、rRNA（核糖体组成） |

### 高频考点

- DNA双螺旋：磷酸-脱氧核糖骨架在外侧，碱基内侧配对
- 核酸变性：双链解链为单链，不破坏磷酸二酯键
- Tm（熔解温度）：50%双链解链的温度，与GC含量正相关
- 核酸杂交：不同来源单链核酸配对

### 记忆口诀

> **碱基配对**：DNA有T，RNA有U；CG配对三氢键，AT配对两氢键
> **RNA类型**：mRNA是模板，tRNA来转运，rRNA组成核糖体

## 第三章 酶学

### 核心概念

| 概念 | 要点 |
|------|------|
| **酶的结构** | 全酶=酶蛋白+辅助因子（辅酶/辅基），辅酶疏松结合可分离，辅基紧密结合不分离 |
| **米氏方程** | v = (Vmax×[S])/(Km+[S])，Km为米氏常数=Vmax一半时的底物浓度 |
| **酶抑制** | 竞争性（Km↑ Vmax不变）、非竞争性（Km不变 Vmax↓）、反竞争性（Km↓ Vmax↓） |
| **酶调节** | 变构调节、共价修饰、酶原激活、同工酶 |

### 高频考点

- 辅酶与维生素：B1→TPP、B2→FAD/FMN、B3→NAD+/NADP+、B5→CoA、B6→PLP、B7→生物素、B9→FH4、B12→甲基B12
- 酶活性中心：与底物结合的区域，由必需基团组成
- 不可逆抑制：有机磷（胆碱酯酶）、重金属盐（巯基酶）、氰化物（细胞色素氧化酶）
- 酶原激活：胃蛋白酶原→胃蛋白酶，胰蛋白酶原→胰蛋白酶

### 记忆口诀

> **米氏方程**：V等于Vmax乘S除以K加S
> **酶抑制**：竞争V不变K变大，非竞K不变V下降，反竞两个都下降

## 第四章 糖代谢

### 核心概念

| 概念 | 要点 |
|------|------|
| **糖酵解** | 场所：细胞质，产物：丙酮酸+ATP，产能：2ATP |
| **有氧氧化** | 场所：细胞质+线粒体，产物：CO2+H2O+ATP，产能：30-32ATP |
| **糖原代谢** | 糖原合成（UDPG为中间产物，糖原合酶）、糖原分解（磷酸化酶） |
| **糖异生** | 部位：肝/肾，原料：乳酸、甘油、氨基酸，逆糖酵解途径 |

### 高频考点

- 糖酵解关键酶：己糖激酶/葡萄糖激酶、磷酸果糖激酶-1（最重要调节酶）、丙酮酸激酶
- 三羧酸循环：乙酰CoA+草酰乙酸→柠檬酸→α-酮戊二酸→琥珀酸→延胡索酸→苹果酸→草酰乙酸
- 磷酸戊糖途径：产生NADPH（用于生物合成）和核糖-5-磷酸（核酸合成）
- 乳酸循环（Cori循环）：肌肉产生乳酸→肝糖异生→肝糖原

### 记忆口诀

> **糖酵解关键酶**：六改果去丙酮酸
> **三羧酸循环**：柠檬异酮琥琥延苹果，来回一圈两个碳

## 第五章 脂类代谢

### 核心概念

| 概念 | 要点 |
|------|------|
| **β-氧化** | 部位：线粒体，产物：乙酰CoA、FADH2、NADH，产能：每次β氧化生成14ATP |
| **脂肪酸合成** | 部位：细胞质，原料：乙酰CoA（来自线粒体，通过柠檬酸-丙酮酸循环转运） |
| **酮体** | 合成：肝线粒体，利用：肝外组织（心/脑），成分：乙酰乙酸、β-羟丁酸、丙酮 |
| **胆固醇代谢** | 合成原料：乙酰CoA，限速酶：HMG-CoA还原酶，转化产物：胆汁酸、类固醇激素 |

### 高频考点

- 胆固醇合成：3分子乙酰CoA→HMG-CoA→甲羟戊酸→鲨烯→胆固醇
- 胆汁酸合成：胆固醇在肝细胞转化为胆汁酸（初级胆汁酸：胆酸/鹅去氧胆酸）
- 血浆脂蛋白：CM（运送膳食TG）、VLDL（运送肝源性TG）、LDL（运送胆固醇至外周）、HDL（逆向转运胆固醇）
- 酮症：糖尿病/饥饿时，脂肪动员增加，酮体生成超过利用，酮症酸中毒

### 记忆口诀

> **β-氧化**：β氧化在 线粒体，脱氢加水再脱氢，硫解生成乙酰CoA
> **酮体**：酮体生成在肝脏，肝脏没有利用酶，肌肉大脑用得好

## 第六章 生物氧化

### 核心概念

| 概念 | 要点 |
|------|------|
| **呼吸链** | 复合物I（NADH-泛醌还原酶）、II（琥珀酸-泛醌还原酶）、III（泛醌-细胞色素c还原酶）、IV（细胞色素c氧化酶） |
| **ATP生成** | 氧化磷酸化：呼吸链复合物I/II/III/IV递氢递电子，偶联ATP合酶（F0+F1）合成ATP |
| **高能化合物** | 高能磷酸键（ATP/ADP/磷酸肌酸）、高能硫酯键（乙酰CoA、脂酰CoA） |

### 高频考点

- NADH呼吸链：NADH→复合物I→CoQ→复合物III→Cytc→复合物IV→O2，生成2.5ATP
- FADH2呼吸链：FADH2→复合物II→CoQ→复合物III→Cytc→复合物IV→O2，生成1.5ATP
- ATP合酶：F0（质子通道）+F1（催化部位），质子回流驱动ATP合成
- 抑制剂：鱼藤酮（复合物I）、萎锈灵（复合物II）、抗霉素A（复合物III）、CN-/CO（复合物IV）

### 记忆口诀

> **呼吸链复合物**：一NAD二黄蛋，三Q三红四氧
> **氧化磷酸化**：NADH两条半，FADH2一条半

## 第七章 氨基酸代谢

### 核心概念

| 概念 | 要点 |
|------|------|
| **脱氨基作用** | 氧化脱氨（氨基酸氧化酶）、转氨（ALT/AST，磷酸吡哆醛为辅酶）、联合脱氨（转氨+氧化脱氨） |
| **氨代谢** | 氨有毒，血氨运输：Gln（谷氨酰胺）、Ala（丙氨酸），肝代谢：尿素循环 |
| **尿素循环** | 部位：肝线粒体+细胞质，产物：无毒尿素，消耗：4ATP |
| **一碳单位** | 来源：His/Trp/Ser/Gly/Met，载体：FH4（叶酸），参与嘌呤/胸腺嘧啶合成 |

### 高频考点

- 氨基酸脱羧基：组氨酸→组胺（舒血管）、色氨酸→5-HT（神经递质）、谷氨酸→GABA（抑制性神经递质）
- 苯丙氨酸/酪氨酸代谢：苯丙酮尿症（PKU，苯丙氨酸羟化酶缺乏）、白化病（酪氨酸酶缺乏）
- 甲硫氨酸循环：Met→SAM（活性甲基供体）→SAH→HCY→Met，维生素B12和叶酸参与
- 糖异生氨基酸：丙酮酸异生途径中的所有氨基酸（除外Lys/Leu）

### 记忆口诀

> **尿素循环**：瓜氨酸精氨酸琥珀酸，最后生成尿素来排氨
> **一碳单位**：记住His-Trp-Ser-Gly-Met，载体都是四氢叶酸

## 第八章 核苷酸代谢

### 核心概念

| 概念 | 要点 |
|------|------|
| **嘌呤核苷酸合成** | 从头合成（核糖-5-磷酸+氨基酸+CO2+NH3），补救合成（嘌呤碱+PRPP） |
| **嘧啶核苷酸合成** | 从头合成：乳清酸→尿嘧啶→UMP，补救合成：嘧啶碱+PRPP |
| **嘌呤分解** | 嘌呤→黄嘌呤→尿酸，人类缺乏尿酸酶，嘌呤代谢终产物为尿酸 |
| **痛风** | 嘌呤代谢障碍导致高尿酸血症，别嘌醇抑制黄嘌呤氧化酶治疗 |

### 高频考点

- 嘌呤环原子来源：N1（Asp）、C2/N3（ Gln）、C4-C5-N7（Gly）、C8（FH4）、C6（CO2）
- 嘧啶环原子来源：N1/C4-C5-C6（Asp）、C2（CO2）、N3（Gln）
- 抗代谢物：6-MP（抑制HGPRT，补救途径）、5-FU（抑制胸苷酸合酶）、MTX（抑制二氢叶酸还原酶）
- 核苷酸还原： NDP→dNDP（核糖核苷酸还原酶），DNA合成所需

### 记忆口诀

> **嘌呤合成**：天冬酰谷氨酰，甘氨酸来帮忙，二氧化碳加氮源
> **痛风治疗**：别嘌醇抑黄嘌呤氧化酶，碱化尿液多喝水

## 第九章 基因信息传递

### 核心概念

| 概念 | 要点 |
|------|------|
| **DNA复制** | 半保留复制，关键酶：DNA拓扑异构酶、DNA聚合酶III（主酶）、DNA连接酶 |
| **RNA转录** | DNA→RNA，关键酶：RNA聚合酶（真核I/II/III分别转录rRNA/mRNA/tRNA） |
| **蛋白质合成** | mRNA为模板，tRNA转运氨基酸，核糖体为场所，经历起始/延长/终止 |

### 高频考点

- DNA聚合酶：5'→3'方向合成，需RNA引物（引物酶），无校对功能（DNApol I有3'→5'外切酶校对）
- 原核RNA聚合酶：全酶=核心酶+σ因子（识别启动子），核心酶=α2ββ'ω
- 启动子：原核Pribnow盒（TATAAT）、真核TATA盒/GC盒/CAAT盒
- 核糖体：原核30S小亚基+50S大亚基=70S，真核40S+60S=80S

### 记忆口诀

> **DNA复制**：半保留，不连续，引物酶来开头，连接酶来封口
> **RNA转录**：RNA聚合酶，不需引物，5到3延长

## 第十章 基因表达调控

### 核心概念

| 概念 | 要点 |
|------|------|
| **原核调控** | 操纵子模型（启动子+操纵基因+结构基因+调节基因），乳糖操纵子（负调控/CAP正调控）、色氨酸操纵子（衰减子机制） |
| **真核调控** | 转录水平调控（染色质重塑/组蛋白修饰/转录因子）、转录后调控（可变剪接/编辑）、翻译调控、蛋白质加工调控 |
| **基因工程** | 限制性内切酶、载体（质粒/噬菌体）、连接、转化、筛选 |

### 高频考点

- 乳糖操纵子：无葡萄糖时，CAP与cAMP结合激活转录，阻遏蛋白与操纵基因结合抑制
- 色氨酸操纵子：Trp充足时，阻遏蛋白结合操纵基因；Trp不足时，阻遏蛋白失活，起始转录
- 顺式作用元件：启动子、增强子、沉默子（可距离很远，有方向性）
- 反式作用因子：转录因子、激活因子、抑制因子

### 记忆口诀

> **乳糖操纵子**：有乳糖阻遏蛋白走，无乳糖有cAMP CAP来激活
> **真核调控**：染色质开放转录活，甲基化沉默乙酰化活

## 第十一章 细胞信号转导

### 核心概念

| 概念 | 要点 |
|------|------|
| **受体类型** | 细胞膜受体（离子通道/G蛋白耦联/酶联受体）、细胞内受体（甾体激素/甲状腺激素） |
| **主要通路** | cAMP/PKA通路（肾上腺素→β受体→Gs→AC→cAMP→PKA）、IP3/DAG/PKC通路（肾上腺素→α受体→Gq→PLC→IP3+DAG→PKC） |
| **第二信使** | cAMP、cGMP、IP3、DAG、Ca2+、NO |

### 高频考点

- G蛋白：Gs（激活AC）、Gi（抑制AC）、Gq（激活PLC）、Go（抑制Ca2+通道）
- 酪氨酸激酶受体：胰岛素受体、生长因子受体（EGF/PDGF/FGF）、自身磷酸化激活
- 酪氨酸激酶偶联受体：细胞因子受体（IL、EPO、IFN），通过JAK-STAT通路
- 常见信号分子：激素（内分泌）、神经递质（旁分泌）、自分泌（自身分泌）

### 记忆口诀

> **G蛋白**：Gs激活腺苷酸环化酶，Gi抑制它，Gq激活磷脂酶C
> **第二信使**：环AMP、环GMP，磷脂酰肌醇三磷酸

## 第十二章 肝生物化学

### 核心概念

| 概念 | 要点 |
|------|------|
| **肝代谢功能** | 糖代谢（糖原合成/分解/糖异生）、脂代谢（脂肪酸β氧化/酮体生成/胆固醇代谢）、蛋白质代谢（氨基酸代谢/尿素合成） |
| **生物转化** | 第一相反应（氧化/还原/水解）、第二相反应（结合反应：葡萄糖醛酸/硫酸/谷胱甘肽/甘氨酸/乙酰化） |
| **胆色素代谢** | 胆红素生成（衰老红细胞→血红素→胆绿素→胆红素）→肝内代谢→肠道→尿胆素原/粪胆素原→尿胆素/粪胆素 |

### 高频考点

- 黄疸分类：溶血性（UCB↑）、肝细胞性（CB/UCB均↑）、阻塞性（CB↑）
- 黄疸鉴别：直接胆红素（CB）=结合胆红素，间接胆红素（UCB）=未结合胆红素
- 黄疸实验：尿胆红素（阻塞性/肝细胞性阳性）、尿胆原（溶血性↑）、粪胆原
- 生物转化酶系：细胞色素P450单加氧酶系最重要，需NADPH和O2参与

### 记忆口诀

> **黄疸**：溶血UCB高，阻塞CB高，肝细胞都高
> **生物转化**：一相氧化还原水解，二相结合来排毒

## 第十三章 维生素

### 核心概念

| 维生素 | 辅酶形式 | 功能 |
|--------|----------|------|
| **B1（硫胺素）** | TPP | α-酮酸脱羧酶辅酶，参与碳代谢 |
| **B2（核黄素）** | FAD、FMN | 氢传递体，参与氧化还原反应 |
| **B3（烟酸）** | NAD+、NADP+ | 氢传递体，脱氢酶辅酶 |
| **B5（泛酸）** | CoA | 酰基转移，糖/脂/蛋白代谢 |
| **B6（吡哆醇）** | PLP | 氨基转移酶辅酶，参与氨基酸代谢 |
| **B7（生物素）** | 生物素 | 羧基转移酶辅酶 |
| **B9（叶酸）** | FH4 | 一碳单位载体，嘌呤/嘧啶合成 |
| **B12（钴胺素）** | 甲基B12、5'-脱氧腺苷B12 | 甲基转移（甲硫氨酸合成酶） |
| **维生素C** | - | 抗氧化，促进Fe吸收，参与胶原合成 |
| **维生素K** | - | γ-羧化，促进凝血因子II/VII/IX/X合成 |
| **维生素A** | - | 视黄醇，参与暗视觉，维持上皮 |
| **维生素D** | - | 1,25-(OH)2-D3，调节钙磷代谢 |
| **维生素E** | - | 抗氧化，保护生物膜 |

### 高频考点

- B1缺乏：脚气病（干性/湿性/混合型），丙酮酸脱氢酶和α-酮戊二酸脱氢酶功能受损
- B2缺乏：口角炎、舌炎、阴囊皮炎
- B12/叶酸缺乏：巨幼红细胞性贫血（生成DNA障碍）
- 维生素K缺乏：凝血障碍（II/VII/IX/X需γ-羧化）

### 记忆口诀

> **B族维生素辅酶**：B1TPP酮脱羧，B2黄素FAD/B2，B3烟酰胺 NAD，B5泛酸来酰基，B6醛磷酸，B7羧基来转移
> **叶酸B12**：叶酸四氢 folate，B12钴胺素，脱氧腺苷甲基
