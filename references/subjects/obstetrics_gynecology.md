# 妇产科学速查表

> 306考纲覆盖，约占总内科33%

## 第一章 产科

### 妊娠诊断

| 项目 | 要点 |
|------|------|
| **早期诊断** | 停经（最重要）、HCG（受精后7-10天可测，8-10周达峰>100000）、超声（孕5周可见孕囊，6-7周可见胎心） |
| **中晚期诊断** | 子宫增大（宫底高度：满20周脐下1横指/24周脐上1横指/28周脐上3横指）、胎动（初产妇18-20周/经产妇16-18周）、胎心（120-160次/分） |
| **胎产式** | 胎体纵轴与母体纵轴的关系（纵产式/横产式） |
| **胎先露** | 最先进入骨盆入口的部分（头先露/臀先露/肩先露） |
| **胎方位** | 先露部指示点与母体骨盆的关系（枕先露/面先露/臀先露/肩先露） |

### 高频考点

- HCG：正常妊娠，受精后第7天开始分泌，着床后进入母血，8-10周达峰>100000，产后2周消失
- 孕周计算：末次月经第1天算起，280天（40周）预产期
- 胎心：12周后多普勒可闻及，正常120-160次/分
- 宫底高度：20周末脐下1横指，24周末脐上1横指，28周末脐上3横指，32周末脐与剑突之间

### 记忆口诀

> **孕周记忆**：12宫底在耻骨上，16周脐下1指量，20周脐下1横指，24周脐上1指强，28周脐上3横指，32周剑下2横指
> **胎心正常值**：120到160，一分钟里不用愁

### 流产

| 类型 | 病史 | 妇科检查 | 超声 | 处理 |
|------|------|----------|------|------|
| **先兆流产** | 少量阴道出血/下腹痛 | 宫颈口未开 | 孕囊存活 | 休息/黄体酮/叶酸 |
| **难免流产** | 阴道出血增多/腹痛加剧 | 宫颈口已开 | 孕囊变形 | 刮宫/缩宫素 |
| **不全流产** | 部分妊娠物排出/出血多 | 宫颈口开/组织堵塞 | 部分孕囊 | 刮宫+输血 |
| **完全流产** | 妊娠物完全排出/出血减少 | 宫颈口关闭 | 宫腔线清晰 | 观察/促进宫缩 |
| **稽留流产** | 胚胎死亡>4周无排出 | 子宫小于孕周 | 无胎心 | 雌激素+刮宫 |

### 高频考点

- 流产最常见原因：染色体异常（50-60%）
- 先兆流产处理：禁止性生活，卧床休息，黄体酮10-20mg/d肌注
- 稽留流产：胚胎死亡后胎盘溶解，凝血功能障碍（DIC）风险
- 刮宫适应证：难免流产/不全流产/稽留流产

### 记忆口诀

> **流产鉴别**：先兆宫颈口关，难免宫颈口开，不全组织在口塞，完全流产宫干净
> **稽留处理**：雌激素备血再刮宫，防止凝血障碍生

### 异位妊娠

| 项目 | 要点 |
|------|------|
| **定义** | 孕卵在子宫体腔以外着床（输卵管95%、卵巢、腹腔、阔韧带、宫颈） |
| **临床表现** | 停经+腹痛+阴道流血（三联征），宫颈举痛/摇摆痛，后穹隆穿刺（不凝血） |
| **诊断** | 超声（宫内无孕囊/附件区包块/盆腔积液）、HCG（动态观察，48h增长<50%提示异常）、腹腔镜（金标准） |
| **治疗** | 药物治疗（甲氨蝶呤MTX，适用于生命体征稳定/包块<3.5cm/HCG<5000）、手术治疗（输卵管切除/开窗取胚） |
| **高危因素** | 输卵管炎/盆腔炎/异位妊娠史/输卵管手术史/宫内节育器 |

### 高频考点

- 异位妊娠三联征：停经+腹痛+阴道流血
- 宫颈举痛：宫颈无病变但摆痛，提示盆腔病变（异位妊娠/盆腔炎）
- 后穹隆穿刺：抽出不凝血>3ml提示腹腔内出血
- HCG动态：正常妊娠48h增长>53%，异位妊娠增长<50%
- 异位妊娠最常见部位：壶腹部（55%）、峡部（25%）、伞部（17%）、间质部（3%）

### 记忆口诀

> **异位妊娠三联征**：停经腹痛阴道流，宫颈举痛后穹抽
> **治疗选择**：包块小来HCG低，MTX打针是前提，破裂出血休克时，开腹手术才保命

### 妊娠高血压

| 类型 | 诊断标准 | 处理 |
|------|----------|------|
| **轻度** | BP≥140/90mmHg（轻度），无蛋白尿 | 休息/左侧卧位/镇静 |
| **重度** | BP≥160/110mmHg，或伴有蛋白尿≥300mg/24h，或伴有脑/视觉/肾/肝/血液系统受累 | 硫酸镁解痉+降压+适时终止妊娠 |
| **子痫** | 重度基础上抽搐/昏迷 | 硫酸镁（首选）+甘露醇+终止妊娠 |

### 高频考点

- 硫酸镁用法：4-6g静滴+1-2g/h维持，血镁浓度1.8-3.0mmol/L为有效治疗浓度
- 硫酸镁中毒：膝反射消失（首发，>3.5mmol/L）、呼吸<16次/分、尿量<25ml/h
- 终止妊娠指征：子痫控制2小时、胎盘早剥、胎儿窘迫、治疗无效
- HELLP综合征：溶血+肝酶升高+血小板减少（<100×10^9/L）

### 记忆口诀

> **重度子痫标准**：160/110两数字，蛋白300尿里示，脑眼肾肝血受累，硫酸镁来解痉配
> **HELLP记忆**：溶血血小板少，肝酶升高三联找

### 前置胎盘与胎盘早剥

| 项目 | 前置胎盘 | 胎盘早剥 |
|------|----------|----------|
| **定义** | 孕28周后胎盘附着于子宫下段/宫颈内口 | 孕20周后正常位置胎盘在胎儿娩出前剥离 |
| **分类** | 完全性/部分性/边缘性/低置胎盘 | I度（轻型，外出血）、II度（剥离1/3）、III度（剥离1/2） |
| **临床表现** | 无痛性反复阴道流血（完全性36周后/边缘性37周后/低置32周后） | 突发持续性腹痛+板状腹+子宫硬如木板+阴道流血（内出血为主） |
| **诊断** | 超声（定位胎盘位置，金标准） | 超声（胎盘后血肿） |
| **处理** | 期待疗法（卧床/抑制宫缩/纠正贫血）/剖宫产（主要分娩方式） | 纠正休克+输血+立即终止妊娠（剖宫产） |
| **高危因素** | 剖宫产史/多产/前置胎盘史/高龄 | 妊娠期高血压/外伤/胎膜早破/吸烟 |

### 高频考点

- 前置胎盘阴道流血特点：无痛性、反复发生、发生时间与类型相关
- 前置胎盘绝对禁止肛查和灌肠
- 胎盘早剥分型：显性剥离（外出血）、隐性剥离（内出血）、混合性剥离
- 胎盘早剥最严重并发症：DIC（凝血功能障碍）
- 子宫胎盘卒中：胎盘早剥血液浸入子宫肌层，子宫呈紫蓝色，收缩力减弱

### 记忆口诀

> **前置胎盘分类**：完全盖住宫颈口，部分盖住是亚型，边缘只到宫颈缘，低置胎盘旁上游
> **胎盘早剥三联**：腹痛板状腹加阴道流血，宫底升高胎心消失要当心

### 分娩

| 项目 | 要点 |
|------|------|
| **先兆临产** | 假宫缩（不规则/弱/夜间重）、胎儿下降感、见红（24-48h内分娩） |
| **临产标志** | 规律且逐渐增强的子宫收缩（持续30秒以上，间歇5-6分钟） |
| **产程** | 第一产程（宫口扩张，0-10cm，初产妇11-12h，经产妇6-8h）、第二产程（胎儿娩出，初产妇1-2h，经产妇数分钟至1h）、第三产程（胎盘娩出，5-15分钟，不超30分钟） |
| **宫缩** | 正常：持续30秒，间歇5-6分钟；强直性宫缩（子宫痉挛性收缩，禁用缩宫素） |
| **胎膜破裂** | 人工破膜（宫口开大3cm以上时），自然破膜（活跃期） |
| **会阴切开** | 指征：胎儿窘迫/会阴过紧/臀位/早产/需产钳助产 |

### 高频考点

- 宫口扩张：潜伏期（0-3cm，慢）、活跃期（3-10cm，快，>0.5cm/h）
- 胎头拨露：宫缩时胎头露出阴道口，宫缩间歇又缩回
- 胎头着冠：胎头双顶径越过骨盆出口
- 新生儿 Apgar评分：1分钟评分（心率/呼吸/肌张力/喉反射/皮肤颜色）每项0-2分，总分10分，<7分需复苏

### 记忆口诀

> **产程分期**：一流宫颈口扩张，二流胎儿快出胎，三流胎盘要剥离
> **Apgar评分**：心率呼吸肌张力，喉反射加皮颜色，每项2分共10，7分以上才安心

### 产后出血

| 项目 | 要点 |
|------|------|
| **定义** | 胎儿娩出后24h内阴道出血>500ml（剖宫产>1000ml） |
| **病因** | 子宫收缩乏力（最常见，70-80%）、胎盘因素（胎盘滞留/植入/残留）、软产道裂伤、凝血功能障碍 |
| **诊断** | 按摩子宫（可触及柔软/轮廓不清=宫缩乏力）、检查胎盘（是否完整）、检查软产道（裂伤）、凝血功能 |
| **处理** | 宫缩剂（缩宫素/米索前列醇/卡前列素氨丁三醇）、按摩子宫、宫腔填塞/球囊、子宫动脉结扎/B-Lynch缝合、介入治疗、切除子宫（最后手段） |
| **失血量** | 称重法（失血量=湿重-干重）×1.05、面积法（10cm²纱布≈10ml） |

### 高频考点

- 产后出血四大原因：宫缩乏力（最常见）、胎盘因素、软产道损伤、凝血障碍
- 宫缩乏力高危因素：巨大儿/多胎/羊水过多/产程延长/妊娠期高血压
- 产后2小时是产后出血高峰期
- 缩宫素用法：10-20U静滴+20-40U肌注，最大剂量<60U（易水中毒）
- 输血指征：Hb<70g/L，或急性失血>30%血容量

### 记忆口诀

> **产后出血原因**：宫缩乏力最常见，胎盘滞留软产裂，凝血障碍四连查
> **出血量估算**：称重法湿减干乘以1.05，面积法10见方10毫升

## 第二章 妇科

### 子宫肌瘤

| 项目 | 要点 |
|------|------|
| **分类** | 肌壁间（60-70%，最常见）、浆膜下（20-30%，可扪及）、粘膜下（10%，月经多） |
| **临床表现** | 月经增多/经期延长（粘膜下/肌壁间）、下腹肿块（浆膜下）、压迫症状（尿频/便秘）、不孕/流产 |
| **诊断** | 超声（金标准，边界清楚的低回声团块）、MRI（更准确） |
| **治疗** | 观察（无症状/小）、药物（GnRH-a/米非司酮缩小）、手术（子宫切除/肌瘤剔除） |
| **变性** | 玻璃样变（最常见）、囊性变、红色变性（妊娠/产褥期，剧烈腹痛）、肉瘤变（迅速增大/恶变） |

### 高频考点

- 子宫肌瘤最常见症状：月经改变（月经量增多/经期延长）
- 子宫肌瘤与妊娠：粘膜下肌瘤可阻碍受精卵着床，致不孕/流产
- 红色变性：妊娠期/产褥期，瘤体迅速增大，剧烈腹痛，需保守治疗（卧床/抗生素/黄体酮）
- 手术指征：月经过多致贫血/子宫>10周妊娠大小/生长过快/怀疑恶变

### 记忆口诀

> **肌瘤分类**：肌壁浆膜粘膜下，症状轻重各不同，月经增多粘膜下，腹部肿块浆膜生
> **红色变性**：妊娠产褥剧烈痛，保守治疗卧床先

### 卵巢囊肿

| 项目 | 要点 |
|------|------|
| **分类** | 功能性囊肿（黄素囊肿/滤泡囊肿，非肿瘤）、卵巢肿瘤（良性/恶性） |
| **良性肿瘤** | 卵巢浆液性囊腺瘤（单房/乳头）、粘液性囊腺瘤（多房/胶冻样）、成熟畸胎瘤（皮样囊肿，20-40岁） |
| **恶性肿瘤** | 卵巢上皮性癌（浆液性/粘液性/子宫内膜样/透明细胞）、生殖细胞肿瘤（无性细胞瘤/内胚窦瘤/畸胎瘤恶变） |
| **临床表现** | 腹胀/腹部肿块/腹水（恶性）、蒂扭转（突发剧痛，恶心呕吐）、破裂（剧烈腹痛） |
| **诊断** | 超声（初步筛查）、肿瘤标志物（CA125/AFP/HE4）、MRI/CT |
| **治疗** | 观察（<5cm生理性）、手术（囊肿剔除/附件切除/分期手术）、化疗（恶性） |
| **蒂扭转** | 常见于中等大小、活动度大的囊肿，体位改变突发，妇科急症 |

### 高频考点

- CA125：上皮性卵巢癌常用标志物（正常<35U/ml），子宫内膜异位症也可升高
- AFP：内胚窦瘤特异性标志物，畸胎瘤恶变也可升高
- 卵巢囊肿蒂扭转：体位改变+突发一侧腹痛+恶心呕吐，超声CDFI显示血流减少/消失
- 成熟畸胎瘤：最常见良性肿瘤，含有2-3个胚层组织，超声特异性"面团征"
- 绝经后卵巢肿块：恶性可能大，需积极处理

### 记忆口诀

> **囊肿扭转**：体位改变来诱发，突发剧痛恶呕求，超声看血流，紧急手术来保宫
> **肿瘤标记物**：CA125上皮癌，AFP内胚窦，畸胎瘤看甲胎

### 宫颈癌

| 项目 | 要点 |
|------|------|
| **病因** | HPV感染（90%以上，16/18型高危）、早婚早育/多产/包皮垢/性伴侣多 |
| **筛查** | 宫颈刮片（巴氏分类，I-V级）、TCT（液基细胞学）、HPV检测（高危型16/18型） |
| **病理** | 鳞状细胞癌（80-85%，宫颈外口柱状-鳞状上皮交界处）、腺癌（15-20%）、腺鳞癌 |
| **分期** | I期（局限于宫颈）、II期（超出子宫但未达盆壁/阴道下1/3）、III期（达盆壁/阴道下1/3）、IV期（超出真骨盆/侵犯膀胱直肠） |
| **早期诊断** | 宫颈刮片→阴道镜→活检（确诊依靠病理） |
| **治疗** | 手术（Ia-Ib1期：广泛子宫切除+盆腔淋巴结清扫）、放疗（Ib2期及以上）、化疗（同步放化疗） |

### 高频考点

- 宫颈刮片筛查时间：21-29岁每3年一次TCT，30-65岁每5年一次TCT+HPV联合筛查
- 宫颈癌最常见病理类型：鳞状细胞癌
- 宫颈癌转移途径：直接蔓延（最常见）、淋巴转移（髂内/闭孔/髂外淋巴结）、血行转移（晚期）
- 早期症状：接触性出血（性交后/妇科检查后），白带增多
- FIGO分期：临床分期（不是手术病理分期）

### 记忆口诀

> **宫颈癌筛查**：21岁开始做刮片，3年一次记心间，30岁后加HPV，5年联合保平安
> **分期记忆**：一期只在宫颈间，二期超离子宫缘，三期到了盆壁前，四期侵犯膀胱肠

### 子宫内膜癌

| 项目 | 要点 |
|------|------|
| **病因** | 雌激素刺激（无排卵性疾病/多囊卵巢/绝经晚/不孕）、肥胖/高血压/糖尿病（三联征） |
| **病理** | 内膜样腺癌（80%，最常见）、浆液性癌（预后差）、透明细胞癌 |
| **临床表现** | 绝经后阴道流血（最常见）、异常子宫出血（育龄期）、白带增多/下腹痛 |
| **诊断** | 超声（内膜厚度>4mm绝经后）、分段诊刮（确诊的金标准）、宫腔镜 |
| **分期** | I期（局限于子宫体）、II期（侵犯宫颈）、III期（超出子宫但在盆腔）、IV期（超出真骨盆） |
| **治疗** | 手术（I期：全子宫+双附件+盆腔淋巴结清扫）、放疗、化疗、激素治疗（孕激素） |

### 高频考点

- 子宫内膜癌最常见症状：绝经后阴道流血
- 高危因素：肥胖/高血压/糖尿病/未育/绝经晚/多囊卵巢
- 超声内膜厚度：育龄期<14mm，绝经后>4mm需警惕
- 确诊：金标准是分段诊刮（区分宫颈/宫体来源）
- 预后：与肌层浸润深度/分化程度/淋巴结转移相关

### 记忆口诀

> **高危因素**：肥糖高压未育伴，绝经太晚多囊卵
> **诊断要点**：绝经后出血压超声，诊刮病理金标准

### 子宫腺肌症

| 项目 | 要点 |
|------|------|
| **定义** | 子宫内膜腺体及间质侵入子宫肌层（内在性子宫内膜异位症） |
| **病理** | 弥漫性（多见，后壁为主，子宫球形增大）、局限性（腺肌瘤） |
| **临床表现** | 痛经（进行性加重，30-50%）、月经增多/经期延长（40-50%）、子宫增大（球形/质硬）、不孕 |
| **诊断** | 超声（肌层内低回声/血流增加）、MRI（更准确）、CA125（升高） |
| **治疗** | 观察/药物（GnRH-a/避孕药/左炔诺孕酮宫内节育器）、手术（子宫切除/腺肌病灶切除） |

### 高频考点

- 子宫腺肌症与子宫肌瘤鉴别：腺肌症子宫球形/均匀增大/压痛，肌瘤有结节感
- CA125：子宫腺肌症可升高，但卵巢肿瘤升高更显著
- 痛经特点：进行性加重，从月经前开始，持续整个经期
- 子宫切除是根治性治疗，适用于无生育要求者

### 记忆口诀

> **腺肌症特点**：内膜腺体侵入肌层，球形增大痛经随，月经增多常相伴
> **与肌瘤鉴别**：腺肌均匀球形压痛，肌瘤结节边界清

### 盆腔炎

| 项目 | 要点 |
|------|------|
| **病原体** | 需氧菌（链球菌/大肠杆菌）、厌氧菌（脆弱类杆菌）、支原体/衣原体、淋球菌 |
| **感染途径** | 上行性感染（最常见）、经淋巴系统蔓延、血行传播、直接蔓延 |
| **临床表现** | 急性：下腹痛+发热+阴道脓性分泌物+宫颈举痛+宫旁压痛；慢性：腰骶酸痛/下腹坠胀/经期加重 |
| **诊断** | 最低标准（宫颈举痛/子宫压痛/附件压痛）、附加标准（体温>38.3℃/异常分泌物/WBC↑）、特异性标准（腹腔镜活检） |
| **治疗** | 急性：抗生素（氧氟沙星+甲硝唑/头孢曲松+多西环素）、支持治疗；慢性：物理治疗/中药/抗生素 |
| **并发症** | 输卵管积水/卵巢囊肿/不孕症/异位妊娠/盆腔粘连 |

### 高频考点

- 盆腔炎最低诊断标准：宫颈举痛/子宫压痛/附件压痛（三选一）
- 治疗方案：门诊治疗（轻中度）、住院治疗（重度/脓肿/耐药）
- 输卵管积水：慢性盆腔炎后遗症，输卵管伞端闭锁
- 不孕：盆腔炎导致输卵管粘连/阻塞，是继发性不孕主要原因

### 记忆口诀

> **盆腔炎诊断**：宫颈举痛子宫压，附件压痛三中一，体温升高白升高，淋病衣原要警惕
> **治疗原则**：抗生素要早用，足量广谱要联合，门诊无效需住院，脓肿破裂要手术

## 第三章 生殖

### 不孕症

| 项目 | 要点 |
|------|------|
| **定义** | 育龄期夫妇同居1年，有正常性生活，未采取避孕措施未妊娠（原发性/继发性） |
| **病因** | 女性（排卵障碍30%/输卵管因素30%/子宫因素/宫颈因素）、男性（精液异常30%/输精障碍）、双方因素（20%）、不明原因 |
| **检查** | 女性：排卵监测（基础体温/B超/激素）、输卵管通畅试验（子宫输卵管造影/腹腔镜）、宫腔镜；男性：精液分析（正常：≥15×10^6/ml，PR≥32%，形态≥4%） |
| **治疗** | 诱发排卵（克罗米芬/来曲唑/促性腺激素）、输卵管手术（造口/吻合）、辅助生殖（IUI/IVF-ET/ICSI） |

### 高频考点

- 不孕症比例：女性因素40-50%，男性因素30-40%，双方因素10-20%，不明原因10%
- 排卵监测：B超（优势卵泡18-20mm），基础体温（双相型），激素（LH峰值）
- 输卵管造影：碘油或碘水，显影剂通过输卵管弥散入盆腔
- IVF-ET指征：输卵管阻塞/排卵障碍/子宫内膜异位症/男性因素/不明原因不孕

### 记忆口诀

> **不孕检查**：排卵监测三连环，超声激素基础温，输卵管通做造影，男方精液要先行
> **治疗顺序**：促排先行通管后，辅助生殖最后备

### 避孕方法

| 方法 | 机制 | 有效率 | 适应证/禁忌证 |
|------|------|--------|---------------|
| **激素避孕** | 抑制排卵/宫颈粘液变稠 | 90-99% | 短效（雌孕激素复方制剂）、长效（孕激素/雌孕激素）、紧急避孕（米非司酮/左炔诺孕酮） |
| **宫内节育器** | 干扰着床/炎性反应 | 90-95% | 铜环（含铜离子）、曼月乐（左炔诺孕酮），月经多者适合曼月乐 |
| **屏障避孕** | 阻止精子进入 | 男用避孕套85-98%、女用避孕套79-95% | 避孕套唯一可预防 STI |
| **输卵管结扎** | 阻断精卵相遇 | 99.5% | 永久性避孕，适用于完成生育计划者 |
| **输精管切除** | 阻断精子排出 | 99.5% | 永久性避孕，术后需排空残余精子（20次射精） |
| **安全期避孕** | 避开排卵期 | 75-80% | 不作为推荐，失败率高 |
| **体外射精** | 射精前抽出 | 70-80% | 失败率高，不推荐 |

### 高频考点

- 短效避孕药：雌激素+孕激素，21天周期，漏服需补救
- 紧急避孕：事后72小时内，米非司酮（10-25mg）或左炔诺孕酮（0.75mg×2次）
- 宫内节育器放置时间：月经干净3-7天，人流后立即放置，产后42天恶露干净后
- 曼月乐：含左炔诺孕酮52mg，可治疗月经过多/痛经，放置5年
- 复通术：输卵管结扎后可复通，但成功率低

### 记忆口诀

> **避孕方法选择**：短效药要每天服，套套防病又避孕，上环适合已生育，皮埋长效未婚人
> **紧急避孕**：事后72h赶紧吃，米非司酮或孕激素，失败率高不推荐，常规避孕是正道

### 卵巢囊肿蒂扭转

| 项目 | 要点 |
|------|------|
| **诱因** | 体位改变（快速转动）、腹压骤增、囊肿活动度大（中等大小） |
| **临床表现** | 突发一侧剧痛（持续性，阵发加重）、恶心呕吐、腹胀、发热 |
| **体征** | 患侧压痛、反跳痛、肌紧张，可触及肿块 |
| **超声** | 囊肿壁增厚/水肿，CDFI显示血流减少/消失 |
| **处理** | 急症手术（腹腔镜/开腹），根据卵巢情况决定（剔除/附件切除） |
| **时间窗** | 扭转>360°超过24小时，卵巢坏死，需切除 |

### 高频考点

- 蒂扭转是卵巢囊肿最常见并发症
- 典型表现：体位改变后突发剧痛+恶心呕吐
- 超声CDFI是关键检查：血流信号减少/消失提示蒂扭转
- 手术原则：尽量保留正常卵巢组织
- 扭转时间长（>24h/720°）：卵巢变黑坏死，需附件切除

### 记忆口诀

> **卵巢囊肿蒂扭转**：体位改变突发痛，恶心呕吐紧跟随，超声看血流，紧急手术保卵巢

## 第五章 妇科检查与操作

### 妇科检查

| 项目 | 要点 |
|------|------|
| **外阴检查** | 发育/有无畸形/炎症/肿块/白斑 |
| **阴道窥器检查** | 阴道壁（充血/溃疡/肿块）、宫颈（肥大/糜烂/息肉/肿瘤） |
| **双合诊** | 子宫位置/大小/活动度/压痛、附件（肿块/压痛） |
| **三合诊** | 直肠-阴道隔/子宫后壁/宫旁组织/骶骨前方（子宫内膜癌/宫颈癌分期） |
| **细胞学** | 宫颈刮片（巴氏分类）、TCT（Bethesda系统） |

### 高频考点

- 未婚女性禁做阴道窥器检查和双合诊，必要时做直肠-腹部诊
- 宫颈刮片取材：宫颈鳞柱交界处（转化区）
- 子宫位置：前位（60%）/中位/后位（30%），后位子宫腹主动脉与下腔静脉之间
- 子宫正常大小：7×5×3cm（长×宽×厚）

### 记忆口诀

> **双合诊**：一手入腹一手阴，子宫大小活动清，附件有无肿块找，压痛提示有炎症
> **三合诊**：食指入肛中指阴，子宫后壁骶骨前，癌肿浸润深度看，宫旁结转移判断
