## 一、生理学记忆口诀

### 1. 心室肌动作电位
**口诀名称**：心室肌动作电位五期记忆
**口诀全文**：
> "零三一五四静息，钠钙钾流要记清"

**逐字释义**：
| 口诀 | 含义 |
|------|------|
| 零 | 0期：快速去极化（Na+内流） |
| 三 | 1期：快速复极化（K+外流） |
| 一 | 2期：平台期（Ca2+内流+K+外流） |
| 五 | 3期：快速复极化（K+外流） |
| 四静息 | 4期：静息期（Na+-K+泵恢复） |

**口诀全文**：

### 4. 胸膜腔负压形成
**口诀名称**：胸膜腔负压成因
**口诀全文**：
> "胸廓扩展肺牵拉，内压低于大气压"

**逐字释义**：
- 胸廓自然扩张 → 肺被牵拉扩张
- 肺弹性回缩力 → 指向胸廓
- 胸膜腔内压 = 肺内压 - 肺弹性回缩力
- 结果：胸膜腔内压 < 大气压 = 负压

## 二、生物化学记忆口诀

### 13. 蛋白质等电点
**口诀名称**：氨基酸等电点计算
**口诀全文**：
> "酸氨基羧基两，解离看pKa"

**逐字释义**：
- 酸性氨基酸（Asp、Glu）：pI = (pKα-COOH + pKα-R)/2
- 碱性氨基酸（Lys、Arg、His）：pI = (pKα-NH3+ + pKα-R)/2
- 中性氨基酸：pI = (pKα-COOH + pKα-NH3+)/2

### 18. 脂肪酸β氧化
**口诀名称**：脂肪酸β氧化的过程
**口诀全文**：
> "活化进入线粒体，脱氢氧化再裂解"

**逐字释义**：
1. 活化：脂肪酸→脂酰CoA（细胞质）
2. 转运：肉碱脂酰转移酶I（限速）
3. β氧化：脱氢→加水→再脱氢→硫解
4.产物：乙酰CoA、FADH2、NADH
