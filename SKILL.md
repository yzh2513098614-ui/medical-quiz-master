## 2. 题目生成规则

### 2.1 每次一题原则

每次只出一道题，用户回答后再出下一题。避免连续出同一学科的题目（轮转出题规则）。

### 2.2 轮转出题规则

默认从已加载学科中轮转，顺序：生理学 → 生物化学 → 病理学 → 内科学 → 外科学 → 神经病学 → 精神病学 → 儿科学 → 妇产科学 → 急诊医学 → 皮肤病学 → 眼科学 → 耳鼻咽喉科学 → 老年医学 → 康复医学 → 麻醉学 → 肿瘤学 → 诊断学 → 临床检验诊断学 → 临床医学人文精神 → 传染病学 → 寄生虫学 → 免疫学 → 循环回生理学。

连续同一章节的题目不得超过2道。

### 2.3 题型分类

| 题型 | 说明 | 触发命令 |
|------|------|---------|
| A型题 | 单选题（五选一，最佳答案） | 默认 |
| B型题 | 配伍题（共用选项，5选2-5题） | `/题型 B型` |
| X型题 | 多选题（2-5个正确答案，全对得分） | `/题型 X型` |
| 简答题 | 简述题，考察记忆性知识点 | `/题型 简答` |
| 病例分析题 | 临床病例，多维度评分 | `/题型 病例` |
| 论述题 | 综合性问题，框架+深度 | `/题型 论述` |
| 306模式 | 按306标准结构出题 | `/考纲306` |
| 混合题型 | 随机混合上述题型 | `/题型 混合` |

**答题说明**：以下问题提供了共用的选项，请根据描述选择最匹配的答案。

## 3. 答题反馈规则

### 3.1 即时判断

用户回答后，立即判断对错，给出正确答案。

### 3.2 解析格式

```markdown
### 📖 解析

**正确答案**：[X]

**解析**：[3-5句话解析为什么会选错/选对]

**记忆口诀**：[一句话概括] "[口诀全文]"
> [口诀释义]

**知识点定位**：[学科] / [章节] / [具体知识点]

**相关考点**：
- [关联考点1]
- [关联考点2]
- [关联考点3]
```

### 3.3 错题记录

答错的题目自动记录到错题本，路径：`f:/Users/1/Documents/[科目]_错题本.md`

记录内容：题目、正确答案、用户答案、解析、口诀、时间戳。

### 3.4 每10题阶段性错题总结

每累计10道题后，自动弹出阶段性错题总结：

```markdown
---

## 📊 阶段性错题总结（第X-X题，共X道错题）

### 错题概览

| 题号 | 学科 | 错题知识点 | 错误原因 |
|------|------|-----------|---------|
| #3 | 生理学 | 心室肌动作电位 | 混淆离子流向 |
| #7 | 生物化学 | TCA循环关键酶 | 未记住限速酶 |
| ... | ... | ... | ... |

### 错题知识点分类

#### 🔴 概念混淆（X道）
- [知识点1]：正确理解 → [正确解释]

#### 🟡 记忆偏差（X道）
- [知识点1]：记忆错误 → 正确记忆

#### 🔵 审题失误（X道）
- [题目] → [正确做法]

### 高频错因分析

1. **最常见错误**：[错误类型]，建议 [改进方法]

### 下阶段复习建议

- **薄弱章节**：[章节名]，建议重新复习相关口诀
- **易错考点**：[具体考点]，建议加强对比记忆

---

**继续答题请回复"继续"**
```

## 4. AI自适应难度系统

### 4.1 难度等级

| 等级 | 标识 | 含义 | 示例 |
|------|------|------|------|
| ★☆☆☆☆ | 基础记忆 | 病因、定义、分类 | "哪种激素由垂体前叶分泌？" |
| ★★☆☆☆ | 理解应用 | 机制、鉴别、推理 | "肝硬化腹水形成的机制是？" |
| ★★★☆☆ | 临床综合 | 诊断、治疗、预后 | "该患者最可能的诊断是？" |
| ★★★★☆ | 复杂病例 | 多系统整合、复杂鉴别 | "结合病史和检查结果，分析该患者的治疗方案" |
| ★★★★★ | 考前冲刺 | 真题难度、综合应用 | "试述急性呼吸窘迫综合征的病理生理改变" |

### 4.2 升降档规则

```
初始难度：★★★☆☆（基础理解型）

连续答对2题 → 升一档（最多升至★★★★★）
连续答错2题 → 降一档（最多降至★☆☆☆☆）

单题答对 → 连续正确计数+1
单题答错 → 连续正确计数归零，连续错误计数+1
```

### 4.3 难度查看

```
/难度    → 查看当前难度
/难度 3  → 设置为★★★☆☆难度
```

> - 三=1期（K+外流）

> - 五=3期（K+外流）
> - 四=4期静息期（Na+-K+泵）

## 12. 注意事项

1. **每次一题**：严格遵守每次只出一道题的原则
2. **轮转出题**：避免连续出同一学科或同一章节的题目
3. **即时反馈**：答完立即给出解析，不等待
4. **错题记录**：答错的题必须记录到错题本
5. **口诀优先**：出题和解析时优先使用记忆口诀
6. **遗忘曲线**：技能启动时检查并推送待复习错题
7. **真实性**：所有题目和解析必须基于references文件中的真实内容，严禁编造
